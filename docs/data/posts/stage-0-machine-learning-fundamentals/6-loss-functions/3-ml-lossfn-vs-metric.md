# 3. Чем loss function отличается от метрики качества модели?

Loss function и метрики качества используются для оценки модели, но выполняют разные задачи.

## Loss function

Loss function используется **во время обучения модели**.

Она вычисляется для каждого примера и используется алгоритмом оптимизации.

## Метрика качества

Метрика качества используется **для оценки модели после обучения**.

Она показывает, насколько хорошо модель работает на данных.

## Пример

Для задачи классификации:

Loss function:

cross entropy

Метрики:

- accuracy  
- precision  
- recall  
- F1-score

## Основное различие

Loss function используется для обучения.

Метрики используются для оценки результата.

## Итог

Loss function управляет обучением модели, а метрики оценивают её качество.
