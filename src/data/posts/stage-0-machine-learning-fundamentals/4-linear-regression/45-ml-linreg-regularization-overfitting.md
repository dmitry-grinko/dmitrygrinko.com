# 45. Как регуляризация помогает бороться с переобучением?

Регуляризация уменьшает сложность модели.

Она делает это через ограничение значений коэффициентов.

## Что происходит без регуляризации

Если модель слишком гибкая, она может подстраиваться под шум в данных.

Это приводит к переобучению.

Модель начинает хорошо работать на training data, но плохо на новых данных.

## Роль регуляризации

Регуляризация вводит штраф за большие коэффициенты.

Это заставляет модель:

- использовать более простые зависимости  
- игнорировать шум

## Практический результат

Регуляризация помогает:

- улучшить обобщающую способность модели  
- уменьшить variance

## Итог

Регуляризация помогает модели не подстраиваться под случайные особенности training dataset.
