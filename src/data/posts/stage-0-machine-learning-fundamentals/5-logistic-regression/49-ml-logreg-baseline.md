# 49. Когда logistic regression может быть хорошим baseline?

Logistic regression часто используется как **baseline модель** для задач классификации.

Она позволяет быстро получить первое решение.

## Табличные данные

Logistic regression хорошо работает на структурированных данных.

## Быстрая проверка гипотез

Модель позволяет быстро проверить, есть ли связь между признаками и классом.

## Сравнение моделей

Baseline помогает понять, дают ли более сложные алгоритмы реальное улучшение.

## Интерпретация

Модель может помочь понять, какие признаки наиболее важны.

## Итог

Logistic regression часто используется как baseline для задач бинарной классификации.
