# 47. Что такое weighted loss?

Weighted loss — это функция потерь, в которой **разные примеры имеют разные веса**.

Это означает, что ошибки некоторых объектов считаются более важными.

## Основная идея

Каждой ошибке назначается коэффициент веса.

loss = weight × error

## Пример

Если ошибки для редкого класса более важны, им можно назначить больший вес.

## Использование

Weighted loss часто применяется в задачах с несбалансированными данными.

## Итог

Weighted loss позволяет контролировать важность различных примеров в обучении.
