# 31. Как logistic regression обучается?

Logistic regression обучается через **минимизацию функции потерь (log loss)**.

Во время обучения модель пытается найти такие параметры, которые делают предсказанные вероятности максимально близкими к правильным классам.

## Общая идея

Процесс обучения выглядит так:

1. модель делает предсказание вероятности  
2. вычисляется log loss  
3. вычисляется градиент функции потерь  
4. параметры модели обновляются  

Этот процесс повторяется много раз.

## Что происходит с моделью

С каждой итерацией коэффициенты модели постепенно изменяются.

В результате модель начинает лучше разделять классы.

## Итог

Logistic regression обучается через оптимизацию функции log loss с помощью итерационного обновления параметров.
