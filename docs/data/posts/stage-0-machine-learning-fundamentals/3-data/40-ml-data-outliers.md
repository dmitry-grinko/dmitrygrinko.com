# 40. Что такое outliers?

Outliers — это наблюдения, которые сильно отличаются от остальных данных в dataset.

Их часто называют **выбросами**.

## Пример

Dataset:

| income |
|------|
| 40000 |
| 42000 |
| 45000 |
| 47000 |
| 5000000 |

Последнее значение сильно отличается от остальных.

Это может быть:

- ошибка данных  
- редкое событие  
- аномальное наблюдение

## Почему outliers возникают

Выбросы могут появляться по разным причинам:

- ошибки измерения  
- ошибки ввода данных  
- редкие реальные события

## Почему это важно

Некоторые модели очень чувствительны к выбросам.

Например, линейная регрессия может сильно изменять свои параметры из-за нескольких экстремальных значений.

## Итог

Outliers — это наблюдения, которые значительно отличаются от основной массы данных и могут влиять на обучение модели.
