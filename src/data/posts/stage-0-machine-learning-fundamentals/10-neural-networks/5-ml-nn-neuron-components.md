# 5. Из каких компонентов состоит нейрон?

Искусственный нейрон состоит из нескольких основных компонентов.

## Основные элементы

**Входы (inputs)**  
Значения, поступающие в нейрон.

**Веса (weights)**  
Параметры, которые определяют важность каждого входа.

**Bias**  
Дополнительный параметр, который позволяет смещать результат.

**Сумматор**  
Вычисляет линейную комбинацию входов.

**Функция активации**  
Преобразует результат в выходное значение.

## Общая схема

inputs → веса → суммирование → функция активации → output

## Итог

Нейрон состоит из входов, весов, bias и функции активации.
