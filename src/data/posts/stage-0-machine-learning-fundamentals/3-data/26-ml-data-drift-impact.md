# 26. Как drift влияет на модель?

Drift может существенно ухудшить качество модели.

Модель обучается на предположении, что данные в будущем будут похожи на training dataset.

Когда это предположение нарушается, модель начинает ошибаться.

## Основные последствия

Drift может приводить к:

- снижению точности модели  
- увеличению количества ошибок  
- ухудшению стабильности предсказаний

## Почему это происходит

Модель фактически выучила закономерности старых данных.

Если новые данные отличаются, модель может делать неправильные выводы.

## Пример

Модель обучена предсказывать спрос на товары.

Если появляются новые тренды или меняется поведение пользователей, модель может продолжать опираться на старые закономерности.

## Итог

Drift приводит к постепенному ухудшению качества модели, потому что данные меняются со временем.
