# 26. Как dataset bias влияет на модель?

Если dataset содержит bias, модель обучается на искажённых данных и начинает воспроизводить те же искажения в своих предсказаниях.

Модель не знает, какие данные являются «правильными» или «репрезентативными». Она просто ищет закономерности в том dataset, который ей дали.

Поэтому если training data перекошены, модель будет считать эти перекосы нормальными.

## Пример

Представим dataset для системы рекомендаций.

Если большинство данных относится к пользователям одной возрастной группы, модель будет лучше понимать предпочтения именно этой группы и хуже работать для остальных пользователей.

Проблема возникает не потому, что алгоритм плохой, а потому что dataset не отражает реальное распределение пользователей.

## Что происходит на практике

Dataset bias может привести к нескольким проблемам:

- модель систематически ошибается для некоторых типов данных
- качество модели сильно отличается для разных групп объектов
- предсказания становятся менее надёжными

Такие ошибки могут долго оставаться незамеченными, если dataset не анализировать внимательно.

## Итог

Dataset bias приводит к тому, что модель учится на искажённой картине данных и воспроизводит эти искажения в своих предсказаниях.
