# 27. Что такое noise в данных?

Noise в данных — это случайные ошибки, неточности или нерелевантная информация в dataset, которые не отражают реальные закономерности.

Другими словами, noise — это данные, которые **мешают модели понять настоящие зависимости**.

Noise может появляться по разным причинам:

- ошибки измерений
- неправильная разметка данных
- технические сбои при сборе данных
- человеческие ошибки

## Пример

Представим dataset с ценами квартир.

| area | rooms | price |
|-----|------|------|
| 45  | 1    | 120000 |
| 70  | 2    | 185000 |
| 80  | 3    | 240000 |
| 75  | 2    | 900000 |

Последняя строка может быть noise, если цена указана ошибочно.

Такие значения могут сбивать модель с толку и ухудшать качество обучения.

## Как noise влияет на модель

Если в dataset много шума:

- модели сложнее находить реальные закономерности
- обучение становится менее стабильным
- качество предсказаний может снижаться

Поэтому в ML-проектах часто проводят **очистку данных (data cleaning)** перед обучением.

## Итог

Noise — это случайные ошибки или нерелевантные данные в dataset, которые не несут полезной информации и могут ухудшать обучение модели.
