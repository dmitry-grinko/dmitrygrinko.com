# 26. Когда MAE предпочтительнее MSE?

MAE предпочтительнее, когда данные содержат **выбросы**.

Поскольку MAE не возводит ошибку в квадрат, большие ошибки не доминируют в функции потерь.

## Пример

Если dataset содержит редкие аномальные значения, MSE может слишком сильно реагировать на них.

MAE остаётся более стабильной.

## Практические случаи

MAE часто используют в:

- задачах прогнозирования спроса  
- временных рядах  
- экономических моделях

## Итог

MAE лучше подходит для данных с выбросами.
