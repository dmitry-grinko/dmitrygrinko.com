# 6. Что значит «обучить модель»?

Обучить модель — значит подобрать её параметры так, чтобы она как можно точнее делала предсказания на основе данных.

Во время обучения алгоритм анализирует примеры, где известен правильный ответ, и постепенно настраивает параметры модели.

Общая схема выглядит так:

```
данные + правильные ответы → обучение → модель
```

После обучения модель можно применять к новым данным.

## Как происходит обучение

Типичный процесс обучения состоит из повторяющегося цикла:

1. модель получает входные данные (features)
2. делает предсказание
3. вычисляется ошибка между предсказанием и правильным ответом
4. параметры модели корректируются
5. цикл повторяется

Цель обучения — найти параметры модели, при которых ошибка становится минимальной.

## Пример модели

Например, простая модель может выглядеть так:

```
price = w1 × area + w2 × rooms + b
```

Где:

- area — площадь квартиры
- rooms — количество комнат
- w1, w2, b — параметры модели

Во время обучения алгоритм подбирает значения этих параметров.

## Какие библиотеки обычно используют

В реальных проектах обучение моделей почти всегда делается с помощью готовых библиотек.

Наиболее распространённые инструменты:

**Python ML-библиотеки**

- scikit-learn — базовые алгоритмы машинного обучения
- XGBoost — популярная библиотека для градиентного бустинга
- LightGBM — быстрый бустинг для больших данных
- CatBoost — бустинг, хорошо работающий с категориальными признаками

**Deep learning**

- PyTorch
- TensorFlow
- JAX

Эти библиотеки реализуют алгоритмы обучения и оптимизации. Разработчику обычно не нужно писать их с нуля.

## Как выглядит обучение на практике

Пример обучения модели в scikit-learn:

```python
from sklearn.linear_model import LinearRegression

model = LinearRegression()

model.fit(X_train, y_train)

prediction = model.predict(X_test)
```

Метод `fit()` запускает процесс обучения.

## Что происходит внутри обучения

Во время обучения выполняются несколько важных шагов:

- вычисляется функция ошибки (loss function)
- используется алгоритм оптимизации (например gradient descent)
- параметры модели постепенно обновляются

В итоге получается набор параметров, который лучше всего описывает данные.

## Что получается после обучения

После обучения модель обычно сохраняют:

- файл модели
- веса нейронной сети
- сериализованный объект

Затем модель используют в production:

```
features → модель → предсказание
```

## Итог

Обучить модель — значит:

- взять данные
- выбрать алгоритм
- запустить процесс оптимизации
- получить набор параметров модели

После этого модель можно использовать для предсказаний на новых данных.
