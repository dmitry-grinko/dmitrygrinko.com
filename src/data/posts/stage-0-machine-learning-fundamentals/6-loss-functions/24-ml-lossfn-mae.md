# 24. Что такое Mean Absolute Error (MAE)?

Mean Absolute Error (MAE) — это функция потерь, которая измеряет **среднюю абсолютную разницу между prediction и target**.

## Формула

MAE = (1/n) Σ |y - ŷ|

где:

- y — реальное значение  
- ŷ — предсказание  
- n — количество примеров

## Как это работает

Для каждого примера вычисляется абсолютная ошибка.

Затем ошибки усредняются.

## Пример

prediction = 80  
target = 100  

error = |100 - 80| = 20

## Итог

MAE измеряет среднюю абсолютную величину ошибок модели.
