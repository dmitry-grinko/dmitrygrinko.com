# 35. Что такое binary cross entropy?

Binary cross entropy — это версия cross entropy для **бинарной классификации**.

Она используется, когда существует только два класса.

## Пример задач

- spam detection  
- fraud detection  
- медицинская диагностика  

## Формула

Binary cross entropy вычисляется так:

loss = −(y log(p) + (1 − y) log(1 − p))

где:

- y — правильный класс  
- p — предсказанная вероятность

## Интерпретация

Если вероятность правильного класса высокая, loss маленький.

Если вероятность низкая, loss становится большим.

## Итог

Binary cross entropy — это функция потерь для бинарной классификации.
