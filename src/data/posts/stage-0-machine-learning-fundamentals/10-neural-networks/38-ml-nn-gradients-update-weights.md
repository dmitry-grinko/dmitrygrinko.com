# 38. Как градиенты используются для обновления весов?

Градиенты используются оптимизационным алгоритмом для обновления параметров модели.

Самый простой метод — gradient descent.

## Формула обновления

w = w - learning_rate × gradient

где:

- w — параметр
- gradient — производная функции потерь
- learning_rate — шаг обучения

## Пример

если:

gradient = 0.5  
learning_rate = 0.1

тогда изменение веса:

Δw = 0.05

## Интерпретация

Параметры постепенно изменяются в направлении уменьшения ошибки.

## Итог

Градиенты показывают, как изменять веса, чтобы уменьшить функцию потерь.
