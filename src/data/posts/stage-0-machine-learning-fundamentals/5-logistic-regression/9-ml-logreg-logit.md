# 9. Что такое logit?

Logit — это значение **линейной комбинации признаков до применения sigmoid функции**.

Это промежуточный результат модели.

## Формула

logit = w₁x₁ + w₂x₂ + ... + b

Это значение может быть любым числом.

## Что происходит дальше

Logit передаётся в sigmoid функцию.

Sigmoid преобразует его в вероятность.

p = sigmoid(logit)

## Пример

logit = 2.1

После sigmoid:

p ≈ 0.89

## Почему используется термин logit

В статистике logit связан с логарифмом отношения вероятностей (odds).

Но в машинном обучении его часто используют просто как название линейного выхода модели.

## Итог

Logit — это линейная комбинация признаков до применения sigmoid функции.
