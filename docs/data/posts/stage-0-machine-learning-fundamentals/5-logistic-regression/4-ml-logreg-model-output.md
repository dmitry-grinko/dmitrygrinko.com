# 4. Что является выходом модели logistic regression?

Выходом logistic regression является **вероятность принадлежности объекта к положительному классу**.

Значение всегда находится в диапазоне:

0 ≤ P ≤ 1

## Пример

Если модель анализирует письмо и выдаёт:

P(spam) = 0.73

это означает, что вероятность того, что письмо является спамом, равна 73%.

## Что происходит дальше

Эта вероятность затем преобразуется в класс.

Например:

если вероятность > 0.5 → класс 1  
если вероятность ≤ 0.5 → класс 0

## Почему используется вероятность

Вероятность позволяет:

- оценить уверенность модели
- менять порог классификации
- использовать разные метрики

## Итог

Logistic regression выдаёт вероятность того, что объект принадлежит определённому классу.
