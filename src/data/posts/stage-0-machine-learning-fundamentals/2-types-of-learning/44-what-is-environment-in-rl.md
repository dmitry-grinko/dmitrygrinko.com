# 44. Что такое environment?

Environment — это среда, в которой действует агент в reinforcement learning.

Среда определяет условия задачи и реагирует на действия агента.

## Как работает environment

Когда агент выполняет действие, среда:

1. изменяет своё состояние  
2. возвращает новое состояние агенту  
3. выдаёт reward

Таким образом агент получает обратную связь о своих действиях.

## Пример

Если reinforcement learning используется для обучения игры:

- environment — это сама игра  
- состояние — положение объектов на игровом поле  
- действия — возможные ходы игрока

Среда изменяется после каждого действия агента.

## Почему environment важен

Среда определяет:

- какие состояния возможны  
- какие действия разрешены  
- какие награды получает агент

Поэтому именно environment задаёт правила всей задачи.

## Итог

Environment — это система или среда, с которой взаимодействует агент и которая определяет результаты его действий.
