# 38. Почему multicollinearity может быть проблемой?

Multicollinearity создаёт несколько проблем для линейной регрессии.

Основная проблема — **нестабильность коэффициентов модели**.

## Нестабильные параметры

Когда признаки сильно коррелируют, модель может менять коэффициенты очень сильно при небольшом изменении данных.

Это делает модель нестабильной.

## Сложная интерпретация

Если признаки дублируют информацию, становится трудно понять:

- какой признак действительно влияет на target  
- насколько сильно он влияет

## Увеличение дисперсии коэффициентов

Multicollinearity увеличивает неопределённость параметров модели.

Это ухудшает статистическую надёжность модели.

## Итог

Multicollinearity делает коэффициенты нестабильными и усложняет интерпретацию линейной модели.
