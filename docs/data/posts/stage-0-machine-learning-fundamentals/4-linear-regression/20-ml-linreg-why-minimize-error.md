# 20. Почему минимизация ошибки важна для обучения модели?

Обучение модели фактически означает **поиск параметров, которые минимизируют ошибку предсказаний**.

Модель начинает обучение со случайных параметров.

Затем постепенно корректирует их так, чтобы уменьшить значение функции потерь.

## Процесс обучения

Во время обучения:

1. модель делает prediction  
2. вычисляется ошибка  
3. параметры модели корректируются

Этот процесс повторяется много раз.

## Почему это работает

Когда ошибка уменьшается, предсказания модели становятся ближе к реальным значениям.

Таким образом модель постепенно улучшает качество.

## Оптимизация модели

Минимизация ошибки — это задача оптимизации.

Алгоритмы обучения ищут такие параметры модели, которые дают минимальное значение функции потерь.

## Итог

Минимизация ошибки — это основной механизм обучения моделей машинного обучения.
