# 6. Что такое признаки (features) в линейной регрессии?

Features — это входные переменные, которые используются моделью для предсказания.

Каждый признак описывает некоторую характеристику объекта.

## Пример

Dataset с информацией о квартирах:

| area | rooms | price |
|----|----|----|
| 45 | 1 | 120000 |

Здесь:

- area — признак  
- rooms — признак  

Модель использует эти признаки, чтобы предсказать цену.

## Роль признаков

Features являются входом модели.

Модель анализирует связь между признаками и целевой переменной.

Чем лучше признаки отражают реальную задачу, тем лучше модель может обучиться.

## Feature engineering

В реальных проектах часто создают новые признаки.

Например:

- price_per_room  
- age_of_building  
- distance_to_center

Такие признаки могут улучшить модель.

## Итог

Features — это входные переменные, которые описывают объект и используются моделью для предсказания.
