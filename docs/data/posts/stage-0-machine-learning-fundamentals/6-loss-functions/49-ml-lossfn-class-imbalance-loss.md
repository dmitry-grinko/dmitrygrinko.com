# 49. Почему class imbalance требует изменения функции потерь?

Class imbalance означает, что один класс встречается намного чаще другого.

Например:

- 99% нормальных транзакций  
- 1% мошеннических

## Проблема

Модель может просто предсказывать всегда нормальный класс.

И всё равно получать высокую accuracy.

## Решение

Можно изменить функцию потерь, чтобы:

- ошибки редкого класса штрафовались сильнее

Это заставляет модель уделять больше внимания редким событиям.

## Итог

При class imbalance функцию потерь часто модифицируют, чтобы учитывать редкие классы.
