# 21. Что такое distribution данных?

Distribution данных — это то, **как значения признаков распределены в dataset**.

Проще говоря, distribution показывает:

- какие значения встречаются чаще  
- какие значения редкие  
- какие диапазоны значений существуют

## Пример

Представим dataset с возрастом пользователей:

| age |
|----|
| 21 |
| 23 |
| 24 |
| 25 |
| 60 |

Если большинство значений находятся в диапазоне 20–30 лет, можно сказать, что distribution смещено в сторону молодых пользователей.

## Почему distribution важно

Модели машинного обучения фактически учатся на распределении данных.

Если структура данных меняется, модель может начать работать хуже.

Например:

- training dataset содержит молодых пользователей  
- production данные содержат пользователей старшего возраста

В этом случае модель может давать плохие предсказания.

## Итог

Distribution данных — это статистическая структура dataset, которая описывает, как распределены значения признаков.
