# 44. Что такое Mean Absolute Error (MAE)?

Mean Absolute Error (MAE) — это метрика regression, которая измеряет **среднюю абсолютную ошибку предсказаний**.

## Формула

MAE = (1/n) Σ |y − ŷ|

## Как она работает

Для каждого объекта вычисляется абсолютная разница между предсказанием и реальным значением.

Затем все ошибки усредняются.

## Особенность

MAE одинаково учитывает все ошибки и не усиливает влияние больших отклонений.

## Итог

MAE показывает среднюю абсолютную разницу между предсказаниями и реальными значениями.
