# 27. Что происходит во время forward pass?

Во время forward pass сеть последовательно вычисляет выход каждого слоя.

Это чисто вычислительный этап: сеть получает вход, преобразует его и строит итоговый output.

## Последовательность шагов

Для каждого слоя происходит следующее:

1. берутся входные значения  
2. умножаются на веса  
3. добавляется bias  
4. применяется функция активации  
5. результат передаётся следующему слою

## Формально

Для одного слоя вычисление выглядит так:

z = Wx + b  
h = activation(z)

где:

- x — вход слоя  
- W — веса  
- b — bias  
- h — выход слоя

## Пример

Пусть нейрон получает два входа:

- x₁ = 2  
- x₂ = 3  

и имеет веса:

- w₁ = 0.5  
- w₂ = -1.0  

bias = 1

Тогда:

z = 0.5 × 2 + (-1.0) × 3 + 1 = -1

Если activation = ReLU, то:

output = max(0, -1) = 0

Это выход нейрона во время forward pass.

## Важная деталь

Во время forward pass сеть **не обучается**. Она только применяет текущие параметры к данным.

## Итог

Во время forward pass сеть вычисляет выход каждого слоя и в конце получает итоговое предсказание.
