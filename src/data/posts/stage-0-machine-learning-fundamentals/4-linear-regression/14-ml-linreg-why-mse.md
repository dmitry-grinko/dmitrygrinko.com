# 14. Почему MSE часто используется как функция потерь?

Mean Squared Error широко используется в линейной регрессии по нескольким причинам.

## Простая математическая форма

Функция MSE имеет удобные математические свойства.

Она:

- непрерывная  
- гладкая  
- дифференцируемая

Это позволяет эффективно использовать методы оптимизации.

## Сильный штраф за большие ошибки

Поскольку ошибка возводится в квадрат, большие ошибки наказываются сильнее.

Это заставляет модель уделять больше внимания крупным отклонениям.

## Удобство для оптимизации

MSE хорошо работает с методами оптимизации, такими как gradient descent.

Благодаря этому параметры модели можно находить эффективно.

## Теоретические свойства

При определённых условиях минимизация MSE даёт статистически оптимальную оценку параметров.

## Итог

MSE часто используется, потому что он удобен для оптимизации и сильно наказывает большие ошибки.
