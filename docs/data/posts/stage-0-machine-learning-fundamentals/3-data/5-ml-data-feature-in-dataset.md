# 5. Что такое feature в dataset?

Feature — это характеристика объекта, которая используется моделью для обучения.

Проще говоря, feature — это **входная переменная**, которая описывает объект.

## Пример

Представим dataset с информацией о квартирах:

| area | rooms | price |
|-----|------|------|
| 45 | 1 | 120000 |
| 70 | 2 | 185000 |

Здесь:

- area — feature  
- rooms — feature  
- price — target

Модель использует признаки (features), чтобы предсказать результат.

## Роль features

Features определяют, какую информацию модель получает о задаче.

Если признаки информативны, модель может найти полезные закономерности.

Если признаки плохо отражают реальность, модель не сможет построить хорошее предсказание.

## Feature engineering

В ML-проектах часто приходится:

- создавать новые признаки  
- преобразовывать существующие  
- удалять нерелевантные признаки

Этот процесс называется **feature engineering**.

## Итог

Feature — это входная переменная dataset, которая описывает объект и используется моделью для обучения.
