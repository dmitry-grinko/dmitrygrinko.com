# 1. Что такое machine learning?

## Что такое машинное обучение

Если объяснить максимально просто, машинное обучение (Machine Learning) — это способ создавать программы, которые учатся на данных, а не программируются жёсткими правилами.

В традиционном программировании всё работает так:

1. человек пишет правила
2. программа выполняет эти правила
3. получается результат

Например:

* если температура < 0 → вывести "мороз"
* если температура ≥ 0 → вывести "не мороз"

Правила задаёт человек.

В машинном обучении подход другой:

1. программе дают много данных
2. алгоритм ищет закономерности
3. программа сама строит модель, которая делает предсказания

То есть вместо того чтобы писать правила вручную, мы позволяем алгоритму их обнаружить.

## Простой пример

Представим задачу: определить, является ли письмо спамом.

Старый подход:

* если письмо содержит "free money" → спам
* если письмо содержит "lottery" → спам

Но спамеры быстро меняют текст, и правила перестают работать.

В машинном обучении делают иначе:

1. собирают тысячи писем
2. отмечают какие из них спам
3. обучают модель

После обучения модель сама понимает, какие слова и структуры чаще встречаются в спаме.

Теперь она может анализировать новые письма.

## Откуда вообще взялась эта идея

Идея машинного обучения появилась задолго до современных нейросетей.

Некоторые важные даты:

### 1950 — Алан Тьюринг

![Алан Тьюринг](alan_turing.jpg)

Британский математик Алан Тьюринг задаётся вопросом:

> Может ли машина мыслить?

Он предлагает знаменитый Turing Test — эксперимент, который проверяет, может ли машина вести разговор как человек.

### 1957 — Перцептрон

![Перцептрон](perceptron.jpg)

Американский учёный Фрэнк Розенблатт создаёт один из первых алгоритмов машинного обучения — Perceptron.

Это была очень простая нейронная сеть, способная распознавать изображения.

### 1986 — обратное распространение ошибки

В 1986 году публикуется работа о backpropagation — алгоритме обучения нейронных сетей.

Это ключевой момент, потому что именно этот метод используется для обучения большинства современных моделей.

### 2006 — начало эпохи Deep Learning

Исследователь Geoffrey Hinton показывает, что большие нейронные сети можно эффективно обучать.

Это запускает новую волну интереса к нейросетям.

### 2012 — прорыв в компьютерном зрении

Модель AlexNet выигрывает соревнование ImageNet.

Ошибка распознавания изображений резко падает.

После этого почти вся индустрия переходит на deep learning.

### 2017 — архитектура Transformer

Google публикует статью ["Attention Is All You Need"](https://research.google/pubs/attention-is-all-you-need/).

Эта архитектура становится основой для современных языковых моделей.

![The Transformer - model architecture](tranformer-model-architecture.png)


### 2022 — массовое распространение LLM

Появляются большие языковые модели (Large Language Models), которые могут:

* писать текст
* переводить
* программировать
* отвечать на вопросы

Именно к этому классу моделей относятся современные AI-ассистенты.

## Как работает машинное обучение

В большинстве случаев процесс выглядит так:

![Цикл машинного обучения: данные, обучение, предсказания](ml-workflow.png)

### 1. Сбор данных

Алгоритму нужны данные.

Например:

* тексты
* изображения
* аудио
* числовые таблицы

Чем больше данных — тем лучше модель может учиться.

### 2. Обучение модели

Алгоритм начинает искать закономерности.

Например:

* какие признаки связаны с результатом
* какие комбинации данных повторяются
* какие структуры чаще встречаются

В результате строится математическая модель.

![Нейронная сеть](neural_network.png)

### 3. Предсказания

После обучения модель может работать с новыми данными.

Например:

* определить, спам ли письмо
* распознать объект на изображении
* перевести текст
* предсказать вероятность события

## Основные типы машинного обучения

Существует несколько основных типов ML.

### Обучение с учителем (Supervised Learning)

Самый распространённый тип.

Модель обучается на данных, где уже известен правильный ответ.

Пример:

* изображение → "кот"
* изображение → "собака"

Модель учится связывать входные данные с правильной меткой.

### Обучение без учителя (Unsupervised Learning)

В этом случае правильных ответов нет.

Алгоритм сам пытается найти структуру данных.

Например:

* группировать похожие объекты
* находить скрытые закономерности

### Обучение с подкреплением (Reinforcement Learning)

Алгоритм учится через награды и штрафы.

Пример:

* обучение роботов
* обучение игровых AI
* оптимизация стратегий

## Где используется машинное обучение

Сегодня ML используется практически везде.

Примеры:

### поисковые системы

Google использует ML для ранжирования результатов поиска.

### рекомендательные системы

Netflix, YouTube, Amazon используют ML для рекомендаций.

### компьютерное зрение

* распознавание лиц
* медицинская диагностика
* автономные автомобили

### обработка языка

* перевод
* чат-боты
* поиск информации

## Важный момент

ML это:

* математика
* статистика
* алгоритмы
* большие объёмы данных
* вычислительные ресурсы

И главное — инженерия.

Потому что создать модель — это только часть задачи.

Настоящая сложность начинается, когда модель нужно:

* обучить на больших данных
* ускорить
* развернуть в production
* масштабировать

## Итог

Машинное обучение — это область компьютерных наук, которая занимается созданием алгоритмов, способных учиться на данных и делать предсказания.

Идея появилась десятки лет назад, но только в последние годы, благодаря:

* огромным datasets
* мощным GPU
* новым архитектурам моделей

машинное обучение стало основой современной AI индустрии.

Если честно, самое интересное начинается не здесь.

Понять что такое ML — это примерно как понять, что такое двигатель.

Настоящая инженерия начинается дальше:

* как обучать большие модели
* как их ускорять
* как масштабировать системы
* как запускать их в production.

И вот там уже начинается действительно интересная часть.
