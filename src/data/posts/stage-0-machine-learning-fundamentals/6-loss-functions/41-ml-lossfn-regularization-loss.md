# 41. Что такое regularization loss?

Regularization loss — это дополнительный компонент функции потерь, который **штрафует слишком большие значения параметров модели**.

Он добавляется к основной функции потерь.

## Общая формула

objective = loss + regularization

где:

- loss — ошибка предсказаний  
- regularization — штраф за сложность модели

## Основная идея

Если коэффициенты модели становятся слишком большими, регуляризация увеличивает общий loss.

Это заставляет модель выбирать более простые решения.

## Зачем это нужно

Регуляризация помогает:

- уменьшить сложность модели  
- улучшить обобщающую способность  
- бороться с переобучением

## Итог

Regularization loss — это дополнительный штраф, добавляемый к функции потерь для ограничения сложности модели.
