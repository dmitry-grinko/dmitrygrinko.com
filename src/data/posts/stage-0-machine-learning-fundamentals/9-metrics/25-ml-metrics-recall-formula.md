# 25. Как вычисляется recall?

Recall вычисляется как отношение **правильно найденных положительных объектов** к **общему числу реальных положительных объектов**.

## Формула

recall = TP / (TP + FN)

где:

TP — true positives  
FN — false negatives

## Что означает формула

TP — сколько положительных объектов модель обнаружила.

FN — сколько положительных объектов модель пропустила.

Recall показывает долю обнаруженных положительных случаев.

## Итог

Recall измеряет полноту обнаружения положительного класса.
