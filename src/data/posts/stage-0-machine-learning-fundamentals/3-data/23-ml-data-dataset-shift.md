# 23. Что такое dataset shift?

Dataset shift — это ситуация, когда **распределение данных меняется** между training dataset и новыми данными.

Это означает, что данные, на которых модель работает после обучения, отличаются от данных, на которых она обучалась.

## Пример

Представим модель, обученную на данных интернет-магазина за прошлый год.

Если поведение пользователей изменилось, новые данные могут выглядеть иначе.

Например:

- изменился спрос  
- появились новые типы товаров  
- изменилось поведение клиентов

Это и есть dataset shift.

## Почему это важно

Модель машинного обучения предполагает, что данные в будущем будут похожи на training data.

Если это предположение нарушается, качество модели может снизиться.

## Итог

Dataset shift — это изменение распределения данных между обучением и использованием модели.
