# 21. Что такое параметры модели?

Параметры модели — это значения, которые модель **обучает на данных**.

В линейной регрессии параметрами являются:

- коэффициенты при признаках  
- intercept (bias)

Именно эти значения определяют, как модель делает предсказание.

## Пример

Модель:

price = 2000 × area + 5000 × rooms + 10000

Параметры модели:

- 2000  
- 5000  
- 10000  

Во время обучения алгоритм пытается найти такие значения параметров, которые минимизируют ошибку.

## Почему параметры важны

Если параметры выбраны правильно:

- модель хорошо описывает данные  
- ошибка предсказания уменьшается

Если параметры выбраны плохо, модель будет давать неточные predictions.

## Итог

Параметры модели — это значения коэффициентов и bias, которые определяют поведение модели и настраиваются во время обучения.
