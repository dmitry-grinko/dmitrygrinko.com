# 34. Почему gradient descent используется чаще при больших данных?

При большом количестве данных аналитическое решение становится вычислительно дорогим.

Gradient descent решает эту проблему.

## Масштабируемость

Gradient descent не требует вычисления обратной матрицы.

Он работает через постепенное обновление параметров.

Это позволяет обучать модель даже на очень больших datasets.

## Работа с большими данными

Алгоритм можно применять к:

- миллионам объектов  
- тысячам признаков

Также существуют версии алгоритма:

- stochastic gradient descent (SGD)  
- mini-batch gradient descent

Они позволяют эффективно обучать модели на больших данных.

## Использование в современных системах

Большинство современных ML и deep learning алгоритмов используют именно gradient descent.

## Итог

Gradient descent используется чаще при больших данных, потому что он масштабируется значительно лучше аналитического решения.
