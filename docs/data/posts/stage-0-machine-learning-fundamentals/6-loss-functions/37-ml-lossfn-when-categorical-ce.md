# 37. Когда используется categorical cross entropy?

Categorical cross entropy используется в задачах **multiclass classification**.

Это означает, что объект может принадлежать одному из нескольких классов.

## Примеры

Классификация изображений

Распознавание рукописных цифр

Классификация текстов

## Типичная архитектура

Обычно categorical cross entropy используется вместе с:

softmax output layer.

Softmax превращает выход модели в вероятности классов.

## Итог

Categorical cross entropy применяется в задачах классификации с несколькими классами.
