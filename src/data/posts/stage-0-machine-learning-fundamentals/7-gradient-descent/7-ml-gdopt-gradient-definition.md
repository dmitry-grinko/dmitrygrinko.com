# 7. Что такое градиент функции?

Градиент — это вектор производных функции по всем её параметрам.

Он показывает, **как функция изменяется при изменении параметров**.

## Интуитивная идея

Представим поверхность функции.

Градиент указывает направление, в котором функция растёт быстрее всего.

![gradient direction](image.png)

## Формально

Если функция зависит от параметров:

L(w₁, w₂, ..., wₙ)

то градиент — это вектор:

∇L = (∂L/∂w₁, ∂L/∂w₂, ..., ∂L/∂wₙ)

## Что это означает

Каждая компонента показывает, как изменение параметра влияет на значение функции.

## Итог

Градиент показывает направление изменения функции в пространстве параметров.
