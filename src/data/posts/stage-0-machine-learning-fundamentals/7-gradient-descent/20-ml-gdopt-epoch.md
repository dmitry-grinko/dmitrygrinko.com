# 20. Что такое epoch?2

Epoch — это полный проход по всему dataset во время обучения.

## Пример

Если dataset содержит 10 000 примеров, один epoch означает, что модель обработала все 10 000 объектов.

## Связь с batch

Если batch size = 100

то один epoch будет содержать:

10 000 / 100 = 100 итераций.

## Почему используют несколько epoch

Обычно одного прохода по данным недостаточно.

Модель обучается несколько раз на одном и том же dataset.

## Итог

Epoch — это один полный проход модели по всему dataset.
