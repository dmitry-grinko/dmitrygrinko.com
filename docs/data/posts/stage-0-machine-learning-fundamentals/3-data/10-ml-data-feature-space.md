# 10. Что такое feature space?

Feature space — это пространство всех возможных значений признаков.

Каждый объект dataset можно представить как точку в этом пространстве.

## Пример

Представим dataset с двумя признаками:

- возраст  
- доход

Каждый пользователь можно представить точкой на графике:

x → возраст  
y → доход

Таким образом формируется двумерное пространство признаков.

## Больше признаков

Если dataset содержит больше признаков, пространство становится многомерным.

Например:

- 10 признаков → 10-мерное пространство  
- 100 признаков → 100-мерное пространство

Такое пространство называется **feature space**.

## Роль feature space

Модель машинного обучения фактически работает в этом пространстве.

Она пытается:

- найти границы между классами  
- определить закономерности  
- построить функцию предсказания

## Итог

Feature space — это пространство всех признаков, в котором каждый объект dataset представлен как точка.
