# 18. Что происходит если выбрать неподходящую loss функцию?

Если выбрать неправильную loss функцию, модель может обучаться плохо или давать неправильные результаты.

## Возможные проблемы

### Нестабильное обучение

Градиенты могут вести модель в неправильном направлении.

### Плохие предсказания

Модель оптимизирует неправильную цель.

### Медленное обучение

Оптимизация становится менее эффективной.

## Пример

Использование MSE для классификации часто приводит к более медленному обучению и худшим результатам.

## Итог

Неподходящая loss функция может значительно ухудшить качество модели.
