# 48. Когда линейная регрессия работает плохо?

Линейная регрессия плохо работает в ситуациях, когда её основные предположения нарушаются.

## Нелинейные зависимости

Если связь между признаками и target сильно нелинейная, линейная модель не сможет её описать.

## Сложные структуры данных

В задачах с:

- сложными взаимодействиями признаков  
- нелинейными эффектами  

линейная регрессия может быть слишком простой.

## Большое количество выбросов

Если dataset содержит много выбросов, модель может строить неправильную зависимость.

## Слабые признаки

Если признаки плохо объясняют target, линейная регрессия будет иметь высокую ошибку.

## Итог

Линейная регрессия работает плохо, когда зависимости между переменными сложные или данные содержат сильный шум.
