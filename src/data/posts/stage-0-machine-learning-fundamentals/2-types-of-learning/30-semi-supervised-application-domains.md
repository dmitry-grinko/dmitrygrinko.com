# 30. В каких областях применяется semi-supervised learning?

Semi-supervised learning применяется в задачах, где данных много, но их разметка требует значительных ресурсов.

Такие ситуации встречаются во многих областях machine learning.

## Computer vision

В задачах компьютерного зрения обычно доступно большое количество изображений, но их разметка требует ручной работы.

Например:

- разметка объектов на изображениях  
- сегментация изображений  
- классификация изображений

Использование semi-supervised методов позволяет обучать модели на небольшом размеченном dataset и большом наборе неразмеченных изображений.

## Обработка текста

В NLP часто доступно огромное количество текстов из интернета, но только небольшая часть из них размечена.

Semi-supervised learning может использоваться для:

- классификации текстов  
- анализа тональности  
- извлечения информации

## Анализ пользовательских данных

Многие онлайн-системы собирают большое количество данных о поведении пользователей.

Но разметка таких данных может быть ограничена.

Semi-supervised learning помогает использовать эти данные для:

- рекомендаций  
- сегментации пользователей  
- анализа поведения

## Итог

Semi-supervised learning применяется в областях, где данные легко собрать, но сложно разметить, например:

- computer vision  
- обработка текста  
- анализ пользовательских данных
