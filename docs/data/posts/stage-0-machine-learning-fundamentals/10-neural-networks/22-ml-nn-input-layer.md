# 22. Что такое входной слой (input layer)?

Входной слой (input layer) — это первый слой нейронной сети, который получает исходные данные.

Он не «думает» и не извлекает признаки в полном смысле слова. Его задача проще: **принять входные значения и передать их дальше в сеть**.

## Что находится во входном слое

Количество элементов входного слоя определяется количеством входных признаков.

### Пример с таблицей

Если объект описывается такими признаками:

- возраст  
- доход  
- количество покупок  

то входной слой будет иметь 3 входа.

### Пример с изображением

Если изображение имеет размер 28 × 28 пикселей, то сеть может получить:

28 × 28 = 784 входных значения

Значит, входной слой будет содержать 784 входа.

## Важная деталь

Часто говорят «нейроны входного слоя», но технически входной слой обычно не выполняет те же вычисления, что скрытые нейроны. Он просто хранит и передаёт значения признаков.

## Что происходит дальше

Значения из input layer поступают в первый скрытый слой, где уже начинаются вычисления:

- умножение на веса  
- добавление bias  
- применение активации

## Итог

Input layer — это часть сети, которая принимает исходные данные и передаёт их в следующие слои.
