# 26. Как learning rate влияет на обучение?

Learning rate напрямую влияет на то, **как быстро и стабильно обучается модель**.

Он контролирует величину изменения параметров на каждом шаге gradient descent.

## Большой learning rate

Если learning rate слишком большой:

- параметры меняются слишком резко  
- алгоритм может перескакивать минимум функции  
- обучение становится нестабильным

## Маленький learning rate

Если learning rate маленький:

- обучение происходит очень медленно  
- алгоритм требует больше итераций

## Баланс

Хороший learning rate позволяет:

- быстро уменьшать ошибку  
- стабильно приближаться к минимуму функции

## Итог

Learning rate определяет скорость обучения и влияет на стабильность оптимизации.
