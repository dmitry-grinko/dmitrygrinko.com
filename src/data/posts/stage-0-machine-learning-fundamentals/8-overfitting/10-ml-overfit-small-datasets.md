# 10. Почему маленькие datasets чаще приводят к overfitting?

Маленькие datasets чаще приводят к overfitting, потому что модель получает **слишком мало информации о реальной структуре данных**.

## Ограниченное число примеров

Когда данных мало, каждый пример имеет большой вес в процессе обучения.

Модель может начать подстраиваться под конкретные наблюдения.

## Шум становится заметнее

В маленьком dataset случайные отклонения выглядят как закономерности.

Модель может попытаться объяснить их.

## Недостаток разнообразия

Dataset может не отражать всю вариативность реальных данных.

Из-за этого модель плохо работает на новых объектах.

## Итог

Маленькие datasets увеличивают риск переобучения, потому что модель видит слишком мало примеров для обучения обобщённым закономерностям.
