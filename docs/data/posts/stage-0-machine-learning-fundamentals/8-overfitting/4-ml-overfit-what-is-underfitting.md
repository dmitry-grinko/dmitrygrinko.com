# 4. Что означает underfitting?

Underfitting — это ситуация, когда модель **слишком простая для описания закономерностей данных**.

В этом случае модель не может хорошо обучиться даже на training dataset.

## Что происходит

Модель пытается найти зависимость между признаками и целевой переменной.

Но её структура слишком ограничена, чтобы описать реальную зависимость.

## Пример

Если данные имеют сложную нелинейную зависимость, а используется простая линейная модель, она не сможет правильно описать данные.

## Признаки underfitting

- высокая training error  
- высокая validation error  
- модель делает очень грубые предсказания

## Итог

Underfitting возникает, когда модель слишком простая и не может выучить структуру данных.
