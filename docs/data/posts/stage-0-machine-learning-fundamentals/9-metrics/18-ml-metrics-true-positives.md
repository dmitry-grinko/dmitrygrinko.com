# 18. Что такое true positives?

True positives (TP) — это случаи, когда **модель правильно предсказала положительный класс**.

## Пример

Задача обнаружения мошенничества.

Если транзакция действительно является мошеннической и модель тоже предсказала мошенничество, это true positive.

## Значение

True positives показывают, сколько реальных положительных случаев модель смогла обнаружить.

## Почему это важно

Во многих задачах важно находить как можно больше положительных случаев.

## Итог

True positives — это правильные предсказания положительного класса.
