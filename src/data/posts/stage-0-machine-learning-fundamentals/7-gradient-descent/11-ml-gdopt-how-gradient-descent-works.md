# 11. Как работает gradient descent?

Gradient descent работает как итерационный алгоритм, который постепенно изменяет параметры модели, уменьшая значение функции потерь.

На каждой итерации выполняются три шага:

1. вычисляется функция потерь  
2. вычисляется градиент функции потерь по параметрам  
3. параметры изменяются в направлении уменьшения функции

Главная идея состоит в том, что градиент показывает направление наибольшего роста функции.  
Поэтому, если двигаться **в противоположном направлении**, можно уменьшать значение функции.

## Интуитивная картина

Представим поверхность функции потерь как холмистый ландшафт.

Gradient descent ведёт себя как человек, который спускается с горы, всегда выбирая направление самого крутого спуска.

## Итог

Gradient descent постепенно уменьшает функцию потерь, шаг за шагом корректируя параметры модели.
