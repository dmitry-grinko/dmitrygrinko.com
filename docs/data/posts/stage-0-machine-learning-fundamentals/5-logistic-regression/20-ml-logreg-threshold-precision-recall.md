# 20. Как threshold влияет на precision и recall?

Threshold напрямую влияет на поведение классификатора.

Изменяя его, можно изменить баланс между precision и recall.

## Уменьшение threshold

Если threshold уменьшить:

- модель будет чаще предсказывать положительный класс
- recall увеличится
- precision может снизиться

## Увеличение threshold

Если threshold увеличить:

- модель станет более осторожной
- precision увеличится
- recall может уменьшиться

## Почему возникает этот компромисс

Более низкий порог позволяет обнаружить больше положительных объектов, но увеличивает количество ложных срабатываний.

## Итог

Threshold управляет балансом между precision и recall.
