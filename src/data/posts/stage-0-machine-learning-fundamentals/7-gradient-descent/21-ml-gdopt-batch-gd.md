# 21. Что такое batch gradient descent?

Batch gradient descent — это вариант алгоритма gradient descent, в котором градиент вычисляется **по всему dataset сразу**.

Это означает, что для обновления параметров используются **все примеры обучающих данных**.

## Как работает алгоритм

На одной итерации происходит следующее:

1. модель делает предсказания для всех объектов dataset  
2. вычисляется функция потерь для всех примеров  
3. вычисляется средний градиент по всему dataset  
4. параметры модели обновляются

## Особенность

Каждое обновление параметров основано на **точном градиенте функции потерь**.

Это делает направление обновления стабильным.

## Ограничения

Если dataset очень большой, вычисление градиента по всем данным становится медленным.

## Итог

Batch gradient descent — это алгоритм, который обновляет параметры модели, используя градиент, вычисленный по всему dataset.
