# 39. Что такое one-hot encoding?

One-hot encoding — это способ преобразования категориального признака в набор бинарных признаков, чтобы модель машинного обучения могла работать с такими данными.

Многие модели машинного обучения работают только с числовыми значениями. Поэтому категориальные признаки, такие как тип устройства или город, нужно сначала преобразовать в числовой формат.

One-hot encoding делает это следующим образом: каждая категория превращается в отдельную колонку, которая может принимать значение 0 или 1.

## Пример

Представим dataset с признаком device.

| device |
|------|
| mobile |
| desktop |
| mobile |

После one-hot encoding данные будут выглядеть так:

| device_mobile | device_desktop |
|--------------|---------------|
| 1 | 0 |
| 0 | 1 |
| 1 | 0 |

Здесь:

- значение **1** означает, что объект относится к данной категории
- значение **0** означает, что категория отсутствует

Каждая строка имеет единицу только в той колонке, которая соответствует её категории.

## Почему используется one-hot encoding

Этот метод решает важную проблему. Если просто присвоить категориям числа (например, mobile = 1, desktop = 2), модель может интерпретировать эти значения как имеющие порядок.

One-hot encoding устраняет эту проблему, потому что категории становятся независимыми бинарными признаками.

## Ограничение метода

Если признак содержит очень много категорий, one-hot encoding может создать большое количество новых колонок.

Например, если признак `city` содержит 1000 уникальных значений, после кодирования появится 1000 новых признаков.

Это может увеличить размер dataset и усложнить обучение модели.

## Итог

One-hot encoding — это способ преобразования категориальных признаков в набор бинарных признаков, который позволяет моделям машинного обучения корректно работать с категориальными данными.
