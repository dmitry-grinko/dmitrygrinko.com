# 3. В каких задачах machine learning применяется чаще всего?

Machine learning используют там, где **сложно написать точные правила**, но есть много данных, из которых можно извлечь закономерности.

Чаще всего это несколько типов задач.

## Распознавание объектов

Когда системе нужно понять, **что находится в данных**.

Примеры:

- распознавание объектов на изображениях
- распознавание лиц
- анализ медицинских снимков
- распознавание речи

Во всех этих задачах невозможно описать набор точных правил вроде `if pixel == ...`.

## Обработка текста

Machine learning активно используется для работы с естественным языком.

Типичные задачи:

- перевод текста
- поиск информации
- анализ тональности текста
- классификация документов
- генерация текста

Текст слишком вариативен, поэтому rule-based системы быстро перестают работать.

## Рекомендательные системы

Это одна из самых распространённых задач ML.

Система пытается предсказать, **что пользователю может понравиться**.

Примеры:

- рекомендации фильмов
- рекомендации товаров
- подбор музыки
- рекомендации видео

Алгоритмы используют историю действий пользователей и ищут похожие паттерны поведения.

## Прогнозирование

Machine learning часто применяют для **предсказания будущих событий**.

Например:

- прогноз спроса
- прогноз цен
- прогноз оттока клиентов
- прогноз загрузки систем

Такие задачи обычно формулируются как регрессия или классификация.

## Поиск аномалий

Задача — обнаружить **нетипичное поведение** в данных.

Примеры:

- мошеннические транзакции
- подозрительная активность в сети
- дефекты на производстве
- ошибки в системах

Такие события редко имеют явные правила, поэтому ML помогает выявлять статистические отклонения.

## Итог

Machine learning чаще всего применяется в задачах, где:

- входные данные сложные (текст, изображения, поведение пользователей)
- правила трудно сформулировать вручную
- доступно большое количество данных

Если задачу можно решить простым алгоритмом, ML обычно не нужен.
