# 37. Что такое categorical features?

Categorical features — это признаки, значения которых представляют **категории**, а не числовые величины.

Такие признаки описывают тип объекта, а не количество.

## Примеры categorical признаков

Например:

| city | device | payment_type |
|------|--------|--------------|
| London | mobile | card |
| Paris | desktop | cash |
| Berlin | mobile | card |

Здесь:

- `city` — город
- `device` — тип устройства
- `payment_type` — способ оплаты

Все эти признаки являются категориальными, потому что их значения — это **категории**, а не числа.

## Почему это важно

Большинство моделей машинного обучения работают с **числовыми данными**.

Поэтому categorical features обычно нужно преобразовать в числовой формат перед обучением модели.

## Распространённые способы преобразования

Чаще всего используют:

- **one-hot encoding** — каждая категория превращается в отдельный бинарный признак
- **label encoding** — каждой категории присваивается числовой индекс

Например, признак device:

| device |
|------|
| mobile |
| desktop |

может быть преобразован в:

| device_mobile | device_desktop |
|---------------|---------------|
| 1 | 0 |
| 0 | 1 |

## Итог

Categorical features — это признаки, значения которых представляют категории.
Перед обучением модели такие признаки обычно преобразуют в числовой формат.
