# 33. Как вычисляется градиент функции потерь?

Градиент функции потерь показывает, **как нужно изменить параметры модели**, чтобы уменьшить ошибку.

Он вычисляется как производная функции потерь по каждому параметру модели.

## Основная идея

Градиент показывает направление самого быстрого роста функции.

Поэтому параметры изменяются в **противоположном направлении**, чтобы уменьшить loss.

## Что происходит на практике

Для каждого параметра модели вычисляется:

∂Loss / ∂w

Это показывает, насколько изменение параметра влияет на ошибку.

## Использование

После вычисления градиента параметры модели обновляются.

## Итог

Градиент показывает направление изменения параметров, которое уменьшает функцию потерь.
