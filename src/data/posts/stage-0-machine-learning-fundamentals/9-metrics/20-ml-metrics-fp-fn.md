# 20. Что такое false positives и false negatives?

False positives и false negatives — это два типа ошибок классификации.

## False Positive (FP)

Модель предсказала положительный класс, но на самом деле объект относится к отрицательному.

Пример: нормальная транзакция помечена как мошенничество.

## False Negative (FN)

Модель предсказала отрицательный класс, но объект на самом деле положительный.

Пример: мошенническая транзакция не была обнаружена.

## Почему это важно

В разных задачах эти ошибки имеют разную стоимость.

Например:

- в медицине особенно опасны false negatives  
- в системах безопасности важны false positives

## Итог

False positives и false negatives — это два типа ошибок модели в задачах классификации.
