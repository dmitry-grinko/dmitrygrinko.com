# 17. Что такое holdout dataset?

Holdout dataset — это часть данных, которая **временно откладывается и не используется при обучении модели**.

Эта часть данных используется позже для оценки модели.

## Основная идея

Dataset делится на две части:

- training data  
- holdout data

Модель обучается только на training data.

После обучения модель проверяется на holdout dataset.

## Зачем это нужно

Holdout dataset помогает оценить, насколько хорошо модель работает на данных, которые она раньше не видела.

Это позволяет проверить способность модели **обобщать знания**.

## Пример

Если dataset содержит 10 000 примеров:

- 8 000 могут использоваться для обучения  
- 2 000 откладываются как holdout dataset

## Связь с test dataset

Во многих случаях holdout dataset фактически выполняет роль test dataset.

Термин "holdout" просто подчёркивает, что эти данные были заранее исключены из обучения.

## Итог

Holdout dataset — это часть данных, которая откладывается и используется для проверки модели после обучения.
