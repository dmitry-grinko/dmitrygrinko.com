# 12. Почему gradient descent позволяет находить минимум функции?

Gradient descent использует информацию о **локальной форме функции**, которая содержится в градиенте.

Градиент показывает:

- направление наибольшего роста функции  
- скорость изменения функции

Если двигаться **против градиента**, значение функции уменьшается.

## Повторяющийся процесс

Алгоритм выполняет множество шагов:

- вычисление градиента  
- обновление параметров  
- пересчёт функции потерь

Постепенно параметры модели приближаются к точке, где функция перестаёт уменьшаться.

## Что происходит в минимуме

В точке минимума градиент становится близким к нулю.

Это означает, что небольшие изменения параметров больше не уменьшают функцию.

## Итог

Gradient descent позволяет находить минимум функции, потому что использует градиент для движения в направлении уменьшения функции.
