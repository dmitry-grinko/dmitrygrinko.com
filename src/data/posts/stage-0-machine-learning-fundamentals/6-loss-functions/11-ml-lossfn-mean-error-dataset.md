# 11. Что такое средняя ошибка по dataset?

Средняя ошибка по dataset — это значение функции потерь, **усреднённое по всем примерам данных**.

Поскольку модель делает предсказания для множества объектов, необходимо агрегировать ошибки отдельных примеров в одно число.

## Как это выглядит

Сначала вычисляется ошибка для каждого объекта:

loss₁  
loss₂  
loss₃  
...  
lossₙ  

Затем они усредняются:

average loss = (loss₁ + loss₂ + ... + lossₙ) / n

где:

- n — количество примеров в dataset

## Зачем это нужно

Средняя ошибка позволяет:

- оценить качество модели на всём dataset  
- сравнивать разные модели  
- отслеживать прогресс обучения

## Итог

Средняя ошибка по dataset — это усреднённое значение функции потерь по всем примерам данных.
