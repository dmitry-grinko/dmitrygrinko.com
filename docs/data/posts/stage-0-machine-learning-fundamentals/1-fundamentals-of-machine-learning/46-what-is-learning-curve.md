# 46. Что такое learning curve?

Learning curve — это график, который показывает, как меняется качество модели во время обучения.

Обычно на таком графике отображается ошибка или метрика качества в зависимости от:

- количества итераций обучения
- количества обучающих данных

Learning curve помогает понять, как модель учится и есть ли проблемы с обучением.

## Как выглядит learning curve

Чаще всего строят две кривые:

- качество на **training data**
- качество на **validation data**

По мере обучения модели эти значения меняются.

Например:

- ошибка на training data постепенно уменьшается
- ошибка на validation data сначала уменьшается, а затем стабилизируется

## Зачем это нужно

Learning curve помогает увидеть:

- обучается ли модель вообще
- есть ли переобучение
- есть ли недообучение
- достаточно ли данных для обучения

## Пример интерпретации

**Переобучение**

- ошибка на training data очень низкая
- ошибка на validation data значительно выше

**Недообучение**

- ошибка высокая и на training, и на validation data

**Нормальное обучение**

- ошибки на training и validation постепенно уменьшаются и остаются близкими

## Итог

Learning curve — это график, который показывает, как меняется качество модели во время обучения и помогает анализировать процесс обучения модели.
