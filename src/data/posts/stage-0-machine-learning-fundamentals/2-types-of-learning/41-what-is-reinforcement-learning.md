# 41. Что такое reinforcement learning?

Reinforcement learning — это тип обучения в machine learning, при котором модель обучается через **взаимодействие со средой**.

Вместо готового dataset модель получает опыт, выполняя действия и наблюдая результаты этих действий.

## Основная идея

Модель (агент) находится в некоторой среде и может выполнять действия.

После каждого действия среда возвращает сигнал, который называется **reward (награда)**.

Цель модели — научиться выбирать такие действия, которые увеличивают суммарную награду.

## Как происходит обучение

Процесс обучения обычно выглядит так:

1. агент наблюдает состояние среды  
2. выбирает действие  
3. среда изменяется  
4. агент получает reward  
5. агент обновляет свою стратегию

Со временем агент учится принимать более выгодные решения.

## Пример

Представим систему, которая обучается играть в игру.

Агент может:

- двигаться  
- выполнять действия в игре

Если действие приводит к успеху, агент получает положительную награду.

Если действие приводит к проигрышу, награда может быть отрицательной.

## Итог

Reinforcement learning — это подход, при котором модель обучается принимать решения через взаимодействие со средой и получение награды.
