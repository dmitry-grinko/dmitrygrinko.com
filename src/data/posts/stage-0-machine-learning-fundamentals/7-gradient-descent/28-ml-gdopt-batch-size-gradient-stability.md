# 28. Как batch size влияет на стабильность градиентов?

Batch size влияет на то, **насколько точно оценивается градиент функции потерь**.

## Маленький batch

Градиент вычисляется на основе небольшого числа примеров.

Это делает оценку градиента шумной.

Направление обновления может сильно меняться между итерациями.

## Большой batch

Градиент вычисляется на основе большого количества данных.

Это делает оценку градиента более точной и стабильной.

## Следствие

Большие batch уменьшают вариацию градиентов.

Маленькие batch увеличивают шум.

## Итог

Batch size влияет на стабильность градиентов через точность оценки градиента.
