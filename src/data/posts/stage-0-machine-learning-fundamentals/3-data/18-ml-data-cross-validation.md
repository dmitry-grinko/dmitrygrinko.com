# 18. Что такое cross-validation?

Cross-validation — это метод оценки модели, при котором dataset используется несколько раз для обучения и тестирования.

Этот метод помогает получить более надёжную оценку качества модели.

## Основная идея

Dataset делится на несколько частей, которые называются **folds**.

Модель обучается несколько раз:

- каждый раз одна часть используется для тестирования  
- остальные части используются для обучения

## Пример

Если dataset разделён на 5 частей:

1. 4 части используются для обучения  
2. 1 часть используется для тестирования  
3. процесс повторяется 5 раз

Каждый раз тестовая часть меняется.

После этого результаты усредняются.

## Почему это полезно

Cross-validation позволяет:

- использовать dataset более эффективно  
- уменьшить случайные ошибки оценки  
- получить более стабильную метрику качества

## Итог

Cross-validation — это метод оценки модели, при котором dataset многократно разделяется на обучающие и тестовые части.
