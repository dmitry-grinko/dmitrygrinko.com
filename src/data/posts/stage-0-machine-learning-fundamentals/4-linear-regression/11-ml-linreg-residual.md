# 11. Что такое residual (ошибка предсказания)?

Residual — это разница между **реальным значением target** и **предсказанием модели**.

Он показывает, насколько модель ошиблась для конкретного наблюдения.

Формально residual вычисляется так:

residual = target − prediction

## Пример

Реальная цена квартиры:

price = 200000

Предсказание модели:

prediction = 185000

Residual:

200000 − 185000 = 15000

Это означает, что модель недооценила цену на 15000.

## Почему residual важен

Residual помогает понять:

- насколько точны предсказания модели  
- где модель ошибается  
- есть ли систематические ошибки

Анализ residual часто используется для диагностики модели.

## Итог

Residual — это ошибка предсказания модели для конкретного объекта.
