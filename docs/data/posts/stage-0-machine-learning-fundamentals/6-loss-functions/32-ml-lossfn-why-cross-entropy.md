# 32. Почему cross entropy используется в классификации?

Cross entropy используется в классификации, потому что она оценивает **качество вероятностных предсказаний**.

Многие модели классификации предсказывают вероятность принадлежности к классу.

Например:

P(class = 1)

Cross entropy измеряет, насколько эта вероятность соответствует реальному классу.

## Основное свойство

Функция:

- поощряет высокую вероятность правильного класса  
- наказывает высокую вероятность неправильного класса

## Следствие

Минимизация cross entropy приводит к улучшению вероятностных предсказаний модели.

## Итог

Cross entropy хорошо подходит для классификации, потому что она напрямую работает с вероятностями классов.
