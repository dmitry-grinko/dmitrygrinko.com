# 26. Что такое learning curves?

Learning curves — это графики, которые показывают, как меняется качество модели **в процессе обучения**.

Обычно по оси X откладывается количество эпох или размер training dataset.

По оси Y — ошибка модели или значение функции потерь.

![learning curves](image.png)

## Какие кривые строят

Обычно отображают две кривые:

- training error  
- validation error

## Что можно увидеть

Learning curves позволяют анализировать:

- процесс обучения  
- признаки переобучения  
- признаки недообучения

## Итог

Learning curves — это инструмент для анализа поведения модели во время обучения.
