# 50. Чем reinforcement learning принципиально отличается от других типов обучения?

Reinforcement learning отличается от других типов машинного обучения тем, что модель обучается **через взаимодействие со средой**, а не на фиксированном dataset.

## Отсутствие готового dataset

В supervised learning модель обучается на заранее подготовленном dataset.

В reinforcement learning данные появляются в процессе взаимодействия агента со средой.

## Последовательность решений

Во многих задачах reinforcement learning каждое действие влияет на будущие состояния системы.

Это означает, что агент должен учитывать долгосрочные последствия своих действий.

## Обучение через reward

Вместо правильного ответа агент получает только сигнал награды.

На основе этой награды он должен понять, какие действия являются наиболее эффективными.

## Динамическая среда

В reinforcement learning среда может постоянно изменяться.

Агент должен адаптироваться к этим изменениям.

## Итог

Reinforcement learning отличается тем, что модель обучается через взаимодействие со средой и принимает последовательные решения, ориентируясь на получаемую награду.
