# 18. Когда MAE лучше MSE?

MAE может быть предпочтительнее MSE в некоторых задачах.

Главная причина — различная чувствительность к большим ошибкам.

## Устойчивость к выбросам

MSE сильно увеличивает влияние больших ошибок, потому что они возводятся в квадрат.

MAE учитывает ошибки линейно.

Поэтому MAE более устойчив к выбросам.

## Интерпретация

MAE проще интерпретировать.

Например:

MAE = 5000

означает, что средняя ошибка модели составляет около 5000.

## Когда использовать MAE

MAE часто используют, когда:

- dataset содержит выбросы  
- большие ошибки не должны иметь слишком сильный вес  
- важна интерпретируемость метрики

## Итог

MAE лучше MSE в задачах, где важно уменьшить влияние выбросов.
