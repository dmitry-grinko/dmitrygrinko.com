# 45. Что такое Adam optimizer?

Adam (Adaptive Moment Estimation) — это алгоритм оптимизации, который сочетает идеи **momentum и адаптивного learning rate**.

Это один из самых популярных оптимизаторов в deep learning.

## Основная идея

Adam использует два типа информации:

1. среднее значение градиентов  
2. среднее значение квадратов градиентов

Эти величины помогают адаптировать обновления параметров.

## Почему это работает

Adam автоматически подбирает масштаб обновлений для каждого параметра.

Это делает обучение более стабильным.

## Итог

Adam — это оптимизатор, который объединяет momentum и адаптивные learning rate.
