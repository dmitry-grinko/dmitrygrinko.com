# 15. Почему ReLU широко используется в нейронных сетях?

ReLU стала стандартной функцией активации в deep learning.

## Причины

**Простота вычисления**

Функция очень проста:

max(0, x)

Это делает обучение быстрее.

**Хорошие свойства градиентов**

В положительной области градиент равен 1, что помогает эффективно обучать сеть.

**Стабильное обучение**

ReLU уменьшает проблему исчезающих градиентов по сравнению с sigmoid и tanh.

## Итог

ReLU широко используется благодаря простоте, скорости вычислений и хорошим свойствам для обучения.
