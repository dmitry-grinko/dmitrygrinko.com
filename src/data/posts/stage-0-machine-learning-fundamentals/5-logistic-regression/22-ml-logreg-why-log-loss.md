# 22. Почему log loss используется в logistic regression?

Logistic regression предсказывает **вероятности классов**.

Поэтому функция потерь должна оценивать, насколько предсказанные вероятности соответствуют реальным меткам.

Log loss делает именно это.

## Основная причина

Log loss сильно штрафует **уверенные неправильные предсказания**.

Это заставляет модель корректно оценивать вероятности.

## Пример

Если истинный класс:

y = 1

и модель предсказывает:

p = 0.99 → очень маленький loss  
p = 0.5 → средний loss  
p = 0.01 → очень большой loss

## Математическое удобство

Log loss является:

- гладкой функцией  
- дифференцируемой  

Это позволяет эффективно обучать модель с помощью gradient descent.

## Итог

Log loss используется в logistic regression, потому что он корректно измеряет ошибку вероятностных предсказаний.
