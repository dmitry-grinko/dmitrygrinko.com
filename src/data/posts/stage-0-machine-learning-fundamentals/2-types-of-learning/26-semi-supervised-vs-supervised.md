# 26. Чем semi-supervised learning отличается от supervised learning?

Главное отличие между этими подходами заключается в том, **какие данные используются для обучения модели**.

## Supervised learning

В supervised learning dataset полностью состоит из размеченных данных.

Каждый пример содержит:

- признаки (features)  
- правильный ответ (label)

Модель использует эти данные, чтобы научиться предсказывать результат.

## Semi-supervised learning

В semi-supervised learning dataset состоит из двух типов данных:

- небольшое количество размеченных примеров  
- большое количество неразмеченных данных

Модель использует размеченные данные для обучения, а неразмеченные — для улучшения понимания структуры dataset.

## Почему это важно

Semi-supervised learning позволяет обучать модели даже тогда, когда размеченных данных недостаточно.

Это делает такой подход полезным для многих реальных задач.

## Итог

Различие можно сформулировать так:

- supervised learning использует только размеченные данные  
- semi-supervised learning использует как размеченные, так и неразмеченные данные
