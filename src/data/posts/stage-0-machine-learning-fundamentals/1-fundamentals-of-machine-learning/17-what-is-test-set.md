# 17. Что такое test set?

Test set — это часть dataset, которая используется для финальной проверки качества модели.

Модель **не обучается** на этих данных и не настраивается с их помощью. Они используются только для того, чтобы понять, как модель будет работать на новых, ранее не виденных данных.

Обычно процесс выглядит так:

- training set — используется для обучения модели
- validation set — используется для настройки модели
- test set — используется для финальной оценки качества

Test set помогает получить **честную оценку модели**, потому что модель никогда не видела эти данные во время разработки.

## Итог

Test set — это набор данных, который используется только для финальной проверки того, насколько хорошо обученная модель работает на новых данных.
