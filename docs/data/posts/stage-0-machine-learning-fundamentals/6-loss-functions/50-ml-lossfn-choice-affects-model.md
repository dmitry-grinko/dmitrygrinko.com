# 50. Как выбор loss функции влияет на конечное поведение модели?

Loss function определяет **что именно модель считает ошибкой**.

Поэтому она напрямую влияет на поведение модели.

## Примеры

MSE

модель сильно избегает больших ошибок

MAE

модель становится устойчивой к выбросам

Cross entropy

модель оптимизирует вероятности классов

## Следствие

Разные функции потерь могут приводить к различным параметрам модели.

Даже если используется один и тот же алгоритм.

## Итог

Выбор функции потерь определяет, какие предсказания модель считает оптимальными.
