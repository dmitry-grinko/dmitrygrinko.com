# 47. Что такое data augmentation?

Data augmentation — это метод увеличения dataset за счёт создания новых данных на основе существующих.

Идея заключается в том, чтобы искусственно увеличить разнообразие данных.

## Как это работает

Новые примеры создаются путём преобразования существующих данных.

Например, изображение можно:

- повернуть  
- отразить  
- изменить яркость  
- немного обрезать

Такие преобразования создают новые варианты тех же данных.

## Почему это полезно

Data augmentation помогает:

- увеличить dataset  
- уменьшить переобучение  
- сделать модель более устойчивой

Модель начинает видеть больше различных примеров.

## Где используется

Data augmentation особенно часто применяется в:

- computer vision  
- обработке аудио  
- некоторых задачах NLP

## Итог

Data augmentation — это способ увеличить dataset, создавая новые примеры на основе существующих данных.
