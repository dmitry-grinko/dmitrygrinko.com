# 2. Чем метрика отличается от функции потерь (loss function)?

Метрика и функция потерь обе измеряют качество модели, но используются **для разных целей**.

## Функция потерь

Loss function используется **во время обучения модели**.

Алгоритм оптимизации минимизирует функцию потерь, изменяя параметры модели.

## Метрика

Метрика используется **для оценки качества модели**.

Она помогает понять, насколько хорошо модель решает задачу.

## Пример

В задаче классификации модель может обучаться с использованием cross entropy loss.

Но качество модели могут оценивать с помощью accuracy или F1 score.

## Итог

Loss function используется для обучения модели, а метрика — для оценки её качества.
