# 24. Что такое covariate shift?

Covariate shift — это тип dataset shift, при котором **изменяется распределение признаков (features)**, но связь между признаками и целевой переменной остаётся той же.

## Формально

Изменяется:

distribution(features)

но остаётся прежним:

relationship(features → target)

## Пример

Представим модель, которая предсказывает вероятность покупки.

Training data:

- пользователи из одного региона

Production data:

- пользователи из другого региона

Распределение признаков (возраст, доход, поведение) может измениться, но зависимость между ними и вероятностью покупки остаётся той же.

## Почему это проблема

Если распределение признаков сильно меняется, модель может оказаться в области feature space, которую она не видела при обучении.

Это может ухудшить качество предсказаний.

## Итог

Covariate shift — это изменение распределения признаков при сохранении зависимости между признаками и target.
