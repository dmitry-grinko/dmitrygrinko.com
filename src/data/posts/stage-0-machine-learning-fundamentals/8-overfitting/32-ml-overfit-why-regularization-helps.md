# 32. Почему регуляризация помогает бороться с переобучением?

Регуляризация помогает бороться с переобучением, потому что она **ограничивает гибкость модели**.

Без регуляризации модель может использовать очень большие веса и подгоняться под training data слишком точно.

## Что делает регуляризация

Она добавляет штраф за большие параметры модели.

Это заставляет алгоритм обучения искать решения, где:

- параметры меньше  
- модель проще  
- зависимость между признаками более гладкая

## Практический эффект

Модель перестаёт пытаться объяснить случайный шум в данных.

Вместо этого она начинает находить более устойчивые закономерности.

## Итог

Регуляризация уменьшает риск переобучения, ограничивая сложность модели.
