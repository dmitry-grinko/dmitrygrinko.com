# 8. Как выглядит формула logistic regression?

Logistic regression использует линейную комбинацию признаков, которая затем преобразуется sigmoid функцией.

Формула выглядит так:

```python
p = σ(w₁x₁ + w₂x₂ + ... + b)
```

где:

- x — признаки
- w — коэффициенты модели
- b — bias
- σ — sigmoid функция
- p — вероятность класса

## Шаги вычисления

1. вычисляется линейная комбинация признаков
2. результат передаётся в sigmoid функцию
3. sigmoid преобразует значение в вероятность

## Пример

```python
z = w₁x₁ + w₂x₂ + b  
p = sigmoid(z)
```

## Итог

Logistic regression сначала вычисляет линейную комбинацию признаков, а затем преобразует её в вероятность с помощью sigmoid функции.
