# 49. Какие практические методы чаще всего используют для борьбы с overfitting?

На практике используется несколько методов для уменьшения переобучения.

## Основные методы

Регуляризация

L1 и L2 штрафы на веса модели.

Dropout

Случайное отключение нейронов.

Early stopping

Остановка обучения при росте validation error.

Увеличение dataset

Больше данных уменьшает риск запоминания.

Упрощение модели

Уменьшение числа параметров.

## Итог

Борьба с переобучением обычно требует комбинации нескольких методов.
