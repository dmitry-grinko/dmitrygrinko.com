# 8. Почему baseline важен при оценке модели?

Baseline важен, потому что он показывает, **насколько сложная модель действительно лучше простого решения**.

Без baseline трудно понять, является ли результат хорошим.

## Пример

Если baseline accuracy = 80%

а новая модель показывает accuracy = 82%

улучшение может быть незначительным.

## Практическое значение

Baseline помогает:

- оценивать реальный прогресс  
- избегать иллюзии улучшения  
- проверять, стоит ли использовать сложную модель

## Итог

Baseline позволяет объективно оценить улучшение модели.
