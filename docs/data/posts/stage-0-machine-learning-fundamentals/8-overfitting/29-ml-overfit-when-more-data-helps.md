# 29. Когда добавление данных помогает модели?

Добавление данных помогает, когда модель уже способна учить закономерности, но **не видит достаточно примеров**.

## Типичная ситуация

Модель имеет:

- низкую training error  
- высокую validation error

Это означает переобучение.

## Почему новые данные помогают

Дополнительные данные:

- увеличивают разнообразие примеров  
- уменьшают влияние случайного шума  
- позволяют модели лучше понять структуру данных

## Ограничение

Если модель слишком простая, добавление данных может не помочь.

## Итог

Добавление данных помогает, когда проблема модели связана с переобучением.
