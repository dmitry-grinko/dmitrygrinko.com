# 28. Какие проблемы возникают при использовании semi-supervised learning?

Хотя semi-supervised learning позволяет использовать большое количество неразмеченных данных, этот подход также создаёт дополнительные сложности.

Основная проблема заключается в том, что неразмеченные данные могут **вводить модель в заблуждение**, если использовать их неправильно.

## Ошибки в псевдоразметке

Некоторые методы semi-supervised learning создают псевдо-метки для неразмеченных данных.

Если модель делает ошибочные предсказания, эти ошибки могут распространяться дальше и ухудшать обучение.

## Низкое качество неразмеченных данных

Неразмеченные datasets могут содержать:

- шум  
- выбросы  
- нерелевантные данные

Если такие данные активно используются в обучении, они могут ухудшить качество модели.

## Сложность алгоритмов

Методы semi-supervised learning часто сложнее, чем обычные supervised подходы.

Они могут требовать:

- дополнительных этапов обучения  
- более сложных моделей  
- более аккуратной настройки параметров

## Зависимость от структуры данных

Многие методы предполагают, что данные имеют определённую структуру.

Например, похожие объекты должны находиться близко друг к другу.

Если dataset не обладает такой структурой, эффективность методов может снижаться.

## Итог

Основные сложности semi-supervised learning связаны с:

- возможными ошибками в псевдоразметке  
- качеством неразмеченных данных  
- сложностью алгоритмов
