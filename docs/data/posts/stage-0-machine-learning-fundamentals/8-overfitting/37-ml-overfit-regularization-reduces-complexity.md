# 37. Почему регуляризация уменьшает сложность модели?

Сложность модели часто связана с величиной её параметров.

Если веса модели становятся очень большими, модель может создавать очень сложные зависимости.

## Что делает регуляризация

Регуляризация штрафует большие значения параметров.

Это заставляет модель выбирать более простые решения.

## Геометрическая интерпретация

Модель начинает предпочитать более гладкие функции вместо сложных и резких зависимостей.

## Практическое значение

Более простые модели обычно лучше обобщают новые данные.

## Итог

Регуляризация уменьшает сложность модели, ограничивая величину параметров.
