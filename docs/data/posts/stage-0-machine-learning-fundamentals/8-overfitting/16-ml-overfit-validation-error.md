# 16. Что такое validation error?

Validation error — это ошибка модели, вычисленная **на validation dataset**.

Validation dataset не используется для обучения модели.

Он используется только для оценки качества модели во время обучения.

## Зачем нужен validation dataset

Он позволяет понять, как модель работает на данных, которые она не видела.

Это помогает обнаружить переобучение.

## Когда вычисляется validation error

Обычно после каждой эпохи обучения.

## Что можно наблюдать

- уменьшение ошибки  
- стабилизацию ошибки  
- рост ошибки при переобучении

## Итог

Validation error показывает, насколько хорошо модель работает на новых данных во время обучения.
