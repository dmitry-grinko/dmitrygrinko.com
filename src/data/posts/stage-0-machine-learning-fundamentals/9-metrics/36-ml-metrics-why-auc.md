# 36. Почему AUC используется для оценки классификаторов?

AUC используется потому, что он оценивает **качество модели при всех возможных значениях threshold**.

Это делает метрику более устойчивой.

## Преимущества

AUC:

- не зависит от конкретного порога классификации  
- учитывает весь диапазон работы модели  
- показывает способность модели разделять классы

## Практическое значение

Это особенно полезно, когда threshold ещё не выбран.

## Итог

AUC оценивает способность модели различать классы независимо от выбранного порога.
