# 41. Что такое baseline модель?

Baseline модель — это простая модель, которая используется как начальная точка для оценки качества более сложных моделей.

Она задаёт **минимальный уровень качества**, с которым сравнивают все последующие модели.

Если новая модель не показывает результат лучше baseline, значит её использование не имеет смысла.

## Пример

Представим задачу классификации, где модель должна определить, является ли письмо спамом.

Самый простой baseline может быть таким:

всегда предсказывать самый частый класс

Если в dataset:

- 80% писем — обычные
- 20% — спам

то baseline модель может просто всегда отвечать:

not spam

В этом случае accuracy baseline модели будет 80%.

Любая более сложная модель должна показывать результат **лучше этого значения**, иначе она не добавляет пользы.

## Почему baseline важен

Baseline помогает:

- понять, насколько сложная модель действительно улучшает результат
- быстро получить первое приближение решения
- обнаружить ошибки в данных или модели

Иногда простая baseline модель работает почти так же хорошо, как сложные алгоритмы.

## Итог

Baseline модель — это простая модель, которая используется как точка отсчёта для оценки качества других моделей.
