# 9. Как нейрон вычисляет своё значение?

Нейрон вычисляет своё значение в два шага.

## Шаг 1 — линейная комбинация

Сначала вычисляется взвешенная сумма входов.

z = w₁x₁ + w₂x₂ + ... + wₙxₙ + b

## Шаг 2 — функция активации

Затем результат передаётся в функцию активации.

y = activation(z)

Функция активации преобразует результат в выходное значение нейрона.

## Итог

Нейрон сначала вычисляет линейную комбинацию входов, а затем применяет функцию активации.
