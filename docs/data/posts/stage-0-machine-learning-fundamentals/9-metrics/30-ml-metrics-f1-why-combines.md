# 30. Почему F1 score объединяет precision и recall?

F1 score объединяет precision и recall, потому что в многих задачах **важно учитывать оба типа ошибок**.

Использование только одной метрики может давать неполную картину.

## Проблема одной метрики

Высокая precision может сопровождаться низким recall.

Высокий recall может сопровождаться низкой precision.

## Решение

F1 score объединяет обе метрики в одну величину.

Это позволяет оценить баланс между точностью и полнотой.

## Практическое применение

F1 score часто используется в задачах:

- информационного поиска  
- обнаружения аномалий  
- анализа текста

## Итог

F1 score объединяет precision и recall, чтобы оценивать общий баланс качества классификации.
