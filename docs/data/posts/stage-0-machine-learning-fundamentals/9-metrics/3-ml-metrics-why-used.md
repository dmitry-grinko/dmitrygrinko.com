# 3. Почему метрики используются для оценки модели?

Метрики используются для оценки модели, потому что они дают **объективный способ измерить качество предсказаний**.

Без метрик невозможно понять, насколько хорошо работает модель.

## Что позволяют делать метрики

Метрики позволяют:

- сравнивать разные модели  
- выбирать лучшую модель  
- отслеживать прогресс обучения

## Пример

Если одна модель имеет accuracy 90%, а другая 85%, можно сделать вывод, что первая модель работает лучше.

## Практическое значение

Метрики превращают оценку модели из субъективной в количественную.

## Итог

Метрики используются для объективной оценки качества модели.
