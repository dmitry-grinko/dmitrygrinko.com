# 27. Как размер batch влияет на обучение модели?

Batch size определяет количество примеров, используемых для одного обновления параметров.

Он влияет на несколько аспектов обучения.

## Маленький batch

Если batch size маленький:

- градиенты становятся шумными  
- обновления происходят чаще  
- обучение может быть менее стабильным

## Большой batch

Если batch size большой:

- градиенты становятся более точными  
- обучение становится стабильнее  
- обновления происходят реже

## Компромисс

Обычно выбирают batch size, который обеспечивает баланс между:

- скоростью обучения  
- стабильностью градиентов

## Итог

Размер batch влияет на частоту обновлений и точность вычисления градиентов.
