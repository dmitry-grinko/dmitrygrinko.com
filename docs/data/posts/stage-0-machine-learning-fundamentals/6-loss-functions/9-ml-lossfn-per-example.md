# 9. Почему loss функция должна быть вычислима для каждого примера?

Loss функция должна быть вычислима для каждого примера, потому что обучение модели происходит на отдельных наблюдениях dataset.

## Основная идея

Для каждого объекта можно вычислить ошибку предсказания.

Эти ошибки затем объединяются, чтобы получить общую ошибку модели.

## Пример

Dataset:

| x | y |
|---|---|
| ... | ... |

Для каждого объекта вычисляется:

loss_i = loss(prediction_i, target_i)

## После этого

Все ошибки объединяются:

- суммируются  
- или усредняются

## Итог

Loss функция должна быть определена для каждого примера, чтобы можно было вычислить общую ошибку модели.
