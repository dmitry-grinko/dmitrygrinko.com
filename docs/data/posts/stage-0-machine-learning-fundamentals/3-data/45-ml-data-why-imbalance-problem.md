# 45. Почему дисбаланс классов может быть проблемой?

Дисбаланс классов может существенно влиять на обучение модели.

Главная проблема заключается в том, что модель может игнорировать редкий класс.

## Смещение модели

Если один класс встречается намного чаще, модель может начать предсказывать только этот класс.

Например:

Dataset:

- 99% normal  
- 1% fraud

Модель может предсказывать "normal" для всех объектов.

Accuracy при этом будет 99%, но модель не обнаружит ни одного случая мошенничества.

## Неправильные метрики

При сильном дисбалансе accuracy перестаёт быть хорошей метрикой.

В таких задачах обычно используют:

- precision  
- recall  
- F1-score

## Реальные последствия

В задачах обнаружения редких событий ошибка может быть критичной.

Например:

- пропущенное мошенничество  
- пропущенная болезнь

## Итог

Class imbalance может привести к тому, что модель будет игнорировать редкие, но важные случаи.
