# 34. Что происходит с параметрами модели при обучении?

Во время обучения параметры модели постепенно изменяются.

Цель — найти такие значения коэффициентов, которые минимизируют log loss.

## Процесс

На каждой итерации происходит следующее:

1. вычисляется ошибка модели  
2. вычисляется градиент  
3. параметры корректируются

## Изменение коэффициентов

Если параметр увеличивает ошибку, алгоритм уменьшает его значение.

Если параметр помогает уменьшить ошибку, его значение может увеличиваться.

## Постепенное улучшение

Со временем модель начинает:

- лучше разделять классы  
- давать более точные вероятности

## Итог

Параметры модели постепенно корректируются так, чтобы уменьшить функцию потерь.
