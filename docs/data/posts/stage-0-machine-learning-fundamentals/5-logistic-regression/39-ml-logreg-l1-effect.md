# 39. Как L1 регуляризация влияет на модель?

L1 регуляризация добавляет к функции потерь **штраф за абсолютные значения коэффициентов**.

## Формула штрафа

penalty = λ Σ |w|

где:

w — коэффициенты модели.

## Основной эффект

L1 регуляризация может сделать некоторые коэффициенты равными нулю.

Это означает, что соответствующие признаки фактически удаляются из модели.

## Практическое значение

Таким образом L1 выполняет одновременно:

- обучение модели  
- выбор признаков

## Итог

L1 регуляризация делает модель более разреженной и может автоматически исключать признаки.
