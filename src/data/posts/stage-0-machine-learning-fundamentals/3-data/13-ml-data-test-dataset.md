# 13. Что такое test dataset?

Test dataset — это часть данных, которая используется для **финальной оценки модели**.

Эти данные не участвуют ни в обучении модели, ни в её настройке.

Цель test dataset — показать, насколько хорошо модель будет работать на новых данных.

## Когда используется test dataset

Test dataset используется только после завершения обучения.

Процесс обычно выглядит так:

1. модель обучается на train dataset  
2. параметры настраиваются с помощью validation dataset  
3. финальная модель проверяется на test dataset

## Почему это важно

Если использовать test dataset во время обучения, результаты оценки будут искажены.

Модель может случайно подстроиться под test dataset, и тогда метрики будут слишком оптимистичными.

## Итог

Test dataset — это данные, которые используются для объективной финальной оценки модели.
