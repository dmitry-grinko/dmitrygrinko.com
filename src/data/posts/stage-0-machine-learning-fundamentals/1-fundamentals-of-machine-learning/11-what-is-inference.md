# 11. Что такое inference?

Inference — это процесс использования обученной модели для получения предсказаний на новых данных.

Другими словами, inference — это момент, когда модель уже обучена и применяется для решения реальной задачи.

Общая схема выглядит так:

```
features → model → prediction
```

Где:

- **features** — входные данные
- **model** — обученная модель
- **prediction** — результат работы модели

Сам процесс получения этого результата и называется **inference**.

## Пример

Представим модель, которая обучена предсказывать цену квартиры.

После обучения в систему поступают новые данные:

| area | rooms |
|-----|------|
| 70  | 2 |

Модель обрабатывает эти признаки и возвращает результат:

predicted price = 185000

Этот момент — когда модель принимает входные данные и выдаёт prediction — и есть inference.

## Где происходит inference

В реальных системах inference может выполняться в разных местах:

**В API**

Например, сервис рекомендаций получает запрос пользователя и возвращает список товаров.

**В batch-процессах**

Модель периодически обрабатывает большие объёмы данных, например прогнозирует спрос на товары.

**В real-time системах**

Например:

- распознавание речи
- обнаружение мошенничества
- рекомендации в ленте

## Важная деталь

Во время inference модель **не обучается**.

Она только использует уже найденные параметры для вычисления предсказаний.

Обучение и inference — это два разных этапа работы ML-системы.

## Итог

Inference — это процесс применения обученной модели к новым данным для получения предсказаний.
