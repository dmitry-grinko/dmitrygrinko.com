# 21. Что такое precision?

Precision — это метрика классификации, которая показывает **какая доля предсказанных положительных объектов действительно является положительной**.

Она отвечает на вопрос:

> Когда модель говорит "положительный класс", насколько часто она права?

## Формула

precision = TP / (TP + FP)

где:

TP — true positives  
FP — false positives

## Интерпретация

Высокая precision означает, что модель редко делает ложные срабатывания.

## Пример

Если модель обнаружила 100 случаев мошенничества, но только 80 из них действительно мошенничество:

precision = 80 / 100 = 0.8

## Итог

Precision показывает точность положительных предсказаний модели.
