# 5. Почему logistic regression предсказывает вероятность?

Logistic regression предсказывает вероятность, потому что её математическая модель ограничивает результат диапазоном от 0 до 1.

Это достигается с помощью специальной функции — sigmoid.

## Проблема линейной модели

Если использовать обычную линейную функцию:

y = w₁x₁ + w₂x₂ + b

значение y может быть любым числом.

Например:

-3  
15  
100

Но вероятность должна находиться в диапазоне:

0 — 1

## Решение

Линейная комбинация признаков пропускается через sigmoid функцию.

Sigmoid преобразует любое число в значение между 0 и 1.

## Результат

После этого значение можно интерпретировать как вероятность.

## Итог

Logistic regression предсказывает вероятность, потому что её выход ограничен диапазоном 0–1 с помощью sigmoid функции.
