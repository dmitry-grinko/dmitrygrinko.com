# 7. Что такое веса (weights) нейрона?

Weights — это параметры нейрона, которые определяют **насколько сильно каждый вход влияет на результат**.

Каждый вход имеет свой вес.

## Что делает вес

Вес масштабирует входное значение.

Если вес большой, соответствующий вход сильнее влияет на результат.

Если вес близок к нулю, вход почти не влияет.

## Формула

z = w₁x₁ + w₂x₂ + ... + wₙxₙ

## Обучение

Во время обучения алгоритм изменяет веса, чтобы улучшить предсказания модели.

## Итог

Weights — это параметры нейрона, которые определяют важность входных сигналов.
