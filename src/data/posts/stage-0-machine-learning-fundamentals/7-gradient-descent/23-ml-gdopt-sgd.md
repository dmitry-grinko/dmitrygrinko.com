# 23. Что такое stochastic gradient descent (SGD)?

Stochastic gradient descent (SGD) — это алгоритм оптимизации, в котором параметры модели обновляются **после обработки одного примера данных**.

В отличие от batch gradient descent, градиент вычисляется не по всему dataset, а только по одному объекту.

## Как работает SGD

Для каждого примера dataset выполняются шаги:

1. вычисляется prediction  
2. вычисляется loss для этого примера  
3. вычисляется градиент  
4. параметры модели обновляются

После этого алгоритм переходит к следующему примеру.

## Особенность

Градиенты становятся шумными, потому что они вычисляются на основе одного наблюдения.

Однако это позволяет быстрее выполнять обновления.

## Итог

SGD — это метод оптимизации, который обновляет параметры модели после обработки каждого отдельного примера.
