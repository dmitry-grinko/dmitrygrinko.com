# 5. Как выбор метрики зависит от задачи?

Метрика должна отражать **то, что действительно важно в конкретной задаче**.

Разные задачи требуют разных критериев качества.

## Классификация

Для задач классификации часто используют:

- accuracy  
- precision  
- recall  
- F1 score

## Регрессия

Для задач регрессии используются:

- Mean Squared Error  
- Mean Absolute Error  
- RMSE

## Специализированные задачи

В некоторых задачах важны специальные метрики.

Например:

- AUC для бинарной классификации  
- BLEU для машинного перевода

## Итог

Метрика выбирается в зависимости от типа задачи и того, что считается хорошим результатом.
