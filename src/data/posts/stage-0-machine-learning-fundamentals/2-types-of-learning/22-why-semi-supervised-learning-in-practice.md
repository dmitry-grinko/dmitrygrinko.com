# 22. Почему semi-supervised learning используется на практике?

В реальных ML-проектах часто возникает ситуация, когда данных очень много, но размеченных примеров мало.

Это происходит потому, что сбор данных обычно автоматизирован, а **разметка данных требует участия человека**.

Semi-supervised learning позволяет использовать эту ситуацию более эффективно.

## Разметка данных может быть дорогой

Во многих задачах создание размеченного dataset требует значительных ресурсов.

Например:

- медицинские изображения должны размечаться врачами  
- тексты требуют ручной классификации  
- видео и изображения требуют долгой обработки

В результате компания может иметь огромное количество неразмеченных данных, но ограниченное количество размеченных примеров.

## Большие datasets без разметки

Современные системы часто генерируют огромные объемы данных.

Например:

- пользовательские логи  
- изображения  
- тексты  
- видео

Эти данные легко собрать, но сложно разметить.

Semi-supervised learning позволяет использовать такие datasets.

## Улучшение качества модели

Неразмеченные данные могут содержать важную информацию о структуре dataset.

Используя их, модель может:

- лучше понять распределение данных  
- улучшить обобщающую способность  
- повысить качество предсказаний

## Итог

Semi-supervised learning используется на практике, потому что он позволяет эффективно использовать большое количество неразмеченных данных при ограниченном количестве размеченных примеров.
