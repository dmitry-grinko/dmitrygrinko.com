# 31. Что такое self-supervised learning?

Self-supervised learning — это подход к обучению моделей, при котором **обучающие сигналы создаются автоматически из самих данных**.

В этом случае модель не требует ручной разметки. Вместо этого она получает задачу, которая формируется на основе структуры dataset.

## Основная идея

Алгоритм создаёт специальную задачу обучения, используя сами данные.

Например:

- часть данных скрывается  
- модель должна восстановить эту часть

Таким образом данные сами становятся источником обучения.

## Пример

В текстовых моделях часть слова или предложения может быть скрыта.

Задача модели — предсказать пропущенный фрагмент.

В изображениях модель может получать изображение с закрытой областью и пытаться восстановить её.

## Чем self-supervised learning отличается от обычного unsupervised learning

В self-supervised learning всё же существует **чёткая задача предсказания**, просто она создаётся автоматически.

Поэтому модель может обучаться с использованием методов, похожих на supervised learning.

## Итог

Self-supervised learning — это подход, при котором модель обучается на задачах, автоматически сформированных из самих данных.
