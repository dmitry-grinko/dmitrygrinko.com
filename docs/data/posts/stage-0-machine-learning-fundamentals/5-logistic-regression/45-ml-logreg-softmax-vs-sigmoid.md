# 45. Чем softmax отличается от sigmoid?

Sigmoid и softmax — это функции, которые преобразуют значения модели в вероятности.

Но они используются в разных задачах.

## Sigmoid

Sigmoid используется в бинарной классификации.

Она выдаёт одну вероятность:

P(class = 1)

## Softmax

Softmax используется в multiclass задачах.

Она выдаёт **вероятности для всех классов одновременно**.

## Пример

Sigmoid:

P(spam) = 0.82

Softmax:

P(cat) = 0.6  
P(dog) = 0.3  
P(horse) = 0.1

## Итог

Sigmoid используется для двух классов, а softmax — для нескольких классов.
