# 11. Что такое generalization?

Generalization — это способность модели **делать точные предсказания на новых данных**, которые она не видела во время обучения.

Именно это является главной целью машинного обучения.

Модель не должна просто запоминать training dataset.  
Она должна находить закономерности, которые работают и для новых примеров.

## Пример

Представим модель, которая предсказывает цену квартиры.

Если модель просто запомнила цены квартир из training dataset, она будет работать только для этих объектов.

Но если модель выучила реальные закономерности:

- влияние площади  
- влияние района  
- влияние года постройки  

то она сможет предсказывать цены и для новых квартир.

## Итог

Generalization — это способность модели применять выученные закономерности к новым данным.
