# 21. Что такое слой (layer) в нейронной сети?

Слой (layer) — это группа нейронов, которые получают одни и те же входные данные, выполняют над ними вычисления и передают результат дальше.

Проще говоря, слой — это **этап преобразования данных** внутри сети.

Если нейронная сеть состоит из нескольких слоёв, это значит, что данные проходят через несколько последовательных преобразований:

входные данные → слой 1 → слой 2 → ... → выход

## Что делает слой

Каждый нейрон в слое:

1. получает входные значения  
2. умножает их на свои веса  
3. добавляет bias  
4. применяет функцию активации  

Все нейроны слоя делают это параллельно.

## Пример

Допустим, у нас есть данные о квартире:

- площадь = 70  
- комнат = 2  
- этаж = 5  

Это три входных признака.

Если первый скрытый слой содержит 4 нейрона, то каждый из этих 4 нейронов получает все три признака и строит своё собственное представление данных.

То есть один и тот же вход преобразуется сразу несколькими способами.

## Почему слои важны

Один слой может извлекать простые зависимости.

Следующий слой может использовать результат предыдущего и строить более сложные признаки.

Именно поэтому нейронные сети могут постепенно переходить:

- от сырых данных  
- к промежуточным представлениям  
- к итоговому предсказанию

## Итог

Слой — это набор нейронов, который преобразует входные данные в новое представление.  
Нейронная сеть состоит из последовательности таких преобразований.
