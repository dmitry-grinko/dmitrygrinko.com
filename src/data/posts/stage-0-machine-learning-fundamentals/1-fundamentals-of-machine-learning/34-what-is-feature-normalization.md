# 34. Что такое normalization признаков?

Normalization признаков — это преобразование числовых признаков к общему масштабу.

В реальных datasets разные признаки могут иметь сильно отличающиеся диапазоны значений. Например:

| feature | значение |
|-------|--------|
| возраст | 18–70 |
| доход | 20000–200000 |
| расстояние до центра | 0–50 |

Если такие признаки подавать в модель напрямую, признаки с большими значениями могут начать сильнее влиять на обучение.

Normalization приводит признаки к сопоставимому масштабу, чтобы модель могла работать с ними более корректно.

## Как работает normalization

Один из самых распространённых способов — **Min-Max normalization**.

Значение преобразуется по формуле:

x_normalized = (x − min) / (max − min)

Где:

- `x` — исходное значение
- `min` — минимальное значение признака
- `max` — максимальное значение признака

В результате все значения оказываются в диапазоне **от 0 до 1**.

## Пример

Исходные значения:

| age |
|----|
| 20 |
| 40 |
| 60 |

Здесь:

min = 20
max = 60

Посчитаем normalization.

Для 20:

(20 − 20) / (60 − 20) = 0

Для 40:

(40 − 20) / (60 − 20) = 0.5

Для 60:

(60 − 20) / (60 − 20) = 1

После normalization:

| age |
|----|
| 0.0 |
| 0.5 |
| 1.0 |

## Пример кода

В Python normalization часто делают с помощью библиотеки scikit-learn.

Например:

```python
from sklearn.preprocessing import MinMaxScaler
import numpy as np

X = np.array([[20], [40], [60]])

scaler = MinMaxScaler()

X_scaled = scaler.fit_transform(X)

print(X_scaled)
```

Здесь:

- `MinMaxScaler` вычисляет минимальное и максимальное значение признака
- затем применяет формулу normalization ко всем значениям

## Когда normalization важна

Normalization особенно важна для алгоритмов, чувствительных к масштабу признаков:

- линейная регрессия
- логистическая регрессия
- нейронные сети
- методы на основе расстояний

## Итог

Normalization — это преобразование признаков к одному масштабу.
Она помогает модели корректно учитывать разные признаки во время обучения.
