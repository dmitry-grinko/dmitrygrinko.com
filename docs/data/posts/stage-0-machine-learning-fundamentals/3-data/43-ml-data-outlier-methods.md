# 43. Какие методы обработки outliers существуют?

После обнаружения выбросов необходимо решить, как с ними работать.

Существует несколько подходов.

## Удаление выбросов

Если выбросы являются результатом ошибок данных, их можно удалить.

Например:

- ошибки ввода  
- повреждённые записи  
- невозможные значения

## Ограничение значений (clipping)

Иногда значения ограничивают определённым диапазоном.

Например:

если значения слишком большие, их можно заменить максимальным допустимым значением.

## Преобразование данных

Некоторые преобразования могут уменьшить влияние выбросов.

Например:

- логарифмическое преобразование  
- scaling  
- нормализация

## Использование устойчивых моделей

Некоторые алгоритмы менее чувствительны к выбросам.

Например:

- деревья решений  
- Random Forest  
- Gradient Boosting

## Итог

Обработка выбросов может включать:

- удаление  
- ограничение значений  
- преобразование данных  
- использование устойчивых моделей
