# 6. Что такое training data в supervised learning?

Training data — это набор данных, на котором обучается модель.

В supervised learning training dataset содержит **примеры с известными правильными ответами**. Именно эти данные позволяют модели понять, какие закономерности существуют между входными признаками и результатом.

Каждый пример обычно состоит из двух частей:

- **features** — входные признаки  
- **label** — правильный ответ

## Как выглядит training data

Например, в задаче предсказания цены квартиры dataset может выглядеть так:

| area | rooms | price |
|-----|------|------|
| 45 | 1 | 120000 |
| 70 | 2 | 185000 |
| 80 | 3 | 240000 |

В этом dataset:

- area и rooms — признаки  
- price — правильный ответ

Модель использует такие примеры, чтобы понять связь между характеристиками квартиры и её ценой.

## Как training data используется в обучении

Во время обучения модель:

1. получает признаки объекта  
2. делает предсказание  
3. сравнивает предсказание с правильным ответом  
4. корректирует параметры

Этот процесс повторяется для множества примеров из training dataset.

Чем больше разнообразных и качественных примеров содержит training data, тем лучше модель может научиться обобщать закономерности.

## Почему training data так важны

Качество модели напрямую зависит от training dataset.

Если training data:

- содержит ошибки  
- плохо отражает реальные данные  
- слишком маленький

то модель будет работать хуже, даже если используется хороший алгоритм.

## Итог

Training data — это набор размеченных примеров, на которых модель учится находить закономерности между признаками и правильными ответами.
