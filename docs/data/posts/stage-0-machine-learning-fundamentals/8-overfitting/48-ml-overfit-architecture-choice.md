# 48. Как выбор архитектуры влияет на переобучение?

Архитектура модели определяет её **сложность и количество параметров**.

Это напрямую влияет на риск переобучения.

## Слишком простая архитектура

Модель может быть неспособна выучить закономерности данных.

Это приводит к underfitting.

## Слишком сложная архитектура

Модель получает слишком много параметров.

Она может начать запоминать training dataset.

## Практический баланс

Хорошая архитектура должна быть достаточно мощной, чтобы выучить закономерности, но не чрезмерно сложной.

## Итог

Выбор архитектуры влияет на баланс между underfitting и overfitting.
