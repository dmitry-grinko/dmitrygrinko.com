# 34. Что такое false positive rate?

False Positive Rate (FPR) — это метрика, которая показывает **долю отрицательных объектов, ошибочно классифицированных как положительные**.

## Формула

FPR = FP / (FP + TN)

где:

FP — false positives  
TN — true negatives

## Что показывает FPR

FPR измеряет, насколько часто модель делает ложные срабатывания.

## Пример

Если среди 100 отрицательных объектов модель ошибочно классифицировала 5 как положительные:

FPR = 0.05

## Итог

False Positive Rate показывает вероятность ложного положительного предсказания.
