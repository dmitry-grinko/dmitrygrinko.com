# 23. Что такое gradient descent?

Gradient descent — это алгоритм оптимизации, который используется для **поиска параметров модели с минимальной ошибкой**.

Он применяется во многих алгоритмах машинного обучения.

## Основная идея

Представим поверхность функции потерь.

Задача алгоритма — найти **точку с минимальным значением функции**.

Gradient descent делает это постепенно.

На каждом шаге он немного изменяет параметры модели, двигаясь в сторону уменьшения ошибки.

## Интуитивная аналогия

Можно представить человека, который спускается с горы в тумане.

Он не видит весь ландшафт, но чувствует наклон поверхности.

Он делает шаг в сторону наибольшего спуска.

## Шаги алгоритма

1. вычисляется градиент функции потерь  
2. параметры модели изменяются в противоположном направлении градиента  
3. ошибка становится меньше  
4. процесс повторяется

## Итог

Gradient descent — это алгоритм, который постепенно изменяет параметры модели, чтобы уменьшить функцию потерь.
