# 4. Что означает минимизация функции потерь?

Минимизация функции потерь означает поиск таких параметров модели, при которых **ошибка предсказаний становится минимальной**.

## Основная идея

Алгоритм обучения изменяет параметры модели так, чтобы значение loss уменьшалось.

## Пример

Если ошибка модели сначала равна:

loss = 2.3

после нескольких итераций она может стать:

loss = 0.8

Это означает, что модель начала делать более точные предсказания.

## Оптимизация

Процесс минимизации обычно выполняется с помощью алгоритмов оптимизации, например:

- gradient descent

## Итог

Минимизация функции потерь — это процесс поиска параметров модели, которые дают наименьшую ошибку.
