# 35. Почему backpropagation используется для обучения сетей?

Backpropagation позволяет эффективно вычислять **градиенты функции потерь по всем параметрам сети**.

Без него обучение глубоких сетей было бы практически невозможно.

## Проблема без backpropagation

Нейронная сеть может иметь миллионы параметров.

Если пытаться изменять каждый параметр вручную или случайно, обучение будет крайне медленным.

## Что делает backpropagation

Он использует правило цепочки (chain rule) из математического анализа и вычисляет градиенты очень эффективно.

Это позволяет обучать большие сети за разумное время.

## Практическая важность

Практически все современные нейронные сети обучаются с использованием backpropagation.

## Итог

Backpropagation позволяет эффективно вычислять градиенты и обучать нейронные сети.
