# 18. Что такое bias-variance tradeoff?

Bias-variance tradeoff — это фундаментальный принцип machine learning, который описывает баланс между **ошибкой из-за упрощённой модели и ошибкой из-за слишком гибкой модели**.

## Два источника ошибки

Bias — ошибка из-за слишком простой модели.

Variance — ошибка из-за высокой чувствительности модели к данным.

## Конфликт

Если модель слишком простая:

возникает высокий bias.

Если модель слишком сложная:

возникает высокая variance.

## Цель обучения

Найти баланс между этими двумя типами ошибок.

## Итог

Bias-variance tradeoff описывает компромисс между недообучением и переобучением.
