# 37. Как self-supervised learning используется в computer vision?

Self-supervised learning также активно используется в задачах компьютерного зрения.

Как и в NLP, идея заключается в том, чтобы обучать модель на неразмеченных данных, создавая задачи автоматически.

## Обучение на изображениях

Модель получает изображение и должна решить некоторую вспомогательную задачу.

Например:

- восстановить скрытую часть изображения  
- определить правильное расположение фрагментов изображения  
- предсказать преобразование изображения

Такие задачи помогают модели понять структуру визуальных данных.

## Почему это полезно

Современные системы компьютерного зрения требуют огромных datasets.

Но ручная разметка изображений часто является дорогой и трудоёмкой задачей.

Self-supervised learning позволяет использовать большие коллекции изображений без необходимости полной разметки.

## Представления изображений

После такого обучения модель может извлекать полезные представления изображений.

Эти представления могут использоваться для:

- классификации изображений  
- обнаружения объектов  
- сегментации изображений

## Итог

В computer vision self-supervised learning помогает обучать модели на больших наборах изображений без ручной разметки.
