# 21. Что такое semi-supervised learning?

Semi-supervised learning — это подход в machine learning, который использует **как размеченные, так и неразмеченные данные**.

В реальных задачах часто возникает ситуация, когда небольшая часть dataset имеет правильные ответы, а большая часть данных остаётся без разметки.

Semi-supervised learning позволяет использовать оба типа данных.

## Основная идея

Алгоритм сначала использует размеченные данные, чтобы получить начальное представление о задаче.

Затем он использует неразмеченные данные, чтобы лучше понять структуру dataset и улучшить модель.

Таким образом модель получает больше информации, чем в обычном supervised learning.

## Почему это важно

Разметка данных часто требует:

- времени  
- человеческого труда  
- экспертных знаний

Поэтому размеченные datasets обычно гораздо меньше, чем доступные неразмеченные данные.

Semi-supervised learning позволяет использовать эту дополнительную информацию.

## Пример

Представим задачу классификации изображений.

Компания может иметь:

- 10 000 размеченных изображений  
- 1 000 000 неразмеченных изображений

Используя semi-supervised подход, модель может обучаться на размеченных данных и одновременно использовать структуру большого неразмеченного dataset.

## Итог

Semi-supervised learning — это подход, который комбинирует размеченные и неразмеченные данные для обучения моделей.
