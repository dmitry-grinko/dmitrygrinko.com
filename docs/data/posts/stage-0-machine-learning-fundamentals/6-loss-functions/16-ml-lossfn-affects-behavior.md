# 16. Как loss function влияет на поведение модели?

Loss function определяет **что именно модель считает ошибкой**.

Из-за этого она напрямую влияет на поведение модели.

## Пример

MSE сильно штрафует большие ошибки.

Поэтому модель старается избегать больших отклонений.

MAE штрафует ошибки линейно.

Из-за этого модель становится менее чувствительной к выбросам.

## Следствие

Выбор функции потерь может изменить:

- параметры модели  
- форму решения  
- устойчивость к шуму

## Итог

Loss function задаёт критерий оптимальности модели.
