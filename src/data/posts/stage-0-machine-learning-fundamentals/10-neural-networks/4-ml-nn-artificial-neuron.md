# 4. Что такое искусственный нейрон?

Искусственный нейрон — это базовый вычислительный элемент нейронной сети.

Он принимает несколько входных значений, выполняет математическое преобразование и выдаёт результат.

## Основная операция нейрона

Нейрон:

1. принимает входные значения  
2. умножает их на веса  
3. суммирует результат  
4. применяет функцию активации

## Формула

z = w₁x₁ + w₂x₂ + ... + b  
y = activation(z)

## Интерпретация

Нейрон можно рассматривать как маленькую модель, которая преобразует входные данные.

## Итог

Искусственный нейрон — это вычислительный блок, который преобразует входные данные с помощью весов и функции активации.
