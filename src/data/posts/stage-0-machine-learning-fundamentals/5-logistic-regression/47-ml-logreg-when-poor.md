# 47. Когда logistic regression работает плохо?

Logistic regression может работать плохо, если структура данных слишком сложная.

## Нелинейные зависимости

Если классы разделяются сложной нелинейной границей, линейная модель не сможет её описать.

## Сложные взаимодействия признаков

Если признаки взаимодействуют между собой сложным образом, модель может не уловить эти зависимости.

## Высокая размерность

При очень большом количестве признаков модель может переобучаться.

## Неподходящие признаки

Если признаки плохо отражают структуру данных, модель не сможет построить хорошую границу решений.

## Итог

Logistic regression плохо работает в задачах со сложными нелинейными зависимостями.
