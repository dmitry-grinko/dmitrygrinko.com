# 22. В каких задачах используется MSE?

Mean Squared Error чаще всего используется в **задачах регрессии**.

Такие задачи предполагают предсказание числового значения.

## Примеры

Предсказание цены квартиры

target = price

Предсказание температуры

target = temperature

Прогнозирование продаж

target = revenue

## Почему MSE подходит

MSE хорошо работает, когда важно сильно наказывать большие ошибки.

Например:

ошибка 10 → штраф 100  
ошибка 20 → штраф 400  

Это заставляет модель избегать больших отклонений.

## Итог

MSE широко используется в регрессионных задачах с числовыми предсказаниями.
