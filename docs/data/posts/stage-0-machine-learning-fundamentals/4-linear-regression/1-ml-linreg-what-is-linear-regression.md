# 1. Что такое linear regression?

Linear regression — это один из самых простых и базовых алгоритмов машинного обучения.  
Он используется для **предсказания числового значения** на основе нескольких признаков.

Основная идея очень простая: модель пытается найти **линейную зависимость между признаками и целевой переменной**.

Например, можно попытаться предсказать цену квартиры на основе:

- площади
- количества комнат
- возраста здания

Модель ищет такую формулу, которая лучше всего описывает связь между этими характеристиками и ценой.

## Интуитивная идея

Если есть только один признак, задача выглядит как поиск **лучшей прямой линии**, проходящей через данные.

Эта линия используется для предсказания значений для новых данных.

## Почему линейная регрессия важна

Linear regression — одна из самых фундаментальных моделей в машинном обучении.

Она важна по нескольким причинам:

- очень проста для понимания
- легко интерпретируется
- часто используется как baseline модель
- лежит в основе многих более сложных методов

## Итог

Linear regression — это модель, которая пытается описать зависимость между признаками и целевой переменной с помощью линейной функции.
