# 12. Что такое validation dataset?

Validation dataset — это часть данных, которая используется для **оценки модели во время обучения**.

Модель не обучается на validation dataset. Вместо этого он используется для проверки того, насколько хорошо модель работает на данных, которые она не видела при обучении.

## Зачем нужен validation dataset

Во время обучения модель может начать переобучаться.

Это означает, что она начинает запоминать training data вместо того, чтобы находить общие закономерности.

Validation dataset помогает обнаружить эту проблему.

## Как используется validation dataset

Обычно процесс выглядит так:

1. модель обучается на train dataset  
2. затем её проверяют на validation dataset  
3. оценивают метрики качества  
4. при необходимости меняют параметры модели

Этот процесс может повторяться много раз.

## Где используется validation dataset

Validation dataset используется для:

- настройки гиперпараметров  
- выбора модели  
- контроля переобучения

## Итог

Validation dataset — это данные, которые используются для проверки модели во время обучения, но не участвуют в обучении напрямую.
