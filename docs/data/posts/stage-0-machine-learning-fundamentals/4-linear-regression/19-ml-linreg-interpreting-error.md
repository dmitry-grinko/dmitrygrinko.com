# 19. Как интерпретировать значение ошибки модели?

Значение ошибки показывает, **насколько сильно предсказания модели отличаются от реальных значений**.

Интерпретация зависит от используемой метрики.

## Пример RMSE

Если модель предсказывает цены квартир и:

RMSE = 12000

это означает, что средняя ошибка модели примерно равна 12000.

## Относительный масштаб ошибки

Важно сравнивать ошибку с масштабом target.

Например:

если цены квартир:

100000 – 500000

ошибка 12000 может быть вполне приемлемой.

Если target:

100 – 200

та же ошибка будет очень большой.

## Сравнение моделей

Ошибку часто используют для сравнения моделей.

Модель с меньшей ошибкой обычно считается лучше.

## Итог

Ошибка модели показывает среднее отклонение предсказаний от реальных значений.
