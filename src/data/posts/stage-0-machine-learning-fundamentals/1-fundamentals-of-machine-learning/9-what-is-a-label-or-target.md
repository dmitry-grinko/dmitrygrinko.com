# 9. Что такое label (target)?

Label (или target) — это правильный ответ, который модель должна научиться предсказывать.

В задачах машинного обучения данные обычно состоят из двух частей:

features → label

Где:

- **features** — входные признаки объекта  
- **label / target** — значение, которое нужно предсказать  

Например, если модель предсказывает цену квартиры:

| area | rooms | price |
|-----|------|------|
| 45  | 1    | 120000 |
| 70  | 2    | 185000 |

Здесь:

- `area`, `rooms` — features  
- `price` — label (target)

Во время обучения модель получает:

- признаки объекта  
- правильный ответ

Алгоритм сравнивает предсказание модели с label и постепенно настраивает параметры модели так, чтобы ошибка уменьшалась.

## Тип label зависит от задачи

Label может иметь разный тип в зависимости от задачи:

**Классификация**

Label — это категория.

Примеры:

- spam / not spam  
- cat / dog  
- positive / negative  

**Регрессия**

Label — числовое значение.

Примеры:

- цена  
- температура  
- вероятность события  

## Итог

Label (target) — это правильный ответ в обучающих данных, который показывает модели, какой результат она должна научиться предсказывать.
