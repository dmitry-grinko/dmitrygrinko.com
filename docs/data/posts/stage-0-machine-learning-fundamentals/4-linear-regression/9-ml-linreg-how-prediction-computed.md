# 9. Как linear regression вычисляет предсказание?

Linear regression вычисляет предсказание как **линейную комбинацию признаков**.

Каждый признак умножается на свой коэффициент, после чего результаты складываются.

## Формула

y = w₁x₁ + w₂x₂ + ... + b

где:

- x — признаки  
- w — коэффициенты  
- b — bias  
- y — prediction

## Пример

Модель:

price = 2000 × area + 5000 × rooms + 10000

Для квартиры:

area = 60  
rooms = 2

получаем:

price = 2000 × 60 + 5000 × 2 + 10000  
price = 140000

## Что происходит внутри модели

Фактически модель выполняет простые математические операции:

- умножение
- сложение

Однако коэффициенты подбираются во время обучения.

## Итог

Linear regression вычисляет prediction как сумму произведений признаков и коэффициентов плюс bias.
