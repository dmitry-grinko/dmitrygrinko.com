# 17. Когда sigmoid используется в нейронных сетях?

Сегодня sigmoid редко используется во внутренних слоях нейронных сетей.

Но она всё ещё применяется в **выходных слоях**.

## Основные случаи

Sigmoid используется:

- в бинарной классификации
- в logistic regression
- когда нужно предсказать вероятность события

## Пример

Модель может выдавать вероятность того, что изображение содержит определённый объект.

## Итог

Sigmoid чаще всего используется в выходных слоях моделей бинарной классификации.
