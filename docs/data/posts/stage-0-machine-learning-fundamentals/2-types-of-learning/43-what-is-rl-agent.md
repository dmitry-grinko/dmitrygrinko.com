# 43. Что такое agent в reinforcement learning?

Agent — это компонент reinforcement learning системы, который принимает решения.

Именно агент взаимодействует со средой и выбирает действия на основе текущего состояния.

Главная задача агента — научиться принимать такие решения, которые приводят к максимальной суммарной награде.

## Роль агента

Агент выполняет несколько основных функций:

- наблюдает состояние среды  
- выбирает действие  
- получает reward  
- корректирует свою стратегию

Постепенно агент учится принимать более эффективные решения.

## Пример

Представим систему, которая обучается играть в игру.

Агент может выполнять различные действия:

- двигаться  
- атаковать  
- защищаться  

После каждого действия среда сообщает, насколько это действие было успешным.

Со временем агент начинает понимать, какие действия приводят к лучшему результату.

## Итог

Agent — это модель, которая взаимодействует со средой и принимает решения в reinforcement learning системе.
