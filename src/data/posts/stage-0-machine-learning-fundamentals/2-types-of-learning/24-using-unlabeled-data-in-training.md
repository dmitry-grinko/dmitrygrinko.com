# 24. Как можно использовать неразмеченные данные для обучения?

Неразмеченные данные могут быть полезны даже тогда, когда в dataset нет правильных ответов.

Существуют методы, которые позволяют использовать такие данные для улучшения модели.

## Обучение структуры данных

Даже без labels dataset может содержать важную информацию о структуре данных.

Например, модель может:

- находить группы похожих объектов  
- изучать распределение данных  
- выявлять скрытые закономерности

Эта информация может помочь модели лучше понимать dataset.

## Предварительное обучение модели

Иногда модель сначала обучают на неразмеченных данных, чтобы она научилась представлять данные в удобной форме.

После этого модель дообучается на небольшом размеченном dataset.

Такой подход часто используется в deep learning.

## Генерация псевдоразметки

В некоторых методах модель сначала обучается на размеченных данных, а затем начинает самостоятельно генерировать **псевдо-метки** для неразмеченных данных.

Эти новые примеры могут использоваться для дальнейшего обучения.

## Итог

Неразмеченные данные могут использоваться для:

- изучения структуры dataset  
- предварительного обучения моделей  
- генерации дополнительных обучающих примеров
