# 25. Как информация проходит через слои сети?

Информация проходит через сеть **последовательно**, слой за слоем.

Каждый слой получает значения от предыдущего слоя, преобразует их и передаёт дальше.

## Общая схема

входные данные → первый слой → второй слой → ... → выходной слой

## Что делает каждый слой

Каждый слой:

1. принимает вход  
2. вычисляет линейную комбинацию  
3. применяет функцию активации  
4. передаёт результат следующему слою

## Пример

Допустим, сеть получает три входа:

- x₁ = 70  
- x₂ = 2  
- x₃ = 5  

Первый скрытый слой преобразует эти числа в новый набор значений, например:

- h₁ = 0.8  
- h₂ = 0.1  
- h₃ = 0.6  
- h₄ = 0.3  

Второй скрытый слой получает уже эти значения и снова преобразует их.

Потом output layer строит итоговый ответ.

## Важная идея

Каждый следующий слой работает не с исходными данными, а с **представлением**, созданным предыдущим слоем.

Именно поэтому нейросеть может постепенно строить всё более сложные абстракции.

## Итог

Информация проходит через сеть поэтапно: каждый слой преобразует результат предыдущего слоя и передаёт его дальше.
