# 42. Почему регуляризация добавляется к функции потерь?

Регуляризация добавляется к функции потерь, чтобы контролировать **сложность модели**.

Без регуляризации модель может слишком точно подгоняться под training data.

Это приводит к переобучению.

## Основная идея

Регуляризация вводит штраф за большие значения параметров.

Таким образом модель ищет баланс между:

- точностью на training data  
- простотой модели

## Следствие

Модель становится менее чувствительной к шуму в данных.

## Итог

Регуляризация помогает строить более устойчивые модели.
