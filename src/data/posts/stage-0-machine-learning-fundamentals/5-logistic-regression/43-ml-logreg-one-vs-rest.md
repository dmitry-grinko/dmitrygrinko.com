# 43. Что такое one-vs-rest подход?

One-vs-rest (или one-vs-all) — это способ использовать бинарную классификацию для задач с несколькими классами.

## Основная идея

Для каждого класса обучается **отдельная бинарная модель**.

Каждая модель отвечает на вопрос:

"Этот объект принадлежит этому классу или нет?"

## Пример

Если есть три класса:

- cat  
- dog  
- horse  

обучаются три модели:

cat vs rest  
dog vs rest  
horse vs rest

## Предсказание

Каждая модель выдаёт вероятность своего класса.

Выбирается класс с наибольшей вероятностью.

## Итог

One-vs-rest превращает multiclass задачу в несколько бинарных задач классификации.
