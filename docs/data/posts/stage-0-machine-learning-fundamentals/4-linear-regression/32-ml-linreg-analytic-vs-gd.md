# 32. Чем аналитическое решение отличается от gradient descent?

Аналитическое решение и gradient descent решают одну и ту же задачу — поиск параметров модели.

Но делают это по-разному.

## Аналитическое решение

Normal equation вычисляет параметры **непосредственно через формулу**.

Характеристики метода:

- не требует итераций  
- даёт точное решение  
- использует матричные операции

## Gradient descent

Gradient descent ищет параметры **постепенно**, шаг за шагом.

Алгоритм:

1. делает предсказание  
2. вычисляет ошибку  
3. корректирует параметры

Этот процесс повторяется много раз.

## Основное различие

Normal equation:

- одно вычисление  
- аналитическое решение

Gradient descent:

- итерационный процесс  
- приближение к минимуму функции потерь

## Итог

Normal equation находит параметры напрямую, а gradient descent делает это постепенно через оптимизацию функции потерь.
