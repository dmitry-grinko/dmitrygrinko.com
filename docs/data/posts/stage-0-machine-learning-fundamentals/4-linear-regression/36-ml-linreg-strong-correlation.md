# 36. Что происходит при сильной корреляции признаков?

Иногда признаки в dataset могут быть сильно связаны друг с другом.

Это означает, что один признак почти полностью объясняет другой.

Такую ситуацию называют **корреляцией признаков**.

## Пример

Представим dataset с двумя признаками:

- площадь квартиры в квадратных метрах  
- площадь квартиры в квадратных футах

Эти признаки почти полностью дублируют друг друга.

## Проблема для линейной регрессии

Когда признаки сильно коррелируют:

- коэффициенты модели становятся нестабильными  
- модель может присваивать странные значения параметрам

Это усложняет интерпретацию модели.

## Что происходит на практике

Модель может:

- увеличивать один коэффициент  
- уменьшать другой

при этом итоговое предсказание остаётся тем же.

## Итог

Сильная корреляция признаков делает параметры модели нестабильными и усложняет интерпретацию результатов.
