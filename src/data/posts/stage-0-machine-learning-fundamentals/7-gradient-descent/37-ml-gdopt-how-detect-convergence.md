# 37. Как понять, что обучение модели сошлось?

Существует несколько признаков того, что обучение модели сошлось.

## Стабилизация функции потерь

Loss перестаёт заметно уменьшаться между итерациями.

## Малые градиенты

Градиенты становятся близкими к нулю.

## Стабильные параметры

Параметры модели перестают сильно изменяться.

## Метрики на validation данных

Метрики качества перестают улучшаться.

## Итог

Обучение считается сошедшимся, когда функция потерь и параметры модели стабилизируются.
