# 2. Зачем нужна функция потерь?

Функция потерь нужна для того, чтобы **обучать модель**.

Она показывает, насколько текущие параметры модели дают правильные предсказания.

## Основная идея

Во время обучения модель:

1. делает предсказание  
2. вычисляет loss  
3. изменяет параметры так, чтобы уменьшить loss

Этот процесс повторяется много раз.

## Что происходит

Если функция потерь уменьшается, значит модель начинает делать более точные предсказания.

Если loss увеличивается, модель работает хуже.

## Роль функции потерь

Loss функция задаёт **цель обучения модели**.

Алгоритм обучения пытается найти параметры, при которых loss минимален.

## Итог

Функция потерь нужна для того, чтобы направлять процесс обучения модели.
