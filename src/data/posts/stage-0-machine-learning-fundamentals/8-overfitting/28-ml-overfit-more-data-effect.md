# 28. Как увеличение данных влияет на переобучение?

Увеличение dataset часто уменьшает риск переобучения.

Это происходит потому, что модель получает больше примеров реальных закономерностей.

## Почему это работает

Когда данных больше:

- случайный шум усредняется  
- закономерности становятся более очевидными  
- модель реже запоминает отдельные наблюдения

## Особенно важно для сложных моделей

Большие модели требуют большого количества данных.

Без достаточного dataset они легко переобучаются.

## Итог

Увеличение данных помогает уменьшить переобучение и улучшить generalization модели.
