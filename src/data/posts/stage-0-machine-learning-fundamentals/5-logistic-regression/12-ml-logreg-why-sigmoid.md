# 12. Почему sigmoid используется в logistic regression?

Sigmoid используется в logistic regression потому, что она преобразует любое число в **вероятность**.

Линейная модель может выдавать значения в любом диапазоне:

-10  
5  
100  

Но вероятность должна находиться между:

0 и 1.

## Решение

Sigmoid преобразует линейный выход модели в значение между 0 и 1.

Это позволяет интерпретировать результат как вероятность класса.

## Дополнительное свойство

Sigmoid является **гладкой и дифференцируемой функцией**.

Это важно для обучения модели с помощью gradient descent.

## Итог

Sigmoid используется в logistic regression, потому что она преобразует линейный выход модели в вероятность.
