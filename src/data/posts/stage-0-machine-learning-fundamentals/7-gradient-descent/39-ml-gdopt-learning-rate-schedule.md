# 39. Что такое learning rate schedule?

Learning rate schedule — это стратегия изменения learning rate **во время обучения модели**.

Вместо постоянного значения learning rate постепенно уменьшается или изменяется по определённому правилу.

## Зачем это нужно

На ранних этапах обучения полезно делать большие шаги.

Позже более маленькие шаги позволяют точнее приблизиться к минимуму функции.

## Примеры стратегий

Step decay  
Exponential decay  
Cosine schedule

## Результат

Learning rate schedule помогает ускорить обучение и повысить стабильность оптимизации.

## Итог

Learning rate schedule — это способ изменять learning rate в процессе обучения.
