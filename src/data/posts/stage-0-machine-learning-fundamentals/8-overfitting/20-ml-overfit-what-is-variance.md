# 20. Что такое variance в модели?

Variance — это ошибка, возникающая из-за **слишком высокой чувствительности модели к training data**.

Модель с высокой variance сильно зависит от конкретных примеров из dataset.

## Что происходит

Небольшие изменения training dataset могут сильно изменить параметры модели.

## Признаки

- очень низкая training error  
- высокая validation error

## Связь с переобучением

Высокая variance обычно означает переобучение.

## Итог

Variance — это ошибка, возникающая из-за чрезмерной гибкости модели.
