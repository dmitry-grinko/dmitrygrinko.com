# 36. Что такое categorical cross entropy?

Categorical cross entropy — это функция потерь для задач **multiclass classification**.

Она используется, когда существует более двух классов.

## Пример

Классификация изображений:

- cat  
- dog  
- horse  
- bird

## Основная идея

Модель предсказывает вероятность для каждого класса.

Loss вычисляется на основе вероятности правильного класса.

## Пример

Если правильный класс — dog:

P(cat) = 0.1  
P(dog) = 0.7  
P(horse) = 0.1  
P(bird) = 0.1  

Loss зависит от значения 0.7.

## Итог

Categorical cross entropy используется для задач классификации с несколькими классами.
