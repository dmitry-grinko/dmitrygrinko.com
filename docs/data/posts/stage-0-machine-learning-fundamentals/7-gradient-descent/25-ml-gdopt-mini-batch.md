# 25. Что такое mini-batch gradient descent?

Mini-batch gradient descent — это алгоритм оптимизации, в котором параметры модели обновляются **после обработки небольшой группы примеров**.

Эта группа называется batch.

## Основная идея

Mini-batch является компромиссом между:

- batch gradient descent  
- stochastic gradient descent

## Как работает алгоритм

Dataset делится на небольшие batch.

Для каждого batch выполняется:

1. вычисление предсказаний  
2. вычисление функции потерь  
3. вычисление среднего градиента  
4. обновление параметров

## Пример

Если batch size = 32

каждое обновление параметров использует 32 примера.

## Итог

Mini-batch gradient descent обновляет параметры модели после обработки небольшой группы данных.
