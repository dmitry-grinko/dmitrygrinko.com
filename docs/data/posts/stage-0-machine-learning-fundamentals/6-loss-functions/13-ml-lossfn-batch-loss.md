# 13. Что такое batch loss?

Batch loss — это значение функции потерь, вычисленное **для одного batch данных**.

Batch — это небольшая часть dataset, используемая на одной итерации обучения.

## Процесс

Во время одной итерации:

1. модель делает предсказания для batch  
2. для каждого примера вычисляется loss  
3. ошибки усредняются

Результат и называется batch loss.

## Пример

Batch содержит 32 примера.

Для каждого вычисляется loss.

Затем они усредняются:

batch_loss = average(loss₁ … loss₃₂)

## Зачем это используется

Batch loss используется для:

- вычисления градиентов  
- обновления параметров модели

## Итог

Batch loss — это средняя ошибка модели на одном batch данных.
