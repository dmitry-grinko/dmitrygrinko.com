# 26. В каких задачах важен высокий recall?

Высокий recall важен в задачах, где **пропуск положительного случая может быть критическим**.

## Примеры

### Медицинская диагностика

Важно обнаружить как можно больше случаев болезни.

### Обнаружение мошенничества

Пропуск мошеннической операции может привести к финансовым потерям.

### Системы безопасности

Пропуск угрозы может привести к серьёзным последствиям.

## Практическая цель

В таких задачах важно минимизировать false negatives.

## Итог

Высокий recall важен, когда пропуск положительных случаев особенно опасен.
