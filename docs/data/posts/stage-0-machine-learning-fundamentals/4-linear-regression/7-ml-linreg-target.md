# 7. Что такое целевая переменная (target)?

Target — это переменная, которую модель должна предсказать.

В задачах линейной регрессии target всегда является **числовым значением**.

## Пример

Dataset:

| area | rooms | price |
|----|----|----|
| 45 | 1 | 120000 |

Здесь:

- area и rooms — признаки  
- price — target

## Роль target

Target используется для обучения модели.

Во время обучения модель сравнивает:

prediction → target

На основе этой ошибки модель корректирует свои параметры.

## В задачах regression

Target может быть:

- цена  
- температура  
- количество продаж  
- время

## Итог

Target — это значение, которое модель должна научиться предсказывать.
