# 32. Чем self-supervised learning отличается от unsupervised learning?

Self-supervised learning и unsupervised learning часто используют неразмеченные данные, но между ними есть важное различие.

## Unsupervised learning

В unsupervised learning модель пытается напрямую анализировать структуру dataset.

Она может:

- находить группы объектов  
- выявлять закономерности  
- уменьшать размерность данных

При этом нет явной задачи предсказания.

## Self-supervised learning

В self-supervised learning создаётся специальная задача обучения.

Модель получает входные данные и должна предсказать часть информации, которая была скрыта.

Таким образом появляется **обучающий сигнал**, похожий на тот, который используется в supervised learning.

## Почему это важно

Self-supervised learning позволяет обучать большие модели даже без ручной разметки данных.

Этот подход широко используется в современных системах deep learning.

## Итог

Главное различие:

- unsupervised learning анализирует структуру данных  
- self-supervised learning создаёт задачу предсказания внутри самих данных
