# 13. Почему без активации сеть остаётся линейной моделью?

Если сеть состоит только из линейных преобразований, их можно объединить в одно.

Например:

Layer 1  
z₁ = W₁x

Layer 2  
z₂ = W₂z₁

Подставим:

z₂ = W₂(W₁x)

Это снова линейная функция.

## Следствие

Сколько бы слоёв ни было в сети, результат всё равно можно представить как одну линейную модель.

## Итог

Без функций активации глубокая сеть не становится мощнее линейной модели.
