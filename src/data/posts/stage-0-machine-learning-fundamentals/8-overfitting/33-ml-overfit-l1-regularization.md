# 33. Что такое L1 regularization?

L1 regularization — это метод регуляризации, при котором к функции потерь добавляется штраф за **абсолютные значения параметров модели**.

## Формула

penalty = λ Σ |w|

где:

w — параметры модели  
λ — коэффициент регуляризации

## Особенность

L1 регуляризация может приводить к тому, что некоторые веса становятся равными нулю.

Это означает, что соответствующие признаки фактически исключаются из модели.

## Практическое значение

L1 регуляризация часто используется для:

- уменьшения числа признаков  
- автоматического feature selection

## Итог

L1 regularization добавляет штраф за абсолютные значения весов и может обнулять часть параметров модели.
