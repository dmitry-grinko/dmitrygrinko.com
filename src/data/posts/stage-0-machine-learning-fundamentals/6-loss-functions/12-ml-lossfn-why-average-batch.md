# 12. Почему loss обычно усредняется по batch?

Во время обучения модели данные обычно обрабатываются **не по одному примеру**, а небольшими группами — batch.

Поэтому ошибка вычисляется для всех примеров batch и затем усредняется.

## Причины

### Стабильность обучения

Усреднение уменьшает влияние случайных колебаний отдельных примеров.

### Сравнимость значений

Если loss усредняется, его значение не зависит от размера batch.

### Удобство оптимизации

Алгоритмы оптимизации работают стабильнее с нормализованными значениями ошибки.

## Итог

Loss усредняется по batch, чтобы обучение было стабильным и масштабируемым.
