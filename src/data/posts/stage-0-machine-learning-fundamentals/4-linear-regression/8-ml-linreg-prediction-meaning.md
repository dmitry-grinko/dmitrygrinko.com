# 8. Что означает предсказание модели?

Предсказание модели — это значение, которое модель вычисляет на основе входных признаков.

Это результат применения формулы линейной регрессии к данным.

## Пример

Модель:

price = 2000 × area + 10000

Если площадь квартиры 50:

price = 2000 × 50 + 10000 = 110000

110000 — это prediction модели.

## Разница между prediction и target

Важно различать два значения:

target — реальное значение из dataset  
prediction — значение, вычисленное моделью

Во время обучения модель пытается сделать prediction максимально близким к target.

## Использование prediction

После обучения модель может использоваться для новых данных.

Например:

- оценка стоимости недвижимости
- прогноз продаж
- прогноз спроса

## Итог

Prediction — это значение, которое модель вычисляет на основе признаков объекта.
