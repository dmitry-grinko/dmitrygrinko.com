# 23. Как баланс bias и variance влияет на качество модели?

Качество модели определяется балансом между bias и variance.

Эти два источника ошибки ведут себя противоположным образом.

## Простая модель

- высокий bias  
- низкая variance

Такая модель недостаточно гибкая.

## Сложная модель

- низкий bias  
- высокая variance

Такая модель может переобучиться.

## Цель обучения

Найти модель, которая имеет:

- умеренный bias  
- умеренную variance

Такой баланс позволяет модели хорошо обобщать данные.

## Итог

Хорошая модель достигает баланса между bias и variance.
