# 38. Что такое logits?

Logits — это **сырые выходы модели до применения функции активации**.

Это значения, которые выдаёт линейный слой модели.

## Пример

Модель может выдать:

[2.3, −1.2, 0.8]

Это logits.

После применения softmax они превращаются в вероятности.

## Почему это важно

Многие loss функции работают напрямую с logits.

Это делает вычисления более стабильными.

## Итог

Logits — это значения модели до преобразования в вероятности.
