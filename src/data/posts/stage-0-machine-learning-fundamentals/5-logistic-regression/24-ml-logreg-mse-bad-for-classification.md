# 24. Почему MSE плохо работает для классификации?

MSE плохо подходит для классификации, потому что он не учитывает вероятностную природу задачи.

## Проблема интерпретации

MSE измеряет разницу между числами.

Но в классификации важна **вероятность правильного класса**.

## Слабое наказание ошибок

MSE не сильно штрафует уверенные неправильные предсказания.

Например:

p = 0.01 при правильном классе 1

это серьёзная ошибка, но MSE не всегда отражает её достаточно сильно.

## Проблемы оптимизации

При использовании MSE вместе с sigmoid обучение может стать менее стабильным.

Градиенты могут становиться очень маленькими.

## Итог

MSE плохо подходит для классификации, потому что не учитывает свойства вероятностей.
