# 39. Как пропущенные значения влияют на обучение модели?

Missing data может негативно влиять на обучение модели.

Это связано с тем, что пропущенные значения нарушают структуру dataset.

## Ограничения алгоритмов

Многие алгоритмы машинного обучения не могут работать с missing values напрямую.

Если такие значения остаются в dataset, обучение может завершиться ошибкой.

## Потеря информации

Если строки с пропущенными значениями удаляются, часть информации может быть потеряна.

Это может уменьшить размер dataset и ухудшить обучение модели.

## Искажение распределения

Некоторые методы imputation могут изменять распределение данных.

Например, массовая замена значений средним может уменьшить вариативность признака.

## Итог

Пропущенные значения могут:

- затруднять обучение модели  
- приводить к потере информации  
- искажать структуру dataset
