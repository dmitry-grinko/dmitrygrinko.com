# 15. Что такое clustering?

Clustering — это задача в machine learning, в которой объекты автоматически разделяются на группы на основе их сходства.

Каждая такая группа называется **cluster**.

Главная идея clustering заключается в том, чтобы объекты внутри одной группы были похожи друг на друга, а объекты из разных групп — отличались.

## Как это работает

Алгоритм получает dataset с признаками объектов и пытается определить, какие объекты похожи.

Например, если dataset содержит информацию о клиентах:

| age | income | purchases |
|----|-------|----------|
| 25 | 40000 | 3 |
| 27 | 42000 | 4 |
| 55 | 90000 | 10 |

алгоритм может обнаружить, что некоторые клиенты имеют похожие характеристики.

В результате он может разделить dataset на группы, например:

- молодые клиенты  
- клиенты с высоким доходом  
- активные покупатели

## Где используется clustering

Clustering часто применяется в задачах анализа данных.

Например:

- сегментация клиентов  
- группировка похожих товаров  
- анализ поведения пользователей  
- исследование больших datasets

Этот метод помогает понять структуру данных без необходимости иметь размеченный dataset.

## Важная особенность clustering

В отличие от supervised learning здесь **нет правильного ответа**, который можно было бы использовать для проверки результата.

Алгоритм просто пытается найти наиболее естественное разделение данных на группы.

## Итог

Clustering — это задача unsupervised learning, в которой объекты автоматически группируются на основе сходства между их признаками.
