# 25. Что такое learning rate?

Learning rate — это параметр алгоритма обучения, который определяет **размер шага при обновлении параметров модели**.

Он используется в алгоритмах оптимизации, таких как gradient descent.

## Формально

После вычисления градиента параметры модели обновляются так:

new_parameter = old_parameter − learning_rate × gradient

Learning rate контролирует, насколько сильно изменяются параметры.

## Интуитивная идея

Если learning rate большой, модель делает большие шаги.

Если learning rate маленький, модель изменяет параметры медленно.

## Почему это важно

Правильный выбор learning rate влияет на:

- скорость обучения  
- стабильность обучения  
- сходимость алгоритма

## Итог

Learning rate — это параметр, который определяет размер шага при обновлении параметров модели во время обучения.
