# 32. Почему используется gradient descent?

Функция потерь logistic regression не имеет простого аналитического решения.

Поэтому параметры модели находят с помощью численных методов оптимизации.

Самый распространённый из них — gradient descent.

## Основная идея

Gradient descent постепенно изменяет параметры модели так, чтобы уменьшить значение функции потерь.

Алгоритм:

1. вычисляет градиент функции потерь  
2. изменяет параметры в направлении уменьшения ошибки  
3. повторяет процесс

## Почему это удобно

Gradient descent:

- работает с большими dataset  
- легко масштабируется  
- используется во многих ML алгоритмах

## Итог

Gradient descent используется для поиска параметров модели, которые минимизируют log loss.
