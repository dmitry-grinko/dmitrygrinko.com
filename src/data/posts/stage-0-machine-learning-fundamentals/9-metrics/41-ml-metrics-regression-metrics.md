# 41. Какие метрики используются для задач regression?

В задачах regression модель предсказывает **не класс, а числовое значение**. Поэтому метрики должны измерять **насколько предсказанные числа отличаются от реальных значений**.

Наиболее распространённые метрики:

- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- Mean Absolute Error (MAE)
- R² (coefficient of determination)

Все эти метрики измеряют ошибку предсказаний, но делают это разными способами.

## Итог

Для regression используются метрики, которые измеряют разницу между предсказанными и реальными значениями.
