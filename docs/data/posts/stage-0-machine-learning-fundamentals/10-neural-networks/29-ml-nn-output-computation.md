# 29. Как вычисляется выход нейронной сети?

Выход нейронной сети вычисляется последовательным применением преобразований на каждом слое.

Каждый слой получает вход, преобразует его и передаёт дальше.

## Общая формула слоя

Для одного слоя:

z = Wx + b  
h = activation(z)

Это означает:

1. вход x умножается на матрицу весов W  
2. добавляется bias b  
3. применяется функция активации  
4. получается выход h

## Как получается финальный output

Выход первого слоя становится входом второго слоя.

Выход второго — входом третьего.

Так продолжается до последнего слоя.

## Пример

Если сеть имеет два слоя:

h₁ = activation(W₁x + b₁)  
output = activation(W₂h₁ + b₂)

Это и есть итоговое вычисление выхода сети.

## Итог

Выход нейронной сети вычисляется как последовательность линейных преобразований и функций активации через все слои.
