# 7. Чем отличаются признаки (features) и целевая переменная (target)?

Features и target выполняют разные роли в dataset.

Features описывают объект, а target представляет результат, который модель должна предсказать.

## Features

Features — это входные данные.

Они содержат информацию, которую модель использует для анализа.

Пример:

- возраст пользователя  
- доход  
- количество покупок

## Target

Target — это значение, которое модель должна предсказать.

Например:

- вероятность покупки  
- цена квартиры  
- категория письма

## Как модель использует их

Во время обучения модель анализирует связь между features и target.

Задача модели — построить функцию:

features → target

После обучения модель сможет принимать новые features и предсказывать target.

## Итог

- features — входные характеристики объекта  
- target — переменная, которую модель должна предсказать
