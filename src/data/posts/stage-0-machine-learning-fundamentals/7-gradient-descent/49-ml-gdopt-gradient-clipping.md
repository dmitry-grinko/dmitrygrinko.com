# 49. Что такое gradient clipping?

Gradient clipping — это техника, которая ограничивает **максимальную величину градиентов** во время обучения.

## Основная проблема

Иногда градиенты могут становиться очень большими.

Это называется **exploding gradients**.

## Решение

Gradient clipping ограничивает величину градиента.

Например:

если |gradient| > threshold

то градиент масштабируется.

## Практическое применение

Gradient clipping часто используется при обучении:

- рекуррентных нейронных сетей  
- больших deep learning моделей

## Итог

Gradient clipping предотвращает слишком большие обновления параметров.
