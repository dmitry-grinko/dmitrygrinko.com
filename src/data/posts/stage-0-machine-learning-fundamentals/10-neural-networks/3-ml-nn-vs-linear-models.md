# 3. Чем нейронные сети отличаются от линейных моделей?

Главное отличие нейронных сетей от линейных моделей заключается в **нелинейности**.

Линейные модели строят предсказание как линейную комбинацию признаков.

Например:

y = w₁x₁ + w₂x₂ + ... + b

Такая модель может описывать только линейные зависимости.

## Нейронные сети

Нейронные сети добавляют **нелинейные функции активации** между слоями.

Это позволяет моделировать сложные зависимости.

Каждый слой выполняет преобразование данных, а несколько слоёв создают сложную функцию.

## Последствие

Нейронные сети могут моделировать гораздо более сложные структуры данных.

## Итог

Линейные модели описывают линейные зависимости, а нейронные сети способны моделировать сложные нелинейные функции.
