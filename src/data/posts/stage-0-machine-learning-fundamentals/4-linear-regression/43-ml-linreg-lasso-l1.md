# 43. Что такое Lasso regression (L1 regularization)?

Lasso regression — это метод линейной регрессии, который использует **L1 регуляризацию**.

В этом случае штраф зависит от абсолютных значений коэффициентов.

## Формула

Функция потерь выглядит так:

loss = MSE + λ Σ |w|

где:

- w — коэффициенты модели  
- λ — параметр регуляризации

## Особенность L1 регуляризации

L1 регуляризация может приводить к тому, что некоторые коэффициенты становятся **равными нулю**.

Это означает, что соответствующие признаки фактически удаляются из модели.

## Почему это полезно

Lasso regression выполняет одновременно:

- обучение модели  
- выбор признаков

Это может сделать модель более простой и интерпретируемой.

## Итог

Lasso regression — это линейная регрессия с L1 регуляризацией, которая может автоматически исключать некоторые признаки.
