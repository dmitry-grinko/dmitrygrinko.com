# 24. Почему SGD часто используется на практике?

SGD часто используется на практике, потому что он **хорошо масштабируется на большие dataset**.

## Причины популярности

### Быстрые обновления

Параметры модели обновляются после каждого примера.

Это позволяет модели быстро реагировать на данные.

### Подходит для больших dataset

Не требуется вычислять градиент по всему dataset.

Это снижает вычислительную нагрузку.

### Помогает избегать локальных минимумов

Шум в градиентах иногда помогает алгоритму выходить из плохих решений.

## Ограничение

Из-за шума обучение может быть менее стабильным.

## Итог

SGD используется на практике благодаря своей эффективности и масштабируемости.
