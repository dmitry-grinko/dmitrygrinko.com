# 41. Что такое momentum в оптимизации?

Momentum — это техника оптимизации, которая помогает алгоритму gradient descent **двигаться быстрее и стабильнее по поверхности функции потерь**.

Идея заимствована из физики: параметрам добавляется «инерция», которая накапливает информацию о предыдущих шагах.

## Основная проблема обычного gradient descent

Обычный gradient descent может двигаться медленно, особенно если поверхность функции вытянута.

В таких случаях алгоритм начинает:

- колебаться  
- двигаться зигзагами  
- медленно продвигаться к минимуму

## Как работает momentum

Momentum накапливает направление предыдущих градиентов.

Это позволяет:

- ускоряться в правильном направлении  
- сглаживать колебания

## Итог

Momentum добавляет «инерцию» обновлениям параметров и делает оптимизацию быстрее и стабильнее.
