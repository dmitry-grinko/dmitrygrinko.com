# 19. Что такое итерация обучения?

Итерация обучения — это **один шаг обновления параметров модели**.

На одной итерации модель:

1. делает предсказание  
2. вычисляет функцию потерь  
3. вычисляет градиент  
4. обновляет параметры

## Что используется на итерации

Обычно используется один batch данных.

После обработки batch параметры модели обновляются.

## Связь с batch

Если dataset разделён на batch, каждая обработка batch считается одной итерацией.

## Итог

Итерация обучения — это один цикл обновления параметров модели.
