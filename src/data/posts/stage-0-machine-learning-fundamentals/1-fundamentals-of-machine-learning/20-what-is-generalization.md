# 20. Что такое generalization?

Generalization — это способность модели машинного обучения сохранять хорошее качество на данных, которые она не видела во время обучения.

Модель обучается на **training set**, но в реальной системе она всегда будет получать новые данные. Поэтому важно, чтобы модель не просто запомнила обучающие примеры, а извлекла из них общие закономерности.

Проще говоря:

training data → обучение → модель → новые данные

Если модель умеет правильно работать на новых данных, говорят, что она **хорошо обобщает (generalizes)**.

## Пример

Представим модель, которая обучается предсказывать цену квартиры.

Если модель просто запомнила training data, она может идеально предсказывать цены для этих конкретных квартир.

Но если ей показать новую квартиру, она может ошибаться.

Если же модель действительно поняла связь между признаками (например, площадью, количеством комнат и ценой), она сможет делать разумные предсказания и для новых объектов.

Это и есть generalization.

## Как понять, что модель обобщает

Обычно это проверяют с помощью **validation set** или **test set**.

Если модель показывает:

- хорошее качество на training set
- похожее качество на test set

значит она, скорее всего, обобщает закономерности.

Если же ситуация выглядит так:

- training accuracy — очень высокая
- test accuracy — сильно ниже

то модель, скорее всего, **переобучилась (overfitting)**.

## Что мешает generalization

Есть несколько распространённых причин, почему модель плохо обобщает:

- слишком маленький dataset
- шумные или неправильно размеченные данные
- слишком сложная модель
- сильное отличие training data от реальных данных

## Почему это одна из главных проблем ML

В машинном обучении легко построить модель, которая хорошо работает на training data.

Гораздо сложнее построить модель, которая:

- не запоминает данные
- извлекает реальные закономерности
- сохраняет качество на новых данных

Именно поэтому в ML так важны:

- разделение dataset на train / validation / test
- регуляризация
- контроль overfitting

## Итог

Generalization — это способность модели применять знания, полученные на обучающих данных, к новым данным и сохранять хорошее качество предсказаний.
