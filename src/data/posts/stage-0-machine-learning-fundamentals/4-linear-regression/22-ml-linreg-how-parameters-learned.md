# 22. Как параметры линейной регрессии определяются во время обучения?

Во время обучения модель ищет такие значения параметров, которые **минимизируют ошибку предсказаний**.

Этот процесс называется **оптимизацией**.

## Основная идея

Модель делает предсказание.

Затем вычисляется ошибка между:

target и prediction.

После этого параметры модели корректируются так, чтобы ошибка стала меньше.

## Процесс обучения

Процесс обычно выглядит так:

1. параметры модели инициализируются случайными значениями  
2. модель делает предсказания  
3. вычисляется функция потерь  
4. параметры корректируются  
5. процесс повторяется

Постепенно параметры приближаются к оптимальным значениям.

## Методы поиска параметров

Для линейной регрессии используются два основных подхода:

- аналитическое решение (normal equation)  
- численные методы оптимизации (gradient descent)

## Итог

Параметры линейной регрессии определяются во время обучения через минимизацию функции потерь.
