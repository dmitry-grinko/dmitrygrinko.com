# 50. Почему logistic regression считается простой, но мощной моделью?

Logistic regression считается простой моделью, потому что её структура математически понятна.

Но при этом она остаётся мощным инструментом для многих задач.

## Простота

Модель имеет простую форму:

линейная комбинация признаков + sigmoid.

## Хорошая обобщающая способность

При правильных признаках logistic regression может давать очень хорошие результаты.

## Интерпретируемость

Коэффициенты модели позволяют понять влияние признаков.

## Практическая ценность

Во многих реальных задачах logistic regression показывает конкурентные результаты.

## Итог

Logistic regression проста по структуре, но при правильных данных может быть очень эффективной моделью.
