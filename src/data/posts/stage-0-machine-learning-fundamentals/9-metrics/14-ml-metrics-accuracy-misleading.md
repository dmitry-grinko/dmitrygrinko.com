# 14. Когда accuracy может быть вводящей в заблуждение?

Accuracy может быть вводящей в заблуждение, когда **dataset содержит сильный дисбаланс классов**.

В таких случаях модель может показывать высокую accuracy, даже если она плохо решает задачу.

## Пример

Представим dataset:

- 99% нормальных транзакций  
- 1% мошенничества

Если модель всегда предсказывает «нормальная транзакция», она получит accuracy = 99%.

Но при этом она не обнаружит ни одного случая мошенничества.

## Почему это проблема

Accuracy не показывает, какие именно ошибки делает модель.

## Итог

Accuracy может быть вводящей в заблуждение при сильном дисбалансе классов.
