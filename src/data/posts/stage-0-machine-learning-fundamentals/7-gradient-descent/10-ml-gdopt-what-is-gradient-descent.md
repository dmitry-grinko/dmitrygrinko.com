# 10. Что такое gradient descent?

Gradient descent — это алгоритм оптимизации, который используется для **минимизации функции потерь**.

Он работает за счёт постепенного изменения параметров модели.

## Основная идея

Алгоритм делает следующие шаги:

1. вычисляет градиент функции потерь  
2. определяет направление уменьшения ошибки  
3. обновляет параметры модели

## Формула обновления

θ = θ - η ∇L(θ)

где:

- θ — параметры модели  
- η — learning rate  
- ∇L — градиент функции потерь

## Как это выглядит

Алгоритм постепенно "спускается" по поверхности функции потерь.

Gradient descent — это основной алгоритм оптимизации, используемый для обучения большинства моделей машинного обучения.
