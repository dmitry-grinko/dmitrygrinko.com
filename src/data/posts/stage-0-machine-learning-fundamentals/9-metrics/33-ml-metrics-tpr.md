# 33. Что такое true positive rate?

True Positive Rate (TPR) — это метрика, которая показывает **долю правильно найденных положительных объектов**.

Она также называется recall или sensitivity.

## Формула

TPR = TP / (TP + FN)

где:

TP — true positives  
FN — false negatives

## Что означает TPR

TPR показывает, какую часть реальных положительных объектов модель смогла обнаружить.

## Пример

Если в dataset есть 100 положительных объектов, а модель обнаружила 90:

TPR = 0.9

## Итог

True Positive Rate измеряет способность модели находить положительные объекты.
