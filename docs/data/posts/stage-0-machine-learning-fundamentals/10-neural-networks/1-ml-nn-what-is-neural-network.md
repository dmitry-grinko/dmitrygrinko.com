# 1. Что такое нейронная сеть (neural network)?

Нейронная сеть — это тип модели машинного обучения, состоящий из множества простых вычислительных элементов, называемых **нейронами**, которые соединены между собой.

Каждый нейрон выполняет простую операцию: он принимает входные значения, умножает их на веса, суммирует и применяет функцию активации.

Нейроны объединяются в **слои**, и данные проходят через эти слои последовательно.

В результате сеть может моделировать сложные зависимости между входными данными и целевой переменной.

## Интуитивная идея

Нейронную сеть можно представить как набор преобразований:

данные → преобразование → преобразование → результат

Каждый слой постепенно извлекает более сложные признаки из данных.

## Итог

Нейронная сеть — это модель, состоящая из соединённых нейронов, которая преобразует входные данные в предсказание через несколько слоёв вычислений.
