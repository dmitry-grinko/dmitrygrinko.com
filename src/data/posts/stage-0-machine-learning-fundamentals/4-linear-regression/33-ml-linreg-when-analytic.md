# 33. Когда используют аналитическое решение?

Аналитическое решение (normal equation) используют, когда размер задачи относительно небольшой.

Основное ограничение связано с вычислениями матриц.

## Когда метод работает хорошо

Normal equation подходит, если:

- dataset небольшой  
- число признаков невелико  
- матричные операции легко вычисляются

В таких случаях параметры можно найти быстро и точно.

## Когда возникают проблемы

Если количество признаков становится большим, вычисление матрицы:

(XᵀX)⁻¹

становится дорогостоящим.

Вычислительная сложность растёт примерно как:

O(n³)

где n — число признаков.

## Практика

Поэтому аналитическое решение редко используется в задачах с большим количеством признаков.

## Итог

Normal equation применяется для небольших задач, где матричные вычисления остаются эффективными.
