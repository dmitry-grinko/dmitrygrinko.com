# 32. Почему нейронные сети могут иметь миллионы параметров?

Количество параметров зависит от:

- числа слоёв
- количества нейронов
- числа соединений между слоями

Каждое соединение между нейронами имеет свой вес.

## Пример

Представим два слоя:

первый слой — 100 нейронов  
второй слой — 200 нейронов

Каждый нейрон второго слоя соединён со всеми нейронами первого.

Количество весов:

100 × 200 = 20 000

Плюс 200 bias.

Всего уже 20 200 параметров.

## В глубоких сетях

В современных моделях:

- миллионы параметров — обычная ситуация  
- в больших моделях — миллиарды

Например:

- ResNet — десятки миллионов параметров  
- большие language models — миллиарды

## Итог

Чем больше нейронов и слоёв, тем больше параметров имеет нейронная сеть.
