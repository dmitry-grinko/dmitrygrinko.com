# 41. Что такое regularization в линейной регрессии?

Regularization — это метод, который добавляет **дополнительное ограничение на параметры модели**, чтобы уменьшить переобучение.

Идея заключается в том, чтобы не позволять коэффициентам модели становиться слишком большими.

## Почему это нужно

Если модель имеет много признаков, она может начать подстраиваться под шум в данных.

Это приводит к переобучению.

Регуляризация вводит штраф за большие коэффициенты и заставляет модель выбирать более простое решение.

## Как это делается

Во время обучения к функции потерь добавляется дополнительный штраф.

Например:

loss = MSE + penalty

где penalty зависит от величины коэффициентов модели.

## Результат

Регуляризация приводит к тому, что:

- коэффициенты становятся меньше  
- модель становится более устойчивой  
- уменьшается переобучение

## Итог

Regularization — это способ контролировать сложность модели через ограничение её параметров.
