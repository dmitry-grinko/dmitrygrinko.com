# 30. В каких задачах используется Huber loss?

Huber loss применяется в задачах регрессии, где данные могут содержать шум или выбросы.

## Примеры

Прогнозирование цен

Финансовые модели

Робототехника и сенсорные данные

Computer vision задачи регрессии

## Почему она используется

Huber loss делает обучение:

- более устойчивым  
- менее чувствительным к аномальным точкам

## Итог

Huber loss используется в регрессионных задачах с шумными данными.
