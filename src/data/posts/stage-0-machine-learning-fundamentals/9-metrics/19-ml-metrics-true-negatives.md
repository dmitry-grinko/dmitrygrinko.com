# 19. Что такое true negatives?

True negatives (TN) — это случаи, когда **модель правильно предсказала отрицательный класс**.

## Пример

В задаче обнаружения спама:

если письмо не является спамом и модель также предсказала «не спам», это true negative.

## Что показывает TN

True negatives показывают, сколько отрицательных объектов модель классифицировала правильно.

## Практическое значение

В некоторых задачах важно минимизировать ложные срабатывания, поэтому правильные отрицательные предсказания имеют большое значение.

## Итог

True negatives — это правильные предсказания отрицательного класса.
