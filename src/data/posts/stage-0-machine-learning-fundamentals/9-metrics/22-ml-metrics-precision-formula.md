# 22. Как вычисляется precision?

Precision вычисляется как отношение **правильных положительных предсказаний** к **общему количеству положительных предсказаний модели**.

## Формула

precision = TP / (TP + FP)

где:

TP — true positives  
FP — false positives

## Что это означает

TP — сколько положительных объектов модель нашла правильно.

FP — сколько объектов модель ошибочно посчитала положительными.

Precision показывает, какая часть положительных предсказаний оказалась правильной.

## Итог

Precision измеряет точность предсказаний положительного класса.
