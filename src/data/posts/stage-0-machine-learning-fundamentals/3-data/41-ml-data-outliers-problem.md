# 41. Почему выбросы (outliers) могут быть проблемой для моделей?

Выбросы могут существенно влиять на обучение модели, особенно если модель чувствительна к экстремальным значениям.

Некоторые алгоритмы пытаются минимизировать ошибку на всех наблюдениях. Если в данных есть экстремальные значения, модель может подстраиваться под них.

## Искажение модели

Например, линейная регрессия строит линию, которая минимизирует среднюю ошибку.

Если в dataset есть несколько очень больших или очень маленьких значений, линия может сильно сместиться, чтобы учесть эти точки.

В результате модель будет хуже описывать основную часть данных.

## Ошибочные данные

Иногда выбросы возникают из-за ошибок:

- ошибка ввода  
- ошибка измерения  
- повреждённые данные

Если такие значения не удалить, модель будет обучаться на некорректной информации.

## Когда выбросы полезны

Важно понимать, что не все выбросы являются ошибками.

Иногда они представляют реальные, но редкие события.

Например:

- крупные финансовые транзакции  
- редкие медицинские случаи

Поэтому перед удалением выбросов важно понять их природу.

## Итог

Выбросы могут искажать обучение модели и ухудшать качество предсказаний, особенно если они являются ошибками данных.
