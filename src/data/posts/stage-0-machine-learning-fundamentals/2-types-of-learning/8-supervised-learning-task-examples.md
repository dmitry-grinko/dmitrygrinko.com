# 8. Какие примеры задач supervised learning существуют?

Supervised learning применяется в задачах, где можно собрать dataset с правильными ответами.

Во многих индустриях такие задачи возникают регулярно, поэтому supervised learning стал одним из самых распространённых подходов в машинном обучении.

## Предсказание числовых значений

Некоторые задачи требуют оценки конкретного числа.

Примеры:

- предсказание цены недвижимости  
- прогнозирование продаж  
- оценка времени доставки  
- прогноз потребления энергии  

В таких задачах модель пытается предсказать числовое значение на основе признаков.

## Классификация объектов

Другие задачи требуют определить, к какому классу относится объект.

Примеры:

- определение спама в электронной почте  
- распознавание объектов на изображениях  
- классификация новостей по темам  
- обнаружение мошеннических транзакций  

В этих задачах модель выбирает одну из возможных категорий.

## Примеры из реальных систем

Supervised learning используется в большом количестве продуктов:

- системы рекомендаций  
- системы распознавания речи  
- системы компьютерного зрения  
- системы медицинской диагностики  

Во всех этих случаях модель обучается на данных, где правильный результат уже известен.

## Итог

Supervised learning применяется в задачах, где можно собрать примеры с правильными ответами и использовать их для обучения модели.
