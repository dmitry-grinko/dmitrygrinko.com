# 28. Что происходит с loss, когда вероятность близка к правильному классу?

Когда вероятность, предсказанная моделью, близка к правильному классу, значение log loss становится маленьким.

Это означает, что модель делает хорошее предсказание.

## Пример

Истинный класс:

y = 1

Предсказание:

p = 0.95

В этом случае log loss будет очень маленьким.

## Почему это происходит

Функция логарифма меняется медленно возле значений, близких к 1.

Поэтому корректные и уверенные предсказания почти не увеличивают loss.

## Итог

Если модель предсказывает вероятность, близкую к правильному классу, значение функции потерь становится минимальным.
