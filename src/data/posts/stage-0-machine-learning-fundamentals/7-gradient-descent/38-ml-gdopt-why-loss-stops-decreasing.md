# 38. Почему loss может перестать уменьшаться?

Loss может перестать уменьшаться по нескольким причинам.

## Достигнут минимум

Алгоритм уже нашёл точку, близкую к минимуму функции.

## Learning rate слишком маленький

Шаги обновления параметров становятся настолько маленькими, что изменения почти незаметны.

## Saddle point

Алгоритм может находиться в области, где градиент почти равен нулю.

## Ограничения модели

Иногда модель просто не может лучше объяснить данные.

## Итог

Loss может перестать уменьшаться из-за достижения минимума, маленького learning rate или ограничений модели.
