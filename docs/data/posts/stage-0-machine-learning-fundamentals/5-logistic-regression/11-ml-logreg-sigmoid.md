# 11. Что такое sigmoid функция?

Sigmoid — это математическая функция, которая преобразует любое число в значение между 0 и 1.

Она имеет S-образную форму и используется в logistic regression для превращения линейного выхода модели в вероятность.

Формула функции:

```python
σ(z) = 1 / (1 + e⁻ᶻ)
```

где:

- z — линейная комбинация признаков (logit)

## Как выглядит функция

Если построить график sigmoid, он будет выглядеть как плавная S-образная кривая.

При больших отрицательных значениях функция стремится к 0.  
При больших положительных значениях функция стремится к 1.

## Почему она используется

Sigmoid позволяет взять любое число (например −10 или 5) и преобразовать его в корректную вероятность.

## Итог

Sigmoid — это функция, которая преобразует линейный выход модели в значение от 0 до 1.
