# 10. Почему loss функция должна быть дифференцируемой?

Во многих алгоритмах машинного обучения параметры модели обновляются с помощью градиентов.

Для этого функция потерь должна быть **дифференцируемой**.

## Основная идея

Градиент — это производная функции потерь по параметрам модели.

Он показывает, как нужно изменить параметры, чтобы уменьшить ошибку.

## Использование

Алгоритмы оптимизации, такие как:

- gradient descent  
- stochastic gradient descent

используют производные функции потерь.

## Что происходит без дифференцируемости

Если функция потерь не имеет производной, вычислить градиент становится сложно.

## Итог

Дифференцируемость позволяет эффективно обучать модели с помощью методов оптимизации.
