# 15. Что такое root mean squared error (RMSE)?

Root Mean Squared Error (RMSE) — это метрика качества модели, основанная на MSE.

Она равна **квадратному корню из MSE**.

## Формула

RMSE = √MSE

## Почему используют RMSE

MSE измеряется в квадрате единиц target.

Например:

если target — цена в долларах, MSE будет в долларах².

RMSE возвращает метрику в **тех же единицах**, что и target.

Это делает её более удобной для интерпретации.

## Пример

Если RMSE = 15000

это означает, что средняя ошибка модели примерно равна 15000.

## Где используется RMSE

RMSE часто используется для:

- оценки regression моделей  
- сравнения разных моделей

## Итог

RMSE — это квадратный корень из MSE, который измеряет среднюю ошибку модели в тех же единицах, что и target.
