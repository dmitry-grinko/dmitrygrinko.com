# 48. Когда data augmentation используется?

Data augmentation применяется в ситуациях, когда dataset недостаточно большой или содержит мало разнообразных примеров.

## Маленький dataset

Если dataset небольшой, модель может быстро переобучиться.

Создание дополнительных примеров помогает увеличить разнообразие данных.

## Обучение глубоких моделей

Deep learning модели часто требуют огромное количество данных.

Data augmentation помогает увеличить dataset без необходимости собирать новые данные.

## Улучшение устойчивости модели

Augmentation может помочь модели лучше справляться с вариациями данных.

Например:

- разные положения объекта  
- изменения освещения  
- небольшие искажения

## Итог

Data augmentation используется, когда dataset небольшой или когда нужно повысить устойчивость модели к изменениям данных.
