# 17. Какие элементы входят в confusion matrix?

Confusion matrix для бинарной классификации состоит из четырёх элементов.

## True Positive (TP)

Модель правильно предсказала положительный класс.

## True Negative (TN)

Модель правильно предсказала отрицательный класс.

## False Positive (FP)

Модель ошибочно предсказала положительный класс.

## False Negative (FN)

Модель ошибочно предсказала отрицательный класс.

## Почему это важно

Все основные метрики классификации вычисляются на основе этих четырёх значений.

## Итог

Confusion matrix состоит из TP, TN, FP и FN.
