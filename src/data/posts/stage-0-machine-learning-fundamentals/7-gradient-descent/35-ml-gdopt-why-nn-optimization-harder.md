# 35. Почему оптимизация нейронных сетей сложнее линейных моделей?

Оптимизация нейронных сетей сложнее из-за **очень высокой размерности пространства параметров** и сложной формы функции потерь.

## Большое число параметров

Линейная модель может иметь десятки параметров.

Современные нейронные сети могут иметь миллионы или миллиарды параметров.

## Сложная поверхность функции потерь

Функция потерь нейронных сетей:

- сильно нелинейная  
- содержит множество локальных минимумов  
- содержит saddle points

## Взаимодействие параметров

Параметры нейронной сети сильно зависят друг от друга.

Изменение одного веса может повлиять на многие другие части модели.

## Итог

Оптимизация нейронных сетей сложнее из-за высокой размерности и сложной формы функции потерь.
