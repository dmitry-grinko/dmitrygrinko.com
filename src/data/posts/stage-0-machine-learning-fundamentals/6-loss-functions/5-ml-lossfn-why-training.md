# 5. Почему функция потерь используется во время обучения модели?

Во время обучения модели необходимо понимать, **насколько хороши её предсказания**.

Функция потерь выполняет именно эту роль.

## Основная идея

Loss функция превращает ошибку модели в число.

Это число показывает, насколько модель ошибается.

## Использование в обучении

Алгоритм обучения использует значение loss для изменения параметров модели.

Если loss уменьшается, обучение движется в правильном направлении.

## Без функции потерь

Без функции потерь модель не могла бы понять:

- насколько её предсказания точны  
- как нужно изменить параметры

## Итог

Функция потерь используется во время обучения, потому что она измеряет ошибку модели.
