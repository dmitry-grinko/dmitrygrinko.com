# 4. Что означает минимизация функции потерь?

Минимизация функции потерь означает поиск таких параметров модели, при которых **ошибка модели становится как можно меньше**.

## Основная идея

Модель делает предсказания на основе своих параметров.

Если параметры выбраны плохо, ошибки будут большими.

Алгоритм обучения постепенно изменяет параметры так, чтобы ошибки уменьшались.

## Пример

Предсказание модели:

prediction = 70

Реальное значение:

target = 100

Ошибка:

30

После обучения модель может начать предсказывать:

prediction = 95

Ошибка уменьшается.

## Процесс

Минимизация происходит постепенно:

- вычисляется ошибка  
- вычисляется направление изменения параметров  
- параметры обновляются

Этот процесс повторяется тысячи или миллионы раз.

## Итог

Минимизация функции потерь — это процесс уменьшения ошибки модели через изменение параметров.
