# 36. Как ошибка распространяется назад по сети?

После forward pass сеть получает ошибку.

Эта ошибка сначала известна **на выходном слое**.

Затем она распространяется назад по сети.

## Последовательность

1. вычисляется ошибка на выходном слое  
2. определяется вклад выходного слоя  
3. ошибка передаётся предыдущему слою  
4. процесс повторяется для всех слоёв

## Пример

Если ошибка большая, алгоритм может обнаружить, что:

- некоторые веса увеличивают ошибку
- некоторые уменьшают

Это позволяет корректировать параметры.

## Почему это работает

Каждый слой знает, как его выход влияет на следующий слой.

Поэтому можно вычислить вклад каждого слоя в итоговую ошибку.

## Итог

Ошибка распространяется назад через слои сети, позволяя вычислить градиенты параметров.
