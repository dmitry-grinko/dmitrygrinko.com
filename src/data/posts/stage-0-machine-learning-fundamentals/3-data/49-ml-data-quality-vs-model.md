# 49. Как качество данных влияет на качество модели?

Качество данных напрямую влияет на качество модели.

Модель может обучаться только на той информации, которая присутствует в dataset.

## Ошибки в данных

Если dataset содержит ошибки, модель может обучиться неправильным закономерностям.

Например:

- неправильные значения  
- неверные labels  
- шум в данных

## Неполные данные

Если dataset плохо отражает реальные условия, модель может плохо работать на новых данных.

Например:

- отсутствуют важные признаки  
- dataset содержит только часть возможных ситуаций

## Низкое разнообразие данных

Если dataset слишком однообразный, модель может плохо обобщать знания.

## Итог

Чем выше качество данных, тем выше вероятность, что модель сможет построить правильные закономерности.
