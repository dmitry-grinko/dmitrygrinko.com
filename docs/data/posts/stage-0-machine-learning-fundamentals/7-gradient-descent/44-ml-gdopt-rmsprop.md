# 44. Что такое RMSProp?

RMSProp — это алгоритм оптимизации, который решает проблему AdaGrad, связанную с уменьшением learning rate.

## Основная идея

RMSProp использует **экспоненциальное скользящее среднее квадратов градиентов**.

Это позволяет адаптировать learning rate, но не уменьшать его слишком быстро.

## Как это работает

Алгоритм:

1. отслеживает величину градиентов  
2. масштабирует шаг обновления параметров  
3. стабилизирует процесс обучения

## Практическое значение

RMSProp хорошо работает для:

- нейронных сетей  
- задач с шумными градиентами

## Итог

RMSProp — это адаптивный алгоритм оптимизации, который стабилизирует learning rate.
