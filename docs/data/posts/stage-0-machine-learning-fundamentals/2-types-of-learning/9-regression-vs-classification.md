# 9. Чем regression отличается от classification?

Regression и classification — это два основных типа задач в supervised learning.

Главное различие между ними заключается в том, **какой тип результата должна предсказать модель**.

## Regression

Regression используется, когда результат представляет собой **числовое значение**.

Модель пытается оценить непрерывную величину.

Примеры:

- цена квартиры  
- количество продаж  
- температура воздуха  
- время доставки

В regression результат может принимать большое количество возможных значений.

## Classification

Classification используется, когда результат представляет собой **категорию или класс**.

Модель должна выбрать один из заранее определённых вариантов.

Примеры:

- spam или not spam  
- мошенничество или нормальная транзакция  
- тип объекта на изображении  
- категория текста

В classification результат принадлежит к одному из классов.

## Различие между задачами

Основное различие можно сформулировать так:

- regression предсказывает **числовые значения**  
- classification предсказывает **категории**

Это различие влияет на выбор алгоритмов и метрик оценки модели.

## Итог

Regression и classification — это два основных типа задач в supervised learning, которые отличаются типом результата, который должна предсказать модель.
