# 42. Как обнаружить outliers в dataset?

Выбросы можно обнаружить с помощью анализа данных и статистических методов.

Обычно используют несколько подходов.

## Визуализация данных

Один из самых простых способов — построить графики распределения.

Например:

- гистограммы  
- box plot  
- scatter plot

Такие графики часто позволяют быстро увидеть аномальные значения.

## Статистические методы

Иногда используются статистические критерии.

Например:

Правило стандартного отклонения

Если значение сильно отличается от среднего (например, больше чем на 3 стандартных отклонения), оно может считаться выбросом.

Метод межквартильного размаха (IQR)

Значения, которые находятся далеко за пределами нормального диапазона, могут считаться выбросами.

## Модели обнаружения аномалий

В некоторых случаях используются специальные алгоритмы:

- Isolation Forest  
- Local Outlier Factor  
- One-Class SVM

Эти методы помогают находить аномальные наблюдения в больших datasets.

## Итог

Выбросы можно обнаруживать с помощью:

- визуализации данных  
- статистических методов  
- алгоритмов обнаружения аномалий
