# 1. Что такое метрика в machine learning?

Метрика — это числовой показатель, который используется для **оценки качества предсказаний модели**.

Она показывает, насколько хорошо модель решает поставленную задачу.

После того как модель делает предсказания, метрика сравнивает эти предсказания с правильными ответами.

## Пример

Если модель решает задачу классификации (например, определяет спам-письма), метрика может измерять:

- долю правильных предсказаний  
- количество ошибок  
- способность находить редкие события

## Роль метрики

Метрики позволяют:

- сравнивать разные модели  
- отслеживать улучшение модели  
- понимать, насколько модель подходит для практического использования

## Итог

Метрика — это количественная мера качества предсказаний модели.
