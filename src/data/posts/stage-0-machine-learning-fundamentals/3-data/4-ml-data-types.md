# 4. Какие типы данных используются в machine learning?

В machine learning могут использоваться разные типы данных. Тип данных влияет на то, какие модели и методы обработки можно применять.

Наиболее распространённые типы данных:

## Числовые данные

Это один из самых простых типов данных.

Примеры:

- возраст  
- цена  
- температура  
- количество покупок

Числовые данные могут быть:

- **дискретными** — количество объектов  
- **непрерывными** — температура, время

## Категориальные данные

Категориальные данные представляют собой категории или классы.

Примеры:

- страна  
- цвет  
- тип товара  
- категория пользователя

Такие признаки обычно нужно преобразовывать в числовой формат перед обучением модели.

## Текстовые данные

Текстовые данные часто используются в задачах NLP.

Примеры:

- отзывы пользователей  
- статьи  
- сообщения  
- документы

Текст обычно преобразуется в числовые представления (embeddings).

## Изображения

Изображения используются в задачах computer vision.

Каждое изображение фактически представляет собой массив чисел — значения пикселей.

## Аудио и временные ряды

Также часто используются:

- аудио сигналы  
- временные ряды  
- данные датчиков

Такие данные требуют специальных методов обработки.

## Итог

Machine learning может работать с разными типами данных:

- числовые  
- категориальные  
- текст  
- изображения  
- временные ряды
