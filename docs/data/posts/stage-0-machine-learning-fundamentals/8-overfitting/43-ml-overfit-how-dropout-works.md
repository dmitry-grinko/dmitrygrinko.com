# 43. Как dropout работает во время обучения?

Во время обучения dropout случайным образом отключает часть нейронов.

## Что происходит на каждой итерации

Для каждого слоя:

- выбирается случайная часть нейронов  
- их выход устанавливается равным нулю  
- остальные нейроны продолжают работать

Вероятность отключения обычно задаётся параметром dropout rate.

Например:

dropout = 0.5 означает, что половина нейронов отключается.

## Что это означает

На каждой итерации сеть фактически становится **немного другой моделью**.

## Итог

Во время обучения dropout случайно отключает часть нейронов на каждой итерации.
