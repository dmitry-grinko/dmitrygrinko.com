# 44. Что такое L2 регуляризация?

L2 регуляризация добавляет к функции потерь штраф за **квадраты параметров модели**.

## Формула

L2 penalty = λ Σ w²

## Основная идея

Большие коэффициенты увеличивают значение штрафа.

Поэтому модель стремится использовать меньшие значения параметров.

## Особенность

В отличие от L1, L2 обычно **не делает коэффициенты равными нулю**.

Она просто уменьшает их величину.

## Итог

L2 регуляризация уменьшает значения параметров модели и делает её более стабильной.
