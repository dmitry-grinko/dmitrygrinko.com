# 11. Что такое unsupervised learning?

Unsupervised learning — это тип обучения в machine learning, при котором модель обучается на **неразмеченных данных**.

Это означает, что dataset содержит только входные данные (features), но **правильные ответы или labels отсутствуют**.

В отличие от supervised learning, модель не получает сигнал о том, какое решение является правильным. Поэтому её задача — самостоятельно найти структуру или закономерности внутри данных.

## Основная идея

В unsupervised learning модель пытается понять, **как устроены данные**, не имея заранее заданной цели предсказания.

Она может пытаться:

- находить группы похожих объектов  
- обнаруживать скрытые структуры  
- уменьшать размерность данных  
- выявлять аномалии  

В этом смысле unsupervised learning больше похож на **анализ данных**, чем на прямое предсказание результата.

## Пример

Представим dataset с информацией о клиентах интернет-магазина.

| age | income | purchases |
|----|-------|----------|
| 25 | 40000 | 3 |
| 27 | 42000 | 4 |
| 55 | 90000 | 10 |

Если у нас нет labels (например, нет информации о сегменте клиента), модель может попытаться сама разделить клиентов на группы с похожим поведением.

Например:

- молодые покупатели  
- постоянные покупатели  
- клиенты с высоким доходом

Такой анализ может быть полезен для маркетинга или сегментации пользователей.

## Чем unsupervised learning отличается от других подходов

Главное отличие заключается в том, что модель **не знает правильных ответов**.

Она не может напрямую проверить правильность своих выводов. Вместо этого алгоритм ищет статистические структуры внутри данных.

## Итог

Unsupervised learning — это подход в машинном обучении, при котором модель анализирует неразмеченные данные и пытается самостоятельно находить закономерности или структуры внутри dataset.
