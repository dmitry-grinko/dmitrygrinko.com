# 33. Что такое saddle point?

Saddle point — это точка функции, в которой градиент равен нулю, но она **не является ни минимумом, ни максимумом**.

В одном направлении функция может увеличиваться, а в другом — уменьшаться.

## Геометрическая интуиция

Название происходит от формы седла.

Если двигаться вперёд-назад, поверхность может идти вверх.

Если двигаться в сторону, она может идти вниз.

## Почему это важно

Алгоритмы оптимизации могут замедляться в таких точках, потому что градиент становится очень маленьким.

## Итог

Saddle point — это точка, где градиент равен нулю, но она не является минимумом функции.
