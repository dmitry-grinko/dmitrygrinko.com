# 3. Чем overfitting отличается от underfitting?

Overfitting и underfitting — это два разных типа проблем обучения модели.

Они возникают по противоположным причинам.

## Overfitting

Модель слишком сложная и слишком точно подстраивается под training data.

Признаки:

- очень низкая training error  
- высокая ошибка на новых данных

## Underfitting

Модель слишком простая и не может уловить закономерности данных.

Признаки:

- высокая ошибка на training data  
- высокая ошибка на test data

## Ключевое различие

Overfitting означает, что модель **слишком хорошо запомнила training data**.

Underfitting означает, что модель **вообще не смогла выучить закономерности**.

## Итог

Overfitting связан с избыточной сложностью модели, а underfitting — с её недостаточной сложностью.
