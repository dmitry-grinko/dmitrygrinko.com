# 40. Как L2 регуляризация влияет на модель?

L2 регуляризация добавляет штраф за **квадраты коэффициентов модели**.

## Формула штрафа

penalty = λ Σ w²

## Основной эффект

L2 регуляризация уменьшает значения коэффициентов.

Но в отличие от L1, она редко делает их равными нулю.

## Почему это полезно

Это помогает:

- уменьшить переобучение  
- стабилизировать модель

## Практический результат

Модель становится менее чувствительной к шуму в данных.

## Итог

L2 регуляризация уменьшает коэффициенты модели и делает обучение более устойчивым.
