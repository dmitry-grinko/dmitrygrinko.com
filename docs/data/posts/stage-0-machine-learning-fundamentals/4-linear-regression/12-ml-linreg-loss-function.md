# 12. Что такое функция потерь (loss function) в линейной регрессии?

Функция потерь — это способ измерить, **насколько хорошо или плохо работает модель**.

Она вычисляет ошибку между предсказаниями модели и реальными значениями.

Во время обучения модель пытается **минимизировать значение функции потерь**.

## Общая идея

Если предсказание модели близко к реальному значению, ошибка маленькая.

Если предсказание сильно отличается, ошибка большая.

Функция потерь объединяет ошибки по всем наблюдениям dataset.

## Пример

Если модель предсказала:

prediction = 180000

а реальное значение:

target = 200000

ошибка составляет 20000.

Функция потерь учитывает такие ошибки для всех объектов.

## Роль функции потерь

Loss function используется для:

- оценки качества модели  
- обучения модели  
- настройки параметров

Именно она определяет, как модель будет корректировать свои коэффициенты.

## Итог

Функция потерь — это математическая функция, которая измеряет ошибку модели и используется для её обучения.
