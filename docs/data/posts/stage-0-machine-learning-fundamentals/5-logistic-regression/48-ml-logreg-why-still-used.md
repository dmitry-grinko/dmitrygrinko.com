# 48. Почему logistic regression всё ещё широко используется?

Несмотря на появление более сложных алгоритмов, logistic regression остаётся популярной моделью.

## Простота

Модель легко реализовать и обучить.

## Интерпретируемость

Коэффициенты модели можно интерпретировать.

Это важно в:

- медицине  
- финансах  
- экономике

## Быстрое обучение

Logistic regression обучается значительно быстрее многих сложных моделей.

## Надёжность

На табличных данных простые модели часто работают достаточно хорошо.

## Итог

Logistic regression популярна благодаря простоте, скорости и интерпретируемости.
