# 16. Как probability превращается в класс?

После того как модель вычисляет вероятность, необходимо преобразовать её в конкретный класс.

Для этого используется **порог классификации**.

## Основная идея

Если вероятность выше порога — объект относится к классу 1.

Если ниже — к классу 0.

## Пример

Пусть модель выдала:

P(class = 1) = 0.73

Если threshold = 0.5

то объект будет классифицирован как класс 1.

## Формально

если P ≥ threshold → класс 1  
если P < threshold → класс 0

## Итог

Вероятность превращается в класс через сравнение с пороговым значением.
