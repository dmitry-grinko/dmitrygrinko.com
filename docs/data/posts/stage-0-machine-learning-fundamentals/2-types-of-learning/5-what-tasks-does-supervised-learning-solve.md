# 5. Какие задачи решает supervised learning?

Supervised learning используется в задачах, где для каждого объекта можно определить **правильный результат**. Модель обучается на примерах, где известен ответ, и затем пытается предсказывать этот ответ для новых данных.

Такие задачи обычно формулируются как предсказание некоторого значения на основе признаков.

## Две основные группы задач

Большинство задач supervised learning можно разделить на два типа.

### Regression

Regression используется, когда нужно предсказать **числовое значение**.

Примеры:

- предсказание цены квартиры  
- прогнозирование продаж  
- оценка времени доставки  
- прогноз температуры  

В таких задачах модель пытается оценить **непрерывную величину**.

### Classification

Classification используется, когда нужно определить **категорию или класс объекта**.

Примеры:

- является ли письмо спамом  
- распознавание объектов на изображении  
- определение категории товара  
- обнаружение мошенничества  

В этих задачах модель выбирает один из нескольких возможных классов.

## Где это используется на практике

Supervised learning применяется в самых разных системах:

- кредитный скоринг в банковских системах  
- рекомендации товаров  
- фильтрация спама  
- распознавание изображений и речи  
- медицинская диагностика  

Общая идея остаётся одной и той же: модель учится на примерах с известным результатом.

## Итог

Supervised learning используется в задачах, где нужно предсказывать известный результат на основе данных.  
Наиболее распространённые типы таких задач — regression и classification.
