# 39. Почему некоторые loss функции работают с logits, а не с вероятностями?

Некоторые функции потерь принимают logits напрямую.

Это делается для **численной стабильности вычислений**.

## Проблема вероятностей

Когда вероятности очень близки к 0 или 1, вычисления могут становиться нестабильными.

Логарифмы могут давать очень большие значения.

## Решение

Loss функции могут объединять:

- softmax  
- cross entropy

в одну операцию.

Это уменьшает численные ошибки.

## Пример

Softmax cross entropy.

## Итог

Работа с logits повышает стабильность и точность вычислений.
