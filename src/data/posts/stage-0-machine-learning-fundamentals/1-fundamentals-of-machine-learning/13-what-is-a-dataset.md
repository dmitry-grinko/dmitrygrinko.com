# 13. Что такое dataset?

Dataset — это набор данных, который используется для обучения, проверки или тестирования модели машинного обучения.

Проще говоря, dataset — это коллекция примеров, на которых работает модель.

Каждая строка обычно представляет один объект, а столбцы содержат его характеристики (features) и иногда правильный ответ (label).

Пример простого dataset:

| area | rooms | price |
|-----|------|------|
| 45  | 1    | 120000 |
| 70  | 2    | 185000 |
| 80  | 3    | 240000 |

Здесь:

- `area` и `rooms` — features
- `price` — label

Модель использует такие данные, чтобы находить закономерности и учиться делать предсказания.

## Из чего обычно состоит dataset

В большинстве задач dataset включает:

- **features** — признаки объектов
- **label / target** — правильный ответ

Но в некоторых задачах (например, clustering) label может отсутствовать.

## В каком виде хранятся datasets

На практике datasets чаще всего выглядят как таблицы.

Распространённые форматы:

- CSV
- Parquet
- JSON
- базы данных
- файлы изображений или текстовые корпуса

Например:

- изображения для computer vision
- тексты для NLP
- таблицы пользовательских событий

## Библиотеки для работы с dataset

В Python чаще всего используют несколько библиотек.

**NumPy**

Библиотека для работы с многомерными числовыми массивами.
Многие ML-библиотеки используют её как базовый формат данных.

**Pandas**

Самая популярная библиотека для работы с табличными данными.

Она позволяет:

- загружать datasets
- очищать данные
- фильтровать строки
- преобразовывать признаки

Пример загрузки dataset:

```python
import pandas as pd

data = pd.read_csv("dataset.csv")
```

**PyTorch / TensorFlow datasets**

В deep learning часто используют специальные классы dataset для потоковой загрузки данных.

Это важно, когда dataset слишком большой и не помещается в память.

## Откуда берутся datasets

Datasets могут появляться из разных источников:

- логи пользовательских действий
- внутренние данные компании
- публичные datasets
- ручная разметка данных

Например:

- ImageNet — большой dataset изображений
- Common Crawl — огромный корпус веб-страниц
- Kaggle datasets — коллекции открытых данных для ML

## Размер datasets

Размер datasets может сильно отличаться.

Иногда это:

- несколько тысяч строк

А иногда:

- миллионы или миллиарды записей.

Для больших datasets часто используют распределённую обработку данных (например Spark).

## Итог

Dataset — это набор примеров, на которых модель обучается и проверяется.

Он содержит:

- признаки объектов
- иногда правильные ответы

И именно качество dataset обычно сильнее всего влияет на качество модели.
