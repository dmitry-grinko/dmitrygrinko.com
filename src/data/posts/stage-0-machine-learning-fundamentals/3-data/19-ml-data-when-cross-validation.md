# 19. Когда используют cross-validation?

Cross-validation используется в ситуациях, когда dataset относительно небольшой.

В таких случаях простое разделение данных на train и test может давать нестабильные результаты.

## Маленькие datasets

Если dataset содержит мало наблюдений, разделение на train и test может привести к тому, что:

- модель обучается на слишком маленьком наборе данных  
- тестовая выборка становится слишком маленькой

Cross-validation помогает использовать данные более эффективно.

## Настройка моделей

Cross-validation часто используется при:

- выборе алгоритма  
- настройке гиперпараметров  
- сравнении разных моделей

Он позволяет получить более надёжную оценку качества.

## Когда его не используют

При очень больших datasets cross-validation может быть слишком дорогим вычислительно.

В таких случаях обычно используют простое разделение dataset.

## Итог

Cross-validation чаще всего используют при небольших datasets и при настройке моделей.
