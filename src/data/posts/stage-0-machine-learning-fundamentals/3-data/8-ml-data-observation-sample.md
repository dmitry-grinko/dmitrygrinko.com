# 8. Что такое observation или sample?

Observation (или sample) — это один объект в dataset.

Каждая строка таблицы dataset представляет собой один observation.

## Пример

Dataset:

| age | income | purchases |
|----|-------|----------|
| 25 | 40000 | 3 |
| 32 | 60000 | 7 |

Каждая строка — это observation.

В данном примере observation описывает одного пользователя.

## Структура observation

Observation обычно содержит:

- набор признаков (features)  
- целевую переменную (target)

То есть один пример обучения модели.

## Почему observations важны

Количество observations влияет на способность модели обучаться.

Чем больше разнообразных примеров содержит dataset, тем лучше модель может обобщать закономерности.

## Итог

Observation или sample — это один пример данных, описывающий объект в dataset.
