# 45. Чем RMSE отличается от MAE?

RMSE и MAE обе измеряют ошибку модели, но делают это по-разному.

## MAE

MAE использует абсолютные значения ошибок.

MAE = Σ |y − ŷ|

Все ошибки учитываются одинаково.

## RMSE

RMSE использует квадрат ошибки.

RMSE = √(Σ (y − ŷ)²)

Большие ошибки получают значительно больший вес.

## Практическое различие

RMSE сильнее наказывает модель за большие ошибки.

MAE более устойчив к выбросам.

## Итог

RMSE усиливает влияние больших ошибок, а MAE учитывает все ошибки равномерно.
