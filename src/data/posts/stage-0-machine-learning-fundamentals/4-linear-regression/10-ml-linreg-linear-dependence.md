# 10. Что такое линейная зависимость между переменными?

Линейная зависимость означает, что изменение одной переменной приводит к **пропорциональному изменению другой**.

Такую зависимость можно описать прямой линией.

## Пример

Если цена квартиры зависит от площади:

каждый дополнительный квадратный метр увеличивает цену на одинаковую величину.

Это линейная зависимость.

## Графическая интерпретация

Если построить график:

- по оси X — признак  
- по оси Y — target

линейная зависимость будет выглядеть как прямая линия.

![Linear relationship](image.png)

## Почему это важно

Линейная регрессия предполагает, что связь между признаками и target **примерно линейная**.

Если зависимость сильно нелинейная, линейная модель может работать плохо.

## Итог

Линейная зависимость означает, что между переменными существует связь, которую можно описать прямой линией.
