# 35. Что такое standardization признаков?

Standardization признаков — это преобразование числовых признаков так, чтобы:

- среднее значение признака стало равно 0
- стандартное отклонение стало равно 1

Это один из способов привести признаки к сопоставимому масштабу перед обучением модели.

В отличие от normalization, где значения приводятся к диапазону 0–1, standardization центрирует данные вокруг среднего значения.

## Как работает standardization

Обычно используется следующая формула:

x_standardized = (x − mean) / std

Где:

- x — исходное значение
- mean — среднее значение признака
- std — стандартное отклонение

После преобразования значения могут быть как положительными, так и отрицательными.

## Пример

Исходные значения признака:

| age |
|----|
| 20 |
| 40 |
| 60 |

Среднее значение:

mean = 40

Стандартное отклонение примерно:

std ≈ 16.33

Теперь преобразуем значения.

Для 20:

(20 − 40) / 16.33 ≈ −1.22

Для 40:

(40 − 40) / 16.33 = 0

Для 60:

(60 − 40) / 16.33 ≈ 1.22

После standardization:

| age |
|----|
| -1.22 |
| 0 |
| 1.22 |

## Пример кода

В Python standardization часто делают с помощью библиотеки scikit-learn.

```python
from sklearn.preprocessing import StandardScaler
import numpy as np

X = np.array([[20], [40], [60]])

scaler = StandardScaler()

X_scaled = scaler.fit_transform(X)

print(X_scaled)
```

В этом примере:

- StandardScaler вычисляет среднее значение и стандартное отклонение
- затем преобразует все значения признака по формуле standardization

## Когда используется standardization

Standardization особенно полезна для алгоритмов, чувствительных к масштабу признаков:

- линейная регрессия
- логистическая регрессия
- нейронные сети
- SVM
- методы на основе расстояний

## Итог

Standardization — это преобразование признаков, при котором данные центрируются вокруг среднего значения и масштабируются с использованием стандартного отклонения. Это помогает модели корректно работать с признаками разного масштаба.
