# 10. Почему метрики обычно вычисляются на validation или test dataset?

Метрики вычисляются на validation или test dataset, потому что эти данные **не использовались при обучении модели**.

Это позволяет объективно оценить качество модели.

## Проблема training dataset

Если вычислять метрики на training data, модель может выглядеть лучше, чем она есть на самом деле.

Это происходит из-за переобучения.

## Роль validation и test данных

Validation dataset используется для настройки модели.

Test dataset используется для финальной оценки.

## Итог

Метрики вычисляют на новых данных, чтобы оценить способность модели обобщать.
