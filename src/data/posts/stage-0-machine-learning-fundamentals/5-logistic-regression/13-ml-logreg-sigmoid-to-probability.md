# 13. Как sigmoid преобразует значения в вероятности?

Sigmoid принимает любое число и преобразует его в значение между 0 и 1.

Она делает это через экспоненциальную функцию.

Формула:

σ(z) = 1 / (1 + e⁻ᶻ)

где:

z — линейная комбинация признаков.

## Пример

Если:

z = 0

тогда:

σ(0) = 0.5

Если:

z = 3

σ(3) ≈ 0.95

Если:

z = −3

σ(−3) ≈ 0.05

## Интерпретация

- большие положительные значения → вероятность близка к 1  
- большие отрицательные значения → вероятность близка к 0  

## Итог

Sigmoid преобразует линейный выход модели в вероятность через нелинейное преобразование.
