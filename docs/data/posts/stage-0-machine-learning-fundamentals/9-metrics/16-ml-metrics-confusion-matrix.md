# 16. Что такое confusion matrix?

Confusion matrix — это таблица, которая показывает **как именно модель ошибается при классификации**.

Она сравнивает предсказания модели с реальными значениями.

## Структура

Для бинарной классификации матрица содержит четыре типа результатов.

|                | Predicted Positive | Predicted Negative |
|---------------|-------------------|-------------------|
| Actual Positive | True Positive | False Negative |
| Actual Negative | False Positive | True Negative |

## Что она показывает

Confusion matrix позволяет увидеть:

- какие ошибки делает модель  
- какие классы путаются между собой

## Итог

Confusion matrix показывает распределение правильных и неправильных предсказаний модели.
