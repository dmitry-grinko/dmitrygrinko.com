# 35. Как количество признаков влияет на линейную регрессию?

Количество признаков напрямую влияет на сложность модели.

Чем больше признаков используется, тем больше параметров должна обучить модель.

## Увеличение числа параметров

Каждый признак добавляет новый коэффициент.

Если модель имеет:

10 признаков → 10 коэффициентов  
100 признаков → 100 коэффициентов

Это увеличивает сложность модели.

## Риск переобучения

Если признаков слишком много, модель может начать подстраиваться под шум в данных.

Это приводит к переобучению.

Особенно если dataset небольшой.

## Вычислительная сложность

Большое количество признаков увеличивает:

- время обучения  
- сложность оптимизации

## Практика

Поэтому в ML часто используют:

- feature selection  
- dimensionality reduction

## Итог

Увеличение числа признаков делает модель более гибкой, но также повышает риск переобучения и усложняет обучение.
