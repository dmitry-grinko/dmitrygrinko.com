# 50. Почему важно анализировать несколько метрик одновременно?

Одна метрика редко отражает все аспекты качества модели.

Разные метрики показывают разные типы ошибок.

## Пример

Модель может иметь:

- высокую accuracy  
- низкий recall

Это означает, что она пропускает много положительных объектов.

## Практический подход

Часто анализируют сразу несколько метрик:

- precision  
- recall  
- F1  
- AUC

## Итог

Анализ нескольких метрик позволяет получить более полное понимание качества модели.
