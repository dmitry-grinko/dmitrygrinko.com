# 36. Что такое convergence в оптимизации?

Convergence означает состояние, когда алгоритм оптимизации **перестаёт существенно изменять параметры модели**.

Это обычно происходит, когда функция потерь больше не уменьшается заметно.

## Что происходит при convergence

Во время обучения:

- градиенты становятся маленькими  
- обновления параметров уменьшаются  
- функция потерь стабилизируется

## Практическое значение

Это означает, что алгоритм приблизился к минимуму функции.

## Итог

Convergence — это состояние, когда процесс оптимизации стабилизируется и параметры модели почти перестают изменяться.
