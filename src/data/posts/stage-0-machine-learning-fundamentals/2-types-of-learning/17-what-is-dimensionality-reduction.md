# 17. Что такое dimensionality reduction?

Dimensionality reduction — это процесс уменьшения количества признаков (features) в dataset.

В некоторых datasets число признаков может быть очень большим. Это может усложнять обучение модели и анализ данных.

Dimensionality reduction помогает представить данные в **более компактной форме**, сохраняя при этом важную информацию.

## Почему возникает высокая размерность

Во многих задачах dataset может содержать сотни или тысячи признаков.

Например:

- изображения могут содержать тысячи пикселей  
- текстовые данные могут иметь тысячи слов  
- биологические данные могут содержать большое количество измерений

Работа с такими данными может быть сложной.

## Основная идея

Методы уменьшения размерности пытаются преобразовать исходные данные так, чтобы:

- количество признаков стало меньше  
- важная информация сохранилась

В результате dataset становится более компактным и удобным для анализа.

## Пример

Представим dataset с сотней признаков.

Метод dimensionality reduction может преобразовать его в dataset с двумя или тремя новыми признаками, которые сохраняют основные закономерности данных.

Это позволяет:

- визуализировать данные  
- упростить обучение моделей  
- убрать избыточные признаки

## Итог

Dimensionality reduction — это процесс уменьшения числа признаков в dataset при сохранении основной структуры данных.
