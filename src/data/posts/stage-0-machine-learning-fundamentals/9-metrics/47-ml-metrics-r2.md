# 47. Что такое R² (coefficient of determination)?

R² — это метрика regression, которая показывает **какую долю вариации данных объясняет модель**.

Она измеряет, насколько хорошо модель объясняет зависимость между признаками и целевой переменной.

## Диапазон значений

R² обычно принимает значения от:

0 до 1

- 0 — модель ничего не объясняет  
- 1 — идеальное объяснение данных

## Интерпретация

Если R² = 0.8, это означает, что модель объясняет 80% вариации данных.

## Итог

R² показывает, насколько хорошо модель объясняет изменчивость данных.
