# 46. Какие методы используются для работы с class imbalance?

Существует несколько методов, которые помогают обучать модели на несбалансированных datasets.

## Изменение dataset

Один из подходов — изменить распределение данных.

Undersampling

Удаляется часть объектов из большинства класса.

Oversampling

Добавляются дополнительные примеры редкого класса.

## Генерация новых данных

Иногда создают новые примеры редкого класса.

Например:

SMOTE (Synthetic Minority Over-sampling Technique).

Этот метод генерирует новые синтетические примеры.

## Использование весов классов

Некоторые алгоритмы позволяют задавать веса классов.

Ошибки на редком классе получают более высокий штраф.

Это заставляет модель уделять больше внимания редким объектам.

## Итог

Для работы с class imbalance используют:

- undersampling  
- oversampling  
- генерацию синтетических данных  
- веса классов
