# 12. Какие задачи решает unsupervised learning?

Unsupervised learning используется в ситуациях, когда dataset не содержит правильных ответов, но всё равно важно понять структуру данных.

Такие методы помогают исследовать dataset, выявлять скрытые закономерности и находить группы похожих объектов.

## Кластеризация объектов

Одна из самых распространённых задач — **clustering**.

В этой задаче модель пытается разделить объекты на группы так, чтобы объекты внутри группы были похожи друг на друга.

Например, clustering может использоваться для:

- сегментации клиентов  
- группировки похожих товаров  
- анализа поведения пользователей  

## Поиск структуры в данных

Иногда dataset содержит сложные зависимости между признаками. Unsupervised методы могут помочь обнаружить такие структуры.

Это может быть полезно, например, при исследовании больших datasets, где структура данных заранее неизвестна.

## Уменьшение размерности данных

В некоторых задачах dataset содержит большое количество признаков.

Методы уменьшения размерности позволяют:

- упростить данные  
- убрать избыточную информацию  
- сделать dataset более удобным для анализа

## Поиск аномалий

Unsupervised learning также используется для обнаружения необычных или редких объектов.

Например:

- обнаружение мошеннических транзакций  
- выявление аномального поведения пользователей  
- обнаружение ошибок в данных

## Итог

Unsupervised learning используется для задач, где необходимо исследовать структуру данных, например:

- clustering  
- уменьшение размерности  
- обнаружение аномалий
