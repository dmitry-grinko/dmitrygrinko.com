# 39. Когда PR curve полезнее ROC curve?

PR curve полезнее ROC curve в задачах с **сильным дисбалансом классов**.

## Причина

Когда отрицательных объектов намного больше, ROC curve может выглядеть слишком оптимистично.

PR curve более чувствительна к изменениям в положительном классе.

## Примеры задач

- обнаружение мошенничества  
- поиск аномалий  
- медицинская диагностика

## Практический эффект

PR curve лучше показывает способность модели находить редкие положительные объекты.

## Итог

PR curve полезнее ROC curve в задачах с редкими положительными событиями.
