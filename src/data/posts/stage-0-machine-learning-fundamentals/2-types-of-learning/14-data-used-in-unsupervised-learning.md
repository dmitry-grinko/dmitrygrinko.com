# 14. Какие данные используются в unsupervised learning?

В unsupervised learning используются **неразмеченные данные**.

Это означает, что dataset содержит только входные признаки, но не содержит правильных ответов.

Каждый объект в dataset описывается набором признаков, и задача модели — анализировать эти признаки, чтобы обнаружить структуру в данных.

## Как выглядит dataset

Например, dataset может выглядеть так:

| age | income | purchases |
|----|-------|----------|
| 25 | 40000 | 3 |
| 27 | 42000 | 4 |
| 55 | 90000 | 10 |

В отличие от supervised learning здесь отсутствует колонка с правильным результатом.

Все данные представлены только характеристиками объектов.

## Откуда берутся такие данные

На практике неразмеченные datasets встречаются очень часто.

Например:

- данные о поведении пользователей  
- транзакции клиентов  
- изображения  
- тексты  
- данные датчиков  

Во многих случаях собрать такие данные относительно легко, потому что они автоматически генерируются системами.

## Почему такие данные полезны

Хотя в dataset нет правильных ответов, он всё равно может содержать важную информацию.

Например, анализируя такие данные можно:

- обнаружить группы похожих пользователей  
- понять структуру dataset  
- выявить необычные объекты

## Итог

В unsupervised learning используются datasets, которые содержат только признаки объектов и не имеют заранее известных правильных ответов.
