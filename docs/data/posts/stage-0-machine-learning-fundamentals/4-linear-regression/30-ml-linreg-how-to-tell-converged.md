# 30. Как понять, что модель сошлась?

Есть несколько признаков, по которым можно понять, что модель достигла сходимости.

## Стабилизация функции потерь

Если значение функции потерь перестаёт заметно уменьшаться между итерациями, модель может считаться сошедшейся.

## Маленькие изменения параметров

Если параметры модели изменяются очень незначительно на каждом шаге, обучение фактически завершилось.

## Малый градиент

Когда градиент функции потерь становится близким к нулю, это означает, что алгоритм находится рядом с минимумом.

## График обучения

Часто строят график:

iteration → loss

Если кривая выходит на плато, это признак сходимости.

## Итог

Модель считается сошедшейся, когда ошибка перестаёт заметно уменьшаться и параметры стабилизируются.
