# 34. Почему cross entropy лучше подходит для вероятностей?

Cross entropy специально разработана для оценки вероятностных предсказаний.

## Основная идея

Функция потерь зависит от **вероятности правильного класса**.

Если модель предсказывает высокую вероятность правильного класса, loss становится маленьким.

Если вероятность низкая, loss становится большим.

## Пример

Правильный класс = 1

p = 0.9 → маленький loss
p = 0.5 → средний loss
p = 0.1 → большой loss

## Следствие

Модель учится:

- увеличивать вероятность правильного класса
- уменьшать вероятность неправильного класса

## Итог

Cross entropy напрямую оптимизирует вероятностные предсказания модели.
