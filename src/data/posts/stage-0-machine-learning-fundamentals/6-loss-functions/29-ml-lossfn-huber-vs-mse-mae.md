# 29. Чем Huber loss отличается от MSE и MAE?

Huber loss объединяет свойства двух функций потерь.

## Поведение

Для маленьких ошибок:

Huber ≈ MSE

Для больших ошибок:

Huber ≈ MAE

## Следствие

Модель:

- чувствительна к небольшим ошибкам  
- устойчива к выбросам

## Сравнение

MSE:

сильно реагирует на выбросы

MAE:

устойчив к выбросам, но менее гладкий

Huber:

компромисс между ними

## Итог

Huber loss — это гибрид MSE и MAE.
