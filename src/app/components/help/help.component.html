Help & Keyboard Shortcuts

Learn how to navigate the blog efficiently using keyboard shortcuts and other features.

🌐 Global Shortcuts

These shortcuts work from anywhere on the site:

Ctrl + S
Toggle Search Open or close the search input
Ctrl + C
Cycle Categories Navigate through main categories
Esc
Close Search Close search overlay and results

🔍 Search Navigation

When search results are visible:

↑ ↓
Navigate Results Move focus between search results
Enter
Open Result Open the currently focused search result

📄 Post Card Navigation

On blog listing pages (Home, Categories, Subcategories):

↑ ↓ ← →
Navigate Posts Move focus between post cards
Enter
Open Post Open the currently focused post
