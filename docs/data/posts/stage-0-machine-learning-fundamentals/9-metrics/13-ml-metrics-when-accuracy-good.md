# 13. Когда accuracy является хорошей метрикой?

Accuracy является хорошей метрикой, когда **классы в dataset распределены примерно равномерно**.

В таких задачах все ошибки имеют примерно одинаковую важность.

## Примеры

Задачи, где accuracy работает хорошо:

- распознавание цифр  
- классификация изображений  
- простые задачи бинарной классификации

## Почему это работает

Когда классы сбалансированы, высокая accuracy действительно означает хорошую модель.

## Ограничение

Если классы сильно несбалансированы, accuracy может вводить в заблуждение.

## Итог

Accuracy хорошо работает в задачах с примерно равным количеством объектов каждого класса.
