# 45. Как регуляризация влияет на параметры модели?

Регуляризация изменяет процесс оптимизации параметров.

Во время обучения алгоритм минимизирует:

loss + regularization penalty

## Что происходит

Если коэффициенты модели становятся слишком большими, штраф увеличивается.

Это заставляет алгоритм уменьшать параметры.

## Следствие

Параметры модели становятся:

- меньше по величине  
- более стабильными

## Практический эффект

Модель становится менее чувствительной к шуму в данных.

## Итог

Регуляризация ограничивает рост параметров модели.
