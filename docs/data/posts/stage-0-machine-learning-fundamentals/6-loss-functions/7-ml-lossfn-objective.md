# 7. Что такое objective function?

Objective function — это функция, которую алгоритм обучения пытается **минимизировать или максимизировать**.

В большинстве задач машинного обучения objective function нужно минимизировать.

## Связь с loss function

Часто objective function включает:

- функцию потерь  
- дополнительные члены, например регуляризацию

## Пример

objective function может выглядеть так:

objective = loss + regularization

## Почему это важно

Objective function определяет, какие параметры модели считаются оптимальными.

## Итог

Objective function — это функция, которую оптимизирует алгоритм обучения.
