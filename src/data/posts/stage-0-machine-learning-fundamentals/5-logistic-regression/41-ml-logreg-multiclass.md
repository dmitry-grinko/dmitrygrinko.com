# 41. Что такое multiclass classification?

Multiclass classification — это задача классификации, в которой существует **больше двух классов**.

Модель должна определить, к какому из нескольких возможных классов относится объект.

## Пример

Классификация изображений животных:

- cat  
- dog  
- horse  
- bird  

Каждый объект может принадлежать только одному классу.

## Чем отличается от бинарной классификации

Бинарная классификация:

два класса (0 или 1)

Multiclass classification:

три и более классов.

## Где встречается

Multiclass задачи часто возникают в:

- распознавании изображений  
- классификации текстов  
- системах рекомендаций  

## Итог

Multiclass classification — это задача классификации, в которой модель выбирает один класс из нескольких возможных.
