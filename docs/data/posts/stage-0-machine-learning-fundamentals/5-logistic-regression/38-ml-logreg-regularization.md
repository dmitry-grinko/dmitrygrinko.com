# 38. Что такое регуляризация в logistic regression?

Регуляризация — это метод, который добавляет **штраф за большие значения коэффициентов модели**.

Это помогает уменьшить переобучение.

## Основная идея

К функции потерь добавляется дополнительный член:

loss = log_loss + penalty

Penalty зависит от величины коэффициентов модели.

## Почему это полезно

Регуляризация заставляет модель:

- использовать более простые зависимости  
- избегать слишком больших коэффициентов

## Результат

Модель становится:

- более устойчивой  
- лучше обобщает новые данные

## Итог

Регуляризация ограничивает величину параметров модели и помогает бороться с переобучением.
