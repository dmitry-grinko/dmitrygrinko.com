# 39. Как обнаружить multicollinearity?

Существует несколько способов обнаружения multicollinearity.

Обычно используют статистический анализ признаков.

## Корреляционная матрица

Один из самых простых методов — вычислить корреляции между признаками.

Если два признака имеют очень высокую корреляцию, это может быть признаком multicollinearity.

## VIF (Variance Inflation Factor)

Более формальный метод — вычисление VIF.

VIF показывает, насколько сильно дисперсия коэффициента увеличивается из-за корреляции признаков.

Обычно считается:

VIF > 5 или 10 может указывать на проблему.

## Анализ коэффициентов модели

Если коэффициенты:

- имеют неожиданные значения  
- сильно меняются между обучениями

это может быть сигналом multicollinearity.

## Итог

Multicollinearity можно обнаружить через:

- корреляционный анализ  
- VIF  
- анализ поведения коэффициентов модели.
