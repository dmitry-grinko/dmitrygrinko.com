# 27. Чем precision отличается от recall?

Precision и recall измеряют **разные аспекты качества модели**.

## Precision

Отвечает на вопрос:

> Насколько точны положительные предсказания модели?

precision = TP / (TP + FP)

## Recall

Отвечает на вопрос:

> Сколько реальных положительных объектов модель обнаружила?

recall = TP / (TP + FN)

## Основное различие

Precision фокусируется на **ошибках типа false positive**.

Recall фокусируется на **ошибках типа false negative**.

## Итог

Precision измеряет точность положительных предсказаний, а recall — полноту обнаружения положительных случаев.
