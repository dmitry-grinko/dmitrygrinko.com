# 49. Почему разные задачи требуют разные метрики?

Разные задачи имеют **разные цели и разные типы ошибок**.

Поэтому одна универсальная метрика не подходит для всех случаев.

## Примеры

Классификация:

- accuracy  
- precision  
- recall  
- F1

Регрессия:

- MSE  
- MAE  
- RMSE

## Практический аспект

Метрика должна отражать то, что считается хорошим результатом в конкретной задаче.

## Итог

Метрики выбираются в зависимости от типа задачи и цели модели.
