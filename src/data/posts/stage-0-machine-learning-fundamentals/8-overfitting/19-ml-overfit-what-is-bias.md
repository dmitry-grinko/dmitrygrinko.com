# 19. Что такое bias в модели?

Bias — это ошибка, возникающая из-за **слишком простых предположений модели о данных**.

Модель с высоким bias не способна уловить сложные закономерности.

## Пример

Если реальные данные имеют сложную зависимость, а используется линейная модель, она может не описывать данные правильно.

## Признаки высокого bias

- высокая training error  
- высокая validation error

## Причина

Модель слишком ограничена и не может описать структуру данных.

## Итог

Bias — это ошибка, возникающая из-за недостаточной сложности модели.
