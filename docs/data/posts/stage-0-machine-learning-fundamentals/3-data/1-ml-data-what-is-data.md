# 1. Что такое данные (data) в machine learning?

Данные — это информация, на которой обучается модель машинного обучения.  
Модель не "понимает" задачу сама по себе. Она анализирует примеры из данных и пытается найти закономерности.

В machine learning данные обычно представлены в виде **таблицы**, где:

- строки — отдельные объекты  
- столбцы — характеристики этих объектов

Например:

| area | rooms | price |
|-----|------|------|
| 45 | 1 | 120000 |
| 70 | 2 | 185000 |

Здесь каждая строка — это объект (квартира), а столбцы — её характеристики.

Именно на основе таких примеров модель учится делать предсказания.

Важный момент: модель **не знает реальный мир**. Она знает только те данные, которые ей показали.

## Итог

Данные — это набор примеров, из которых модель извлекает закономерности для решения задачи.
