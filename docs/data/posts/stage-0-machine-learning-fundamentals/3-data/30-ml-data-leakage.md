# 30. Что такое data leakage?

Data leakage — это ситуация, когда в training dataset случайно попадает информация, которая **не должна быть доступна во время обучения**.

Это приводит к тому, что модель показывает слишком хорошие результаты во время тестирования.

## Пример

Представим модель, которая предсказывает, будет ли клиент делать покупку.

Если в dataset случайно попадёт информация о том, **произошла ли покупка**, модель сможет легко "предсказать" результат.

Но это не будет настоящим обучением.

## Почему это опасно

Data leakage создаёт ложное впечатление, что модель работает хорошо.

Однако при использовании в реальных условиях её качество может резко снизиться.

## Где возникает leakage

Data leakage часто возникает из-за:

- неправильного разделения dataset  
- использования будущей информации  
- ошибок в подготовке данных

## Итог

Data leakage — это ошибка в данных, при которой модель получает доступ к информации, которая не должна использоваться при обучении.
