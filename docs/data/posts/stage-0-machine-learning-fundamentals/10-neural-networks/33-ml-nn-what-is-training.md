# 33. Что означает обучение нейронной сети?

Обучение нейронной сети — это процесс **подбора параметров (весов и bias)**, которые минимизируют ошибку модели.

Идея проста:

1. сеть делает предсказание
2. вычисляется ошибка
3. параметры немного изменяются
4. процесс повторяется

## Пример

Допустим, модель предсказывает цену квартиры.

Истинная цена:  
200000

Предсказание сети:  
180000

Ошибка:  
20000

Алгоритм обучения пытается изменить веса так, чтобы в следующий раз ошибка была меньше.

## Как происходит обучение

Обучение состоит из повторения двух шагов:

- forward pass — вычисление предсказания
- backward pass — корректировка параметров

## Итог

Обучение нейронной сети — это процесс изменения параметров модели для уменьшения ошибки.
