# 19. Когда threshold стоит изменить?

Иногда стандартный threshold = 0.5 не является оптимальным.

Его изменяют, когда разные типы ошибок имеют разную стоимость.

## Пример

В задачах обнаружения мошенничества важно **не пропустить мошенническую операцию**.

Поэтому threshold могут уменьшить.

Например:

threshold = 0.3

Это позволит обнаруживать больше подозрительных операций.

## Другой пример

В медицинской диагностике иногда лучше увеличить threshold, чтобы уменьшить количество ложных тревог.

## Настройка threshold

Порог обычно выбирается на основе:

- precision  
- recall  
- ROC curve

## Итог

Threshold изменяют, когда необходимо контролировать баланс между различными типами ошибок.
