# 19. Как loss функция влияет на градиенты?

Градиенты вычисляются как **производные функции потерь**.

Поэтому форма loss функции напрямую определяет градиенты.

## Основная идея

Если loss меняется резко, градиенты будут большими.

Если loss меняется медленно, градиенты будут маленькими.

## Следствие

Loss функция влияет на:

- скорость обучения  
- стабильность оптимизации  
- направление обновления параметров

## Пример

Cross entropy даёт более информативные градиенты для классификации.

## Итог

Loss функция определяет форму градиентов, используемых при обучении модели.
