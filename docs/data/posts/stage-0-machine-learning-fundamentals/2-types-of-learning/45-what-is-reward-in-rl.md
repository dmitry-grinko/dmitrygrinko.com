# 45. Что такое reward?

Reward — это сигнал обратной связи, который агент получает после выполнения действия.

Награда показывает, насколько полезным было выбранное действие.

## Основная роль reward

Reward направляет обучение агента.

Если действие приводит к хорошему результату, агент получает положительную награду.

Если действие приводит к плохому результату, награда может быть отрицательной.

## Пример

В игре награда может быть:

- +1 за успешное действие  
- -1 за ошибку  
- 0 за нейтральный результат

Агент пытается выбирать действия, которые увеличивают суммарную награду.

## Накопление награды

В reinforcement learning обычно важна не отдельная награда, а **суммарная награда за последовательность действий**.

Это заставляет агента думать о долгосрочных последствиях своих решений.

## Итог

Reward — это сигнал, который сообщает агенту, насколько успешным было его действие.
