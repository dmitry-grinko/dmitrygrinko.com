# 3. Как выглядит математическая форма линейной регрессии?

Линейная регрессия описывает зависимость между признаками и целевой переменной с помощью линейного уравнения.

Самая простая форма выглядит так:

y = w₁x₁ + b

где:

- x₁ — признак  
- w₁ — коэффициент признака  
- b — свободный член (intercept)  
- y — предсказание модели

## Несколько признаков

Если признаков несколько, уравнение расширяется:

y = w₁x₁ + w₂x₂ + w₃x₃ + ... + b

Каждый признак имеет свой коэффициент.

## Что означает это уравнение

Модель берёт значения признаков, умножает их на коэффициенты и складывает.

Результат этой операции и есть предсказание модели.

## Пример

Если модель выглядит так:

price = 2000 × area + 10000

это означает:

каждый дополнительный квадратный метр увеличивает цену на 2000.

## Итог

Линейная регрессия представляет предсказание как линейную комбинацию признаков и коэффициентов.
