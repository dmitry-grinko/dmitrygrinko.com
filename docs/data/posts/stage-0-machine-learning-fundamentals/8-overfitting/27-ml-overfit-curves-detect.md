# 27. Как learning curves помогают обнаружить переобучение?

Learning curves позволяют визуально увидеть разницу между training и validation ошибками.

## Типичный сценарий переобучения

Во время обучения:

- training error продолжает уменьшаться  
- validation error сначала уменьшается, затем начинает расти

Это означает, что модель начинает запоминать training dataset.

## Разрыв между кривыми

Чем больше разница между training и validation ошибками, тем выше вероятность переобучения.

## Практическое использование

Learning curves помогают понять:

- когда остановить обучение  
- нужна ли регуляризация  
- хватает ли данных

## Итог

Learning curves помогают обнаружить переобучение через анализ поведения ошибок.
