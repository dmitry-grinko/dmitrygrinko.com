# 4. Почему одна метрика редко отражает всю картину качества модели?

Одна метрика редко отражает полную картину, потому что **разные метрики измеряют разные аспекты качества модели**.

Одна и та же модель может выглядеть хорошей по одной метрике и плохой по другой.

## Пример

В задаче обнаружения мошенничества:

- accuracy может быть высокой  
- но модель может пропускать почти все мошеннические операции

В этом случае важнее смотреть на recall или precision.

## Разные аспекты качества

Метрики могут измерять:

- точность предсказаний  
- способность находить редкие события  
- количество ложных тревог

## Итог

Для правильной оценки модели обычно используют несколько метрик.
