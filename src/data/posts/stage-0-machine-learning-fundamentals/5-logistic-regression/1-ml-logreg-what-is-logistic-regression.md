# 1. Что такое logistic regression?

Logistic regression — это алгоритм машинного обучения, который используется для **задач классификации**.

Несмотря на слово regression в названии, модель не предсказывает числовое значение.  
Она предсказывает **вероятность того, что объект принадлежит определённому классу**.

Например, модель может оценивать вероятность того, что:

- письмо является спамом
- клиент совершит покупку
- транзакция является мошеннической

Если вероятность высокая, объект относится к положительному классу.

## Основная идея

Модель принимает признаки объекта и вычисляет вероятность того, что объект принадлежит одному из классов.

Например:

P(spam | email features) = 0.87

Это означает, что модель считает вероятность спама равной 87%.

## Почему модель называется regression

Название связано с тем, что внутри модели используется **линейная комбинация признаков**, похожая на линейную регрессию.

Но результат этой линейной функции преобразуется в вероятность.

## Итог

Logistic regression — это модель классификации, которая предсказывает вероятность принадлежности объекта к определённому классу.
