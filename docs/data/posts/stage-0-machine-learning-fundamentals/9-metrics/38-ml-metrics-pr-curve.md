# 38. Что такое PR curve (precision–recall curve)?

PR curve — это график, который показывает зависимость между **precision и recall**.

Он строится при разных значениях порога классификации.

## Оси графика

- по оси X — recall  
- по оси Y — precision

![precision recall curve](image.png)

## Что показывает PR curve

График показывает компромисс между:

- точностью положительных предсказаний
- полнотой обнаружения положительных объектов

## Практическое значение

PR curve особенно полезна при сильном дисбалансе классов.

## Итог

PR curve показывает зависимость между precision и recall при разных threshold.
