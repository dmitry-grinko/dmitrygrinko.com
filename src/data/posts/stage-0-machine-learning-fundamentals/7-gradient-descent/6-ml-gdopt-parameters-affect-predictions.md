# 6. Как параметры модели влияют на предсказания?

Параметры модели напрямую определяют **какое предсказание будет сделано для каждого входного объекта**.

Любое изменение параметров изменяет поведение модели.

## Пример

Линейная модель:

y = wx + b

Если:

w = 1

то предсказания растут медленно.

Если:

w = 10

то предсказания меняются гораздо быстрее.

## Интерпретация

Параметры модели определяют:

- наклон функции  
- форму границы решений  
- чувствительность к признакам

## Что делает обучение

Алгоритм обучения корректирует параметры так, чтобы предсказания становились точнее.

## Итог

Параметры модели управляют тем, как входные данные преобразуются в предсказания.
