# 2. Почему обучение модели формулируется как задача оптимизации?

Обучение модели можно формально описать как задачу оптимизации, потому что цель обучения — **найти параметры модели, которые минимизируют функцию потерь**.

Это превращает обучение в чётко сформулированную математическую задачу.

## Формальная постановка

Пусть:

- θ — параметры модели  
- L(θ) — функция потерь  

Тогда задача обучения выглядит так:

minimize L(θ)

То есть нужно найти такие параметры θ, при которых функция потерь минимальна.

## Почему это удобно

Такая формулировка позволяет использовать мощные методы оптимизации:

- gradient descent  
- stochastic gradient descent  
- Adam и другие оптимизаторы

Эти методы разработаны именно для поиска минимума функций.

## Итог

Обучение модели формулируется как задача оптимизации, потому что цель обучения — минимизировать функцию потерь.
