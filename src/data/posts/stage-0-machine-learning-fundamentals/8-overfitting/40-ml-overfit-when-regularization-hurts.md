# 40. Когда регуляризация может ухудшить модель?

Регуляризация может ухудшить модель, если она применяется слишком сильно.

## Основная проблема

Слишком большой коэффициент регуляризации может сделать модель **слишком простой**.

## Что происходит

Модель начинает сильно ограничивать параметры.

Она теряет способность описывать реальные закономерности данных.

## Последствие

Это может привести к underfitting.

Модель будет плохо работать:

- на training dataset  
- на новых данных

## Итог

Слишком сильная регуляризация может привести к недообучению модели.
