# 23. Чем log loss отличается от MSE?

Log loss и MSE — это разные функции потерь, предназначенные для разных типов задач.

## MSE

Mean Squared Error используется в задачах регрессии.

Он измеряет квадрат разницы между:

prediction и target.

MSE предполагает, что предсказания — это **числовые значения**.

## Log loss

Log loss используется в задачах классификации.

Он измеряет, насколько хорошо модель предсказывает **вероятности классов**.

## Основное различие

MSE:

- подходит для числовых значений  
- плохо работает с вероятностями

Log loss:

- специально разработан для классификации  
- учитывает уверенность предсказаний

## Итог

MSE используется для регрессии, а log loss — для классификации.
