# 24. Что такое выходной слой (output layer)?

Выходной слой (output layer) — это последний слой нейронной сети, который выдаёт **финальный результат модели**.

То есть именно здесь внутренние вычисления сети превращаются в ответ, который можно интерпретировать.

## Что выдаёт output layer

Это зависит от задачи.

### Бинарная классификация

Если нужно ответить «да / нет», выходной слой часто содержит один нейрон с sigmoid.

Пример:

- вероятность спама  
- вероятность мошенничества  
- вероятность болезни

### Многоклассовая классификация

Если классов несколько, выходной слой может содержать несколько нейронов и использовать softmax.

Пример:

- cat  
- dog  
- horse  

### Регрессия

Если нужно предсказать число, выходной слой часто содержит один нейрон без активации.

Пример:

- цена квартиры  
- температура  
- спрос

## Пример

Пусть сеть решает задачу бинарной классификации.

Выходной слой выдаёт:

0.87

Это можно интерпретировать как вероятность положительного класса 87%.

## Итог

Output layer — это последний слой сети, который выдаёт итоговое предсказание в форме, соответствующей задаче.
