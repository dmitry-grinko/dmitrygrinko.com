# 13. Что такое learning rate?

Learning rate — это параметр алгоритма оптимизации, который определяет **размер шага при обновлении параметров модели**.

Он контролирует, насколько сильно параметры изменяются на каждой итерации.

## Формула обновления

θ = θ − η ∇L

где:

- θ — параметры модели  
- η — learning rate  
- ∇L — градиент функции потерь

## Интуитивная идея

Learning rate отвечает за скорость обучения.

- маленький learning rate → маленькие шаги  
- большой learning rate → большие шаги

## Почему это важно

Если шаги слишком большие, алгоритм может пропустить минимум.

Если шаги слишком маленькие, обучение будет идти очень медленно.

## Итог

Learning rate определяет размер шага при обновлении параметров модели.
