# 36. Что происходит если модель не сходится?

Иногда обучение может не приводить к устойчивому решению.

Это означает, что модель **не сходится**.

## Возможные причины

Слишком большой learning rate

Алгоритм постоянно перескакивает минимум функции.

Плохие данные

Сильный шум или плохо подготовленные признаки.

Проблемы с масштабом признаков

Если признаки имеют разные масштабы, обучение может быть нестабильным.

## Симптомы

- функция потерь не уменьшается  
- параметры модели постоянно колеблются  
- обучение становится нестабильным

## Итог

Если модель не сходится, необходимо проверить learning rate, данные и preprocessing.
