# 35. Как learning rate влияет на обучение logistic regression?

Learning rate определяет **размер шага при обновлении параметров модели**.

Он используется в gradient descent.

## Маленький learning rate

Если learning rate маленький:

- обучение происходит медленно  
- требуется больше итераций

## Большой learning rate

Если learning rate слишком большой:

- параметры могут изменяться слишком резко  
- обучение может стать нестабильным

## Баланс

Хороший learning rate позволяет:

- быстро уменьшать ошибку  
- сохранять стабильность обучения

## Итог

Learning rate контролирует скорость обучения модели.
