# 22. Почему высокий bias связан с недообучением?

Высокий bias возникает, когда модель делает **слишком сильные упрощения** о структуре данных.

Из-за этого она не может уловить реальные зависимости между признаками и целевой переменной.

## Что происходит

Модель использует слишком простую форму зависимости.

Например:

- линейная модель для сложной нелинейной задачи  
- слишком маленькая нейронная сеть

## Последствие

Модель делает систематические ошибки.

Она плохо работает:

- на training dataset  
- на новых данных

## Итог

Высокий bias связан с недообучением, потому что модель слишком простая, чтобы описать данные.
