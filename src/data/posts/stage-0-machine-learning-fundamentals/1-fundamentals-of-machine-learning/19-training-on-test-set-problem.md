# 19. Что произойдёт, если обучать модель на test dataset?

Если модель обучать на test dataset, то test set перестаёт выполнять свою главную задачу — **честно проверять качество модели**.

Test set должен содержать данные, которые модель **никогда не видела** во время разработки.

Если эти данные используются для обучения или настройки модели, возникает проблема **data leakage** — утечка информации из тестовых данных в модель.

## Что происходит в этом случае

Модель начинает подстраиваться под test dataset.

В результате:

- метрики на test set выглядят очень хорошими
- кажется, что модель работает отлично
- на новых реальных данных качество оказывается хуже

Это происходит потому, что модель частично **запоминает тестовые данные**, а не учится обобщать закономерности.

## Почему это проблема

Главная цель test set — проверить, как модель работает на **данных, которые она не видела раньше**.

Если модель обучается на test set, эта проверка становится бессмысленной.

## Итог

Если использовать test dataset для обучения или настройки модели, результат оценки становится ненадёжным.

Поэтому test set используют только один раз — для финальной проверки качества модели.
