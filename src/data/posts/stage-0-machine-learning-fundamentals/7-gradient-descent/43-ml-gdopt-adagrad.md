# 43. Что такое AdaGrad?

AdaGrad — это алгоритм оптимизации, который **адаптирует learning rate для каждого параметра модели**.

Идея заключается в том, что разные параметры могут требовать разных скоростей обучения.

## Основная идея

AdaGrad увеличивает learning rate для редких признаков и уменьшает его для часто обновляемых параметров.

## Почему это полезно

Это особенно важно в задачах с:

- разреженными признаками  
- большими словарями  
- текстовыми данными

## Ограничение

Со временем learning rate может становиться слишком маленьким, что замедляет обучение.

## Итог

AdaGrad автоматически изменяет learning rate для разных параметров модели.
