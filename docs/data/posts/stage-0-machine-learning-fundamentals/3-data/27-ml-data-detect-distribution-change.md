# 27. Как обнаружить изменение distribution данных?

Изменение распределения данных можно обнаружить с помощью анализа данных и мониторинга моделей.

Это важная часть поддержки ML-систем в production.

## Сравнение статистик

Можно сравнивать статистики training dataset и новых данных.

Например:

- среднее значение признаков  
- стандартное отклонение  
- диапазон значений

Если эти показатели сильно изменяются, это может быть признаком drift.

## Анализ распределений

Иногда строят графики распределения признаков:

- гистограммы  
- плотности распределения

Это позволяет увидеть изменения в данных.

## Мониторинг метрик модели

Если качество модели начинает снижаться, это может быть сигналом изменения данных.

Например:

- падение accuracy  
- рост ошибок  
- нестабильные predictions

## Итог

Изменение distribution можно обнаружить через статистический анализ данных и мониторинг метрик модели.
