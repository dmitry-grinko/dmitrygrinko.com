# 11. Что такое accuracy?

Accuracy — это метрика классификации, которая показывает **долю правильных предсказаний модели**.

Она измеряет, какой процент объектов был классифицирован правильно.

## Интуитивная идея

Если модель делает предсказание для каждого объекта, accuracy показывает, как часто это предсказание совпадает с правильным ответом.

## Формула

accuracy = (количество правильных предсказаний) / (общее количество объектов)

## Пример

Если модель правильно классифицировала 90 объектов из 100, то:

accuracy = 0.90 или 90%

## Итог

Accuracy измеряет долю правильных предсказаний модели.
