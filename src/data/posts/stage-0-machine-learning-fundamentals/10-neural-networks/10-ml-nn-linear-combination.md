# 10. Что такое линейная комбинация входов?

Линейная комбинация — это сумма входных значений, умноженных на соответствующие веса.

## Формула

z = w₁x₁ + w₂x₂ + ... + wₙxₙ + b

## Что это означает

Каждый вход:

- умножается на вес  
- затем все значения суммируются

## Интерпретация

Линейная комбинация определяет, какой сигнал получит нейрон до применения функции активации.

## Итог

Линейная комбинация — это взвешенная сумма входов нейрона.
