# 1. Что такое оптимизация в machine learning?

Оптимизация в machine learning — это процесс поиска таких параметров модели, при которых **ошибка предсказаний минимальна**.

Практически все модели машинного обучения имеют параметры.  
Например:

- коэффициенты линейной регрессии  
- веса нейронной сети  
- параметры логистической модели  

Задача обучения модели состоит в том, чтобы найти **лучшие значения этих параметров**.

## Интуитивная аналогия

Представим функцию потерь как поверхность.

Каждая точка на этой поверхности соответствует набору параметров модели.

Высота поверхности — это значение ошибки.

Оптимизация означает поиск **самой низкой точки на этой поверхности**.

## Что делает алгоритм обучения

Алгоритм оптимизации:

1. выбирает начальные параметры  
2. вычисляет ошибку модели  
3. изменяет параметры  
4. повторяет процесс

Со временем параметры модели начинают давать всё более точные предсказания.

## Итог

Оптимизация — это процесс поиска параметров модели, которые минимизируют ошибку предсказаний.
