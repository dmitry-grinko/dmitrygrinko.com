# 6. Что такое бинарная классификация?

Бинарная классификация — это задача классификации, в которой существует **только два возможных класса**.

Модель должна определить, к какому из двух классов относится объект.

## Примеры

Бинарная классификация встречается во многих задачах:

- spam / not spam  
- fraud / normal  
- disease / healthy  
- click / no click

## Как работает модель

Модель оценивает вероятность одного из классов.

Например:

P(class = 1)

Если вероятность выше порога, объект относится к этому классу.

## Почему logistic regression подходит

Logistic regression изначально разработана именно для бинарной классификации.

Она естественно предсказывает вероятность одного класса.

## Итог

Бинарная классификация — это задача, в которой объект может принадлежать только одному из двух классов.
