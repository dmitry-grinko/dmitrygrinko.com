# 14. Как learning rate влияет на процесс обучения?

Learning rate напрямую влияет на скорость и стабильность обучения модели.

## Маленький learning rate

Если learning rate слишком маленький:

- обучение становится очень медленным  
- требуется много итераций  
- модель долго приближается к минимуму

## Большой learning rate

Если learning rate слишком большой:

- алгоритм может перескакивать минимум  
- обучение становится нестабильным  
- функция потерь может колебаться

## Баланс

Хороший learning rate позволяет:

- быстро уменьшать функцию потерь  
- сохранять стабильность обучения

## Итог

Learning rate управляет тем, насколько быстро и стабильно обучается модель.
