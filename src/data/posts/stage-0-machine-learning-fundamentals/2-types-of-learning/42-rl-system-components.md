# 42. Какие компоненты входят в reinforcement learning систему?

Система reinforcement learning состоит из нескольких ключевых компонентов.

Эти компоненты описывают, как модель взаимодействует со средой и как происходит обучение.

## Agent

Agent — это модель, которая принимает решения.

Агент наблюдает состояние среды и выбирает действие.

## Environment

Environment — это среда, с которой взаимодействует агент.

Среда реагирует на действия агента и изменяет своё состояние.

## State

State описывает текущее состояние среды.

Агент принимает решения на основе этой информации.

## Action

Action — это действие, которое агент может выполнить.

Набор возможных действий зависит от конкретной задачи.

## Reward

Reward — это сигнал, который агент получает после выполнения действия.

Награда показывает, насколько хорошим было принятое решение.

## Итог

Основные компоненты reinforcement learning системы:

- agent  
- environment  
- state  
- action  
- reward
