# 6. Что такое target или label?

Target (или label) — это переменная, которую модель должна предсказать.

Если features описывают объект, то target — это **правильный результат**, связанный с этим объектом.

## Пример

Dataset:

| area | rooms | price |
|-----|------|------|
| 45 | 1 | 120000 |

Здесь:

- area и rooms — признаки  
- price — target

Модель получает признаки и пытается предсказать target.

## В разных задачах target может быть разным

Например:

Regression:

- цена дома  
- температура  
- количество продаж

Classification:

- spam / not spam  
- тип объекта на изображении  
- категория текста

## Роль target

Target используется для обучения модели.

Во время обучения модель:

1. делает prediction  
2. сравнивает prediction с target  
3. корректирует свои параметры

## Итог

Target (label) — это правильный результат, который модель должна научиться предсказывать.
