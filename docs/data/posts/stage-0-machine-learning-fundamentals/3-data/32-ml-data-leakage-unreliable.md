# 32. Почему data leakage делает результаты модели недостоверными?

Data leakage приводит к тому, что модель показывает **нереалистично хорошие результаты**.

Метрики могут выглядеть очень высокими, но это не отражает реальную способность модели работать на новых данных.

## Почему это происходит

Если модель получает информацию, связанную с target, задача становится слишком простой.

Модель фактически "подсматривает" правильный ответ.

## Что происходит на практике

Во время тестирования:

- accuracy может быть очень высокой  
- ошибки могут быть минимальными

Но когда модель используется в production, эта информация уже недоступна.

В результате качество резко падает.

## Опасность leakage

Data leakage особенно опасен, потому что его трудно заметить.

Модель может выглядеть очень успешной, пока её не начинают использовать в реальной системе.

## Итог

Data leakage делает результаты недостоверными, потому что модель обучается на информации, которая не будет доступна при реальном использовании.
