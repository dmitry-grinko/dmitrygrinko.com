# 14. Что такое expected loss?

Expected loss — это **ожидаемое значение функции потерь по распределению данных**.

Это теоретическое понятие, которое описывает среднюю ошибку модели на всех возможных данных.

## Идея

В реальности dataset — это лишь выборка из распределения данных.

Expected loss — это ошибка модели на **всём распределении**, а не только на конкретном dataset.

## Приближение

На практике expected loss неизвестен.

Поэтому его оценивают через:

- training loss  
- validation loss  
- test loss

## Итог

Expected loss — это теоретическая средняя ошибка модели на всём распределении данных.
