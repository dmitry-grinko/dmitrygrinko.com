# 2. Почему модель может переобучиться?

Модель может переобучиться, когда она получает **слишком много свободы для подгонки под training data**.

Это происходит по нескольким причинам.

## Слишком сложная модель

Если модель имеет много параметров, она может описывать очень сложные зависимости.

Иногда она начинает моделировать не закономерности, а случайный шум.

## Маленький dataset

Когда данных мало, модель видит ограниченное количество примеров.

В этом случае она может начать запоминать конкретные объекты.

## Шум в данных

Если dataset содержит ошибки или случайные колебания, модель может попытаться объяснить и их.

## Слишком долгое обучение

Если обучение продолжается слишком долго, модель может начать подгоняться под training data всё точнее.

## Итог

Переобучение возникает из-за сочетания сложной модели, недостатка данных и шума в dataset.
