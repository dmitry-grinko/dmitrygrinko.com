# 36. Какие способы обработки пропущенных значений существуют?

Существует несколько способов обработки missing data.

Выбор метода зависит от задачи и структуры dataset.

## Удаление строк

Самый простой подход — удалить строки, содержащие пропущенные значения.

Это работает, если:

- таких строк немного  
- dataset достаточно большой

Но если пропусков много, это может привести к потере данных.

## Заполнение значений

Другой подход — заменить пропущенные значения.

Например:

- средним значением признака  
- медианой  
- наиболее частым значением

Этот метод называется **imputation**.

## Использование специальных моделей

Некоторые алгоритмы могут работать с missing values напрямую.

Например, некоторые реализации градиентного бустинга умеют учитывать пропущенные значения.

## Итог

Основные способы обработки missing data:

- удаление строк  
- заполнение значений  
- использование алгоритмов, поддерживающих missing values
