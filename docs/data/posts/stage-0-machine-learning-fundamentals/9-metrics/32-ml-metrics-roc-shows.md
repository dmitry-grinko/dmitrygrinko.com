# 32. Что показывает ROC curve?

ROC curve показывает **способность модели различать положительные и отрицательные классы**.

Она демонстрирует, как меняется баланс между:

- обнаружением положительных объектов
- количеством ложных срабатываний

## Что происходит при изменении threshold

Когда threshold уменьшается:

- модель находит больше положительных объектов
- увеличивается число ложных срабатываний

Когда threshold увеличивается:

- модель делает меньше ложных срабатываний
- но может пропускать положительные объекты

ROC curve показывает этот компромисс.

## Интерпретация

Чем ближе кривая к верхнему левому углу графика, тем лучше модель.

## Итог

ROC curve показывает баланс между обнаружением положительных объектов и ложными срабатываниями.
