# 35. Что такое pretext task?

Pretext task — это вспомогательная задача, которая используется в self-supervised learning для обучения модели.

Эта задача создаётся автоматически на основе исходных данных.

Модель обучается решать эту задачу, чтобы научиться извлекать полезные представления из данных.

## Основная идея

Pretext task не является конечной целью системы.

Она используется только для того, чтобы модель могла обучиться на большом количестве неразмеченных данных.

После этого модель может применяться для других задач.

## Пример в NLP

В языковых моделях часть слов в предложении может быть скрыта.

Например:

"The cat sits on the ___."

Задача модели — предсказать пропущенное слово.

Это и есть pretext task.

## Пример в computer vision

В задачах компьютерного зрения модель может получать изображение с закрытой частью.

Задача модели — восстановить скрытую область.

Таким образом модель учится понимать структуру изображений.

## Итог

Pretext task — это автоматически созданная задача обучения, которая помогает модели извлекать полезные представления из неразмеченных данных.
