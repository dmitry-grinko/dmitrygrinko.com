# 24. Как gradient descent используется в линейной регрессии?

В линейной регрессии gradient descent используется для поиска таких коэффициентов модели, которые **минимизируют функцию потерь**.

Чаще всего минимизируется MSE.

## Общая идея

Алгоритм начинает с случайных значений параметров.

После этого он постепенно корректирует коэффициенты, чтобы уменьшить ошибку.

## Процесс

На каждой итерации происходит следующее:

1. модель делает предсказания  
2. вычисляется ошибка  
3. вычисляется градиент функции потерь  
4. коэффициенты модели обновляются

Этот процесс повторяется много раз.

## Почему это работает

Градиент показывает направление, в котором функция потерь растёт быстрее всего.

Поэтому алгоритм двигается в противоположном направлении — туда, где ошибка уменьшается.

## Итог

Gradient descent используется в линейной регрессии для поиска оптимальных коэффициентов модели через минимизацию функции потерь.
