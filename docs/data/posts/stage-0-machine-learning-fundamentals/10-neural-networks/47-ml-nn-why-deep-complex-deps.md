# 47. Почему глубокие сети могут моделировать сложные зависимости?

Глубокие сети применяют **много последовательных нелинейных преобразований**.

Каждый слой добавляет новую степень сложности.

## Интуитивная идея

Если объединить несколько простых функций, можно получить очень сложную функцию.

## Пример

Первый слой может выделять простые признаки:

- линии
- границы

Следующий слой комбинирует их в:

- формы

Следующий слой может распознавать:

- части объектов

## Итог

Глубина сети позволяет постепенно строить сложные представления данных.
