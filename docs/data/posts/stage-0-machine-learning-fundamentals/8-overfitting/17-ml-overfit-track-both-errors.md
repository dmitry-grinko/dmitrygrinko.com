# 17. Почему важно отслеживать обе ошибки?

Отслеживание training error и validation error позволяет понять **как именно обучается модель**.

Эти две величины дают разную информацию.

## Training error показывает

Насколько хорошо модель подогнана под training dataset.

## Validation error показывает

Насколько хорошо модель работает на новых данных.

## Что можно обнаружить

Если:

training error низкая  
validation error высокая

это признак переобучения.

Если:

обе ошибки высокие

это признак недообучения.

## Итог

Сравнение training и validation ошибок помогает понять состояние обучения модели.
