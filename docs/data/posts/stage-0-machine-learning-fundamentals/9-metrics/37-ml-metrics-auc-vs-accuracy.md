# 37. Когда AUC полезнее accuracy?

AUC полезнее accuracy в задачах, где:

- классы сильно несбалансированы
- важна способность модели ранжировать объекты

## Пример

В задачах обнаружения мошенничества или медицинской диагностики положительные случаи встречаются редко.

Accuracy может быть высокой даже у плохой модели.

AUC лучше показывает способность модели отличать классы.

## Дополнительное преимущество

AUC не зависит от конкретного threshold.

## Итог

AUC полезнее accuracy, когда dataset несбалансирован или threshold ещё не определён.
