# 42. Что такое Ridge regression (L2 regularization)?

Ridge regression — это вариант линейной регрессии, который использует **L2 регуляризацию**.

В этом методе к функции потерь добавляется штраф, зависящий от квадратов коэффициентов.

## Формула

Функция потерь выглядит так:

loss = MSE + λ Σ w²

где:

- w — коэффициенты модели  
- λ — параметр регуляризации

## Что делает регуляризация

L2 регуляризация уменьшает значения коэффициентов, но редко делает их равными нулю.

Это помогает:

- уменьшить переобучение  
- сделать модель более устойчивой

## Когда используется

Ridge regression часто применяется, когда:

- признаков много  
- между признаками есть корреляция

## Итог

Ridge regression — это линейная регрессия с L2 регуляризацией, которая уменьшает значения коэффициентов модели.
