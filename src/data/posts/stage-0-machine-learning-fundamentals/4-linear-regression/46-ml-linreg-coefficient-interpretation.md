# 46. Как интерпретировать коэффициенты линейной модели?

Коэффициенты линейной модели показывают, **как изменение признака влияет на предсказание модели**.

Каждый коэффициент связан с одним признаком.

## Интерпретация

Если коэффициент положительный:

увеличение признака увеличивает prediction.

Если коэффициент отрицательный:

увеличение признака уменьшает prediction.

## Пример

Модель:

```
price = 2000 × area + 5000 × rooms + 10000
```

Это означает:

- каждый дополнительный квадратный метр увеличивает цену на 2000  
- каждая дополнительная комната увеличивает цену на 5000

## Важность масштабирования

Если признаки имеют разные масштабы, сравнивать коэффициенты напрямую может быть сложно.

Поэтому иногда признаки нормализуют перед обучением.

## Итог

Коэффициенты линейной модели показывают направление и силу влияния признаков на предсказание.
