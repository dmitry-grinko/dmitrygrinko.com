# 9. Что такое evaluation dataset?

Evaluation dataset — это набор данных, который используется **для оценки качества модели**.

Он не используется для обучения модели.

## Основная цель

Проверить, насколько хорошо модель работает на данных, которые она не видела раньше.

## Типы evaluation dataset

Обычно используют:

- validation dataset  
- test dataset

## Почему это важно

Если оценивать модель на training data, результат будет слишком оптимистичным.

## Итог

Evaluation dataset — это данные, предназначенные для проверки качества модели.
