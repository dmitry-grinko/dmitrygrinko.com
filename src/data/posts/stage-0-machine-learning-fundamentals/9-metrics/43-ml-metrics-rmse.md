# 43. Что такое Root Mean Squared Error (RMSE)?

Root Mean Squared Error (RMSE) — это квадратный корень из MSE.

## Формула

RMSE = √MSE

или

RMSE = √((1/n) Σ (y − ŷ)²)

## Почему используется RMSE

Главное преимущество RMSE — единицы измерения совпадают с единицами исходной переменной.

Например:

если модель предсказывает цену в долларах, RMSE тоже будет измеряться в долларах.

## Интерпретация

RMSE показывает средний масштаб ошибки модели.

## Итог

RMSE — это квадратный корень из MSE, который возвращает ошибку в исходных единицах измерения.
