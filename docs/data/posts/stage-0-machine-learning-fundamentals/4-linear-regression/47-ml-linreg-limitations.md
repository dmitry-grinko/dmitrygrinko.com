# 47. Какие ограничения есть у линейной регрессии?

Несмотря на простоту и популярность, линейная регрессия имеет ряд ограничений.

## Линейность

Модель предполагает линейную зависимость между признаками и target.

Если зависимость сильно нелинейная, модель может работать плохо.

## Чувствительность к выбросам

Линейная регрессия чувствительна к выбросам.

Несколько экстремальных значений могут сильно изменить параметры модели.

## Multicollinearity

Сильная корреляция между признаками может делать коэффициенты нестабильными.

## Ограниченная гибкость

Линейная модель может не улавливать сложные зависимости в данных.

## Итог

Основные ограничения линейной регрессии связаны с предположением линейности и чувствительностью к особенностям данных.
