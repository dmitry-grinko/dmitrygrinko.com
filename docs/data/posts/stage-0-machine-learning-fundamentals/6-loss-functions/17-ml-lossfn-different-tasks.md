# 17. Почему разные задачи требуют разные функции потерь?

Разные задачи машинного обучения имеют **разные типы предсказаний**.

Поэтому требуется использовать разные функции потерь.

## Примеры

Regression задачи:

- MSE  
- MAE

Классификация:

- cross entropy

Detection и ranking задачи:

- специальные loss функции

## Причина

Функция потерь должна соответствовать типу предсказания:

- числа  
- вероятности  
- классы

## Итог

Разные задачи требуют разные функции потерь, потому что тип ошибок в них различается.
