# 13. Как измерить способность модели обобщать?

Способность модели к обобщению измеряют с помощью **данных, которые не использовались при обучении**.

Обычно для этого используют validation dataset или test dataset.

## Основная идея

Модель обучается на training dataset.

После этого её качество оценивается на данных, которые она не видела.

Если модель показывает хорошие результаты и на новых данных, значит она обладает хорошей generalization.

## Метрики

Обычно используют те же метрики, что и при обучении:

- accuracy  
- mean squared error  
- cross entropy  
- F1-score

## Интерпретация

Если ошибка на test dataset близка к training error, модель хорошо обобщает.

## Итог

Generalization измеряется качеством модели на новых данных.
