# 11. Что такое train dataset?

Train dataset — это часть данных, на которой **модель обучается**.

Именно на этих данных модель пытается найти зависимость между признаками (features) и целевой переменной (target).

Во время обучения модель многократно проходит по train dataset и постепенно настраивает свои параметры.

## Как используется train dataset

Процесс обучения обычно выглядит так:

1. модель получает признаки объекта  
2. делает prediction  
3. сравнивает prediction с правильным ответом  
4. вычисляет ошибку  
5. обновляет параметры модели

Этот процесс повторяется для всех объектов train dataset.

## Почему train dataset важен

Train dataset фактически формирует знания модели.

Если training data:

- маленький  
- содержит ошибки  
- плохо отражает реальные данные

то модель будет обучаться неправильным закономерностям.

## Итог

Train dataset — это набор данных, на котором происходит обучение модели и настройка её параметров.
