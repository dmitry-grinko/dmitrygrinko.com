# 1. Что такое loss function в machine learning?

Loss function — это функция, которая измеряет **насколько сильно предсказание модели отличается от правильного ответа**.

Она превращает ошибку модели в числовое значение.

Чем меньше значение loss, тем лучше модель делает предсказания.

## Пример

Если модель должна предсказать цену квартиры:

Реальное значение:

price = 200000

Предсказание модели:

prediction = 180000

Loss функция измеряет разницу между этими значениями.

## Зачем это нужно

Модель не может напрямую понимать, что предсказание «плохое» или «хорошее».

Loss функция переводит эту оценку в число, которое можно минимизировать.

## Итог

Loss function — это математическая функция, которая измеряет ошибку предсказаний модели.
