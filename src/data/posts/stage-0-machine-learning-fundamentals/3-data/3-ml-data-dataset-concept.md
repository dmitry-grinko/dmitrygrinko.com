# 3. Что такое dataset?

Dataset — это структурированный набор данных, который используется для обучения и оценки моделей.

Обычно dataset представляет собой таблицу, где:

- строки — отдельные объекты  
- столбцы — признаки или переменные

Пример dataset:

| age | income | purchases |
|----|-------|----------|
| 25 | 40000 | 3 |
| 32 | 60000 | 7 |
| 45 | 80000 | 12 |

Каждая строка описывает один объект.

Dataset может содержать:

- входные признаки (features)
- целевую переменную (target)

## Где используются datasets

Dataset используется на разных этапах:

- обучение модели  
- настройка гиперпараметров  
- оценка качества модели

Обычно dataset делят на несколько частей:

- training dataset  
- validation dataset  
- test dataset

## Размер dataset

Datasets могут сильно различаться по размеру:

- сотни строк  
- миллионы строк  
- миллиарды примеров

Размер dataset влияет на то, какие модели можно эффективно обучать.

## Итог

Dataset — это структурированный набор данных, который используется для обучения и оценки моделей машинного обучения.
