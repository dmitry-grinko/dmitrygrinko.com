# 27. Чем semi-supervised learning отличается от unsupervised learning?

Хотя оба подхода используют неразмеченные данные, между ними есть важное различие.

Главное отличие заключается в том, **используются ли размеченные данные вообще**.

## Unsupervised learning

В unsupervised learning dataset полностью состоит из неразмеченных данных.

Модель получает только признаки объектов и пытается самостоятельно найти структуру в данных.

Например:

- разделить данные на группы  
- обнаружить закономерности  
- уменьшить размерность данных

При этом алгоритм не знает, каким должен быть правильный результат.

## Semi-supervised learning

В semi-supervised learning используется **смесь данных**:

- небольшое количество размеченных примеров  
- большое количество неразмеченных данных

Размеченные данные дают модели сигнал обучения, а неразмеченные помогают лучше понять структуру dataset.

## Почему это различие важно

Unsupervised learning обычно используется для анализа структуры данных.

Semi-supervised learning применяется тогда, когда задача всё-таки требует предсказания, но размеченных данных недостаточно.

## Итог

Основное различие:

- unsupervised learning использует только неразмеченные данные  
- semi-supervised learning использует и размеченные, и неразмеченные данные
