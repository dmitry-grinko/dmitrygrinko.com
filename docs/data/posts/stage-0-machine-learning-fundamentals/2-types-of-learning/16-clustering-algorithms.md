# 16. Какие алгоритмы используются для clustering?

Clustering — одна из основных задач unsupervised learning, и для её решения существует несколько популярных алгоритмов. Все они пытаются разделить данные на группы на основе сходства между объектами.

Разные алгоритмы используют разные подходы к определению того, какие объекты похожи.

## K-Means

K-Means — один из самых известных и простых алгоритмов кластеризации.

Алгоритм работает следующим образом:

1. выбирается количество кластеров K  
2. алгоритм случайно размещает центры кластеров  
3. каждый объект относится к ближайшему центру  
4. центры пересчитываются  
5. процесс повторяется до стабилизации

K-Means хорошо работает на относительно простых datasets, где кластеры имеют примерно сферическую форму.

## Hierarchical clustering

Hierarchical clustering строит **иерархию кластеров**.

Алгоритм постепенно объединяет похожие объекты в группы, формируя дерево кластеров (dendrogram).

Такой подход полезен, когда нужно увидеть структуру данных на разных уровнях.

## DBSCAN

DBSCAN — алгоритм, который ищет **плотные области данных**.

Он объединяет объекты, которые находятся близко друг к другу, и может обнаруживать:

- кластеры сложной формы  
- выбросы (outliers)

Это делает DBSCAN полезным для datasets, где структура данных сложнее, чем у K-Means.

## Итог

Некоторые из самых распространённых алгоритмов clustering:

- K-Means  
- Hierarchical clustering  
- DBSCAN  

Выбор алгоритма зависит от структуры данных и задачи анализа.
