# 7. Какие задачи можно решать с помощью logistic regression?

Logistic regression используется в задачах бинарной классификации.

Она особенно хорошо работает с табличными данными.

## Примеры задач

Spam detection

Определение, является ли письмо спамом.

Fraud detection

Определение мошеннических транзакций.

Credit scoring

Оценка вероятности невозврата кредита.

Medical diagnosis

Определение наличия заболевания.

Customer churn

Предсказание ухода клиента.

## Когда модель работает хорошо

Logistic regression эффективна, когда:

- данные табличные
- зависимость между признаками и классом относительно простая
- требуется интерпретируемость модели

## Итог

Logistic regression применяется во многих задачах бинарной классификации в бизнесе, финансах и медицине.
