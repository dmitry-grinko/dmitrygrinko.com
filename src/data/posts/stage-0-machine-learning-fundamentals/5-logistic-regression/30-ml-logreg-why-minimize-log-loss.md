# 30. Почему минимизация log loss обучает модель?

Logistic regression обучается через минимизацию функции потерь.

Когда log loss уменьшается, предсказанные вероятности становятся ближе к реальным классам.

## Процесс обучения

Во время обучения происходит следующее:

1. модель делает предсказание
2. вычисляется log loss
3. вычисляется градиент функции потерь
4. параметры модели обновляются

## Что происходит с вероятностями

Минимизация log loss заставляет модель:

- увеличивать вероятность правильного класса
- уменьшать вероятность неправильного класса

## Результат

Со временем модель начинает делать всё более точные предсказания.

## Итог

Минимизация log loss заставляет модель улучшать вероятностные оценки классов.
