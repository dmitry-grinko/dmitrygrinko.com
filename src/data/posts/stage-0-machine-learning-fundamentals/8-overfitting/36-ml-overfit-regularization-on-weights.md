# 36. Как регуляризация влияет на веса модели?

Регуляризация влияет на веса модели через дополнительный штраф в функции потерь.

Алгоритм обучения минимизирует:

loss + regularization penalty

## Что происходит во время обучения

Если веса становятся слишком большими, штраф увеличивается.

Чтобы уменьшить общий loss, алгоритм начинает уменьшать параметры модели.

## Результат

Веса модели становятся:

- меньше по величине  
- более стабильными  
- менее чувствительными к шуму

## Практический эффект

Модель начинает использовать более простые зависимости.

## Итог

Регуляризация ограничивает величину весов модели.
