# 30. Почему иногда данные важнее архитектуры модели?

В machine learning качество данных часто влияет на результат сильнее, чем архитектура модели.

## Причина

Даже очень сложная модель не сможет обучиться без качественных данных.

Если dataset:

- маленький  
- шумный  
- плохо размеченный

модель будет обучаться плохо.

## Что дают хорошие данные

Качественный dataset позволяет модели:

- видеть реальные закономерности  
- лучше обобщать  
- уменьшать переобучение

## Практическое наблюдение

Во многих проектах улучшение dataset даёт больший эффект, чем изменение модели.

## Итог

Качество и количество данных часто оказываются важнее архитектуры модели.
