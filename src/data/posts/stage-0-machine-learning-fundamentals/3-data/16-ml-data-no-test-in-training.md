# 16. Почему test dataset нельзя использовать при обучении?

Test dataset используется для **объективной оценки модели**.

Если модель обучается на этих данных, оценка качества перестаёт быть честной.

## Основная проблема

Когда модель видит данные во время обучения, она может частично запомнить их.

Если те же данные используются для тестирования, модель будет показывать слишком хорошие результаты.

Это создаёт иллюзию высокой точности.

## Пример

Представим, что модель обучалась на dataset и затем тестируется на тех же данных.

Модель может выучить конкретные примеры.

Но когда появятся новые данные, качество может резко упасть.

## Аналогия

Это похоже на экзамен, где студент заранее знает ответы на вопросы.

На экзамене он покажет высокий результат, но это не означает, что он действительно понимает тему.

## Итог

Test dataset нельзя использовать при обучении, потому что он должен оставаться независимым источником для честной оценки модели.
