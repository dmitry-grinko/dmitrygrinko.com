# 33. Чем cross entropy отличается от MSE?

Cross entropy и MSE предназначены для разных типов задач.

## MSE

Используется в задачах регрессии.

Она измеряет квадрат разницы между числовыми значениями.

## Cross entropy

Используется в классификации.

Она измеряет ошибку предсказанных вероятностей.

## Основное различие

MSE сравнивает числа.

Cross entropy сравнивает вероятности.

## Практическое следствие

Cross entropy обычно обучает классификационные модели быстрее и стабильнее.

## Итог

Cross entropy лучше подходит для задач классификации, чем MSE.
