# 24. Какие признаки указывают на переобучение?

Переобучение можно обнаружить по нескольким характерным признакам.

## Разрыв между ошибками

Training error становится очень маленькой,  
а validation или test error остаётся высокой.

## Ухудшение validation ошибки

Во время обучения training error продолжает уменьшаться,  
но validation error начинает расти.

## Слишком сложная модель

Модель имеет очень большое количество параметров относительно размера dataset.

## Нестабильные предсказания

Модель начинает сильно реагировать на небольшие изменения данных.

## Итог

Основной признак переобучения — хорошая работа на training data и плохая работа на новых данных.
