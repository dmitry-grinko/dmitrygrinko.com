# 28. Почему между precision и recall существует компромисс?

Precision и recall часто находятся в компромиссе, потому что изменение порога классификации влияет на обе метрики.

## Если увеличить порог

Модель будет предсказывать положительный класс реже.

Это приводит к:

- росту precision  
- снижению recall

## Если уменьшить порог

Модель будет чаще предсказывать положительный класс.

Это приводит к:

- росту recall  
- снижению precision

## Практическая задача

Нужно выбрать баланс между этими метриками в зависимости от задачи.

## Итог

Компромисс возникает из-за влияния порога классификации на ошибки разных типов.
