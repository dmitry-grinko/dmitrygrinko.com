# 16. Что такое validation set?

Validation set — это часть dataset, которая используется для настройки модели во время разработки.

Модель **не обучается** на этих данных. Они используются для проверки того, как хорошо модель работает на данных, которые она не видела во время training.

Validation set обычно помогает:

- выбирать гиперпараметры модели
- сравнивать разные модели
- контролировать переобучение (overfitting)

Типичная схема работы выглядит так:

training set → обучение модели
validation set → настройка и выбор модели

## Итог

Validation set — это данные, которые используются для оценки и настройки модели во время разработки, но не участвуют напрямую в её обучении.
