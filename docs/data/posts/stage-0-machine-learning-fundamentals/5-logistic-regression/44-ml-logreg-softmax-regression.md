# 44. Что такое softmax regression?

Softmax regression — это расширение logistic regression для задач **multiclass classification**.

Она позволяет модели сразу предсказывать вероятности для нескольких классов.

## Основная идея

Модель вычисляет линейную комбинацию признаков для каждого класса.

Затем применяется функция softmax, которая преобразует эти значения в вероятности.

## Свойства

Softmax гарантирует, что:

- все вероятности находятся между 0 и 1  
- сумма вероятностей равна 1

## Пример

Модель может выдать:

P(cat) = 0.6  
P(dog) = 0.3  
P(horse) = 0.1

Класс с максимальной вероятностью выбирается как итоговый.

## Итог

Softmax regression — это обобщение logistic regression для задач с несколькими классами.
