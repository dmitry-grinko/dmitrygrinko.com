# 46. Почему регуляризация помогает бороться с переобучением?

Переобучение возникает, когда модель слишком точно запоминает training data.

Регуляризация уменьшает сложность модели.

## Основная идея

Большие параметры позволяют модели строить сложные зависимости.

Регуляризация штрафует такие решения.

## Результат

Модель начинает искать более простые закономерности.

Это улучшает её способность работать на новых данных.

## Итог

Регуляризация уменьшает риск переобучения, ограничивая сложность модели.
