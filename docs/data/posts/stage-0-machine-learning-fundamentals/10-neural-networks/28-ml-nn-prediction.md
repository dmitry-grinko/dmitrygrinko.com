# 28. Что такое предсказание нейронной сети?

Предсказание нейронной сети — это итоговое значение, которое сеть выдаёт после обработки входных данных.

Это результат всего forward pass.

## Что именно считается предсказанием

Это зависит от задачи.

### Для бинарной классификации

Сеть может выдать вероятность:

0.91

Это означает 91% вероятности положительного класса.

### Для многоклассовой классификации

Сеть может выдать вероятности для всех классов:

- cat = 0.7  
- dog = 0.2  
- horse = 0.1  

### Для регрессии

Сеть может выдать число:

- цена = 185000

## Пример

Если сеть предсказывает, является ли письмо спамом, её предсказание может быть:

0.83

Если threshold = 0.5, письмо будет классифицировано как spam.

## Итог

Предсказание нейронной сети — это итоговый output модели для конкретного входного примера.
