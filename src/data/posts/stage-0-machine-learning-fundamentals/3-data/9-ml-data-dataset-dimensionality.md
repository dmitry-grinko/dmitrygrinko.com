# 9. Что такое размерность dataset?

Размерность dataset описывает структуру данных и обычно включает два параметра:

- количество наблюдений  
- количество признаков

Эти параметры определяют форму dataset.

## Количество наблюдений

Это число строк dataset.

Каждая строка — отдельный объект.

Например:

10 000 пользователей → 10 000 observations.

## Количество признаков

Это число характеристик каждого объекта.

Например:

- возраст  
- доход  
- количество покупок

Если dataset содержит 10 признаков, значит каждый объект описывается 10 переменными.

## Высокая размерность

Когда dataset содержит очень много признаков, говорят о **высокой размерности данных**.

Такие datasets могут создавать дополнительные сложности:

- более сложные модели  
- риск переобучения  
- увеличение вычислительных затрат

## Итог

Размерность dataset определяется количеством объектов и количеством признаков, описывающих эти объекты.
