# 15. Почему accuracy плохо работает при дисбалансе классов?

При дисбалансе классов один класс встречается намного чаще другого.

В этом случае accuracy начинает отражать **частоту самого распространённого класса**, а не качество модели.

## Что происходит

Модель может просто предсказывать самый частый класс.

Это даёт высокую accuracy, но модель не выполняет реальную задачу.

## Пример

Если 95% писем не являются спамом, модель может всегда говорить «не спам».

Accuracy будет 95%, но модель не обнаружит спам.

## Итог

При дисбалансе классов accuracy перестаёт отражать реальную эффективность модели.
