# Что такое preprocessing данных

Preprocessing данных — это этап подготовки данных перед обучением модели.

На этом этапе данные очищают и преобразуют так, чтобы модель могла корректно с ними работать.

Реальные datasets редко бывают готовыми к использованию. Они могут содержать ошибки, пропущенные значения или данные в неудобном формате. Поэтому перед обучением модели данные обычно предварительно обрабатывают.

## Примеры preprocessing

Типичные операции preprocessing включают:

- удаление или заполнение пропущенных значений
- удаление дубликатов
- исправление ошибок в данных
- преобразование типов данных
- нормализацию или масштабирование числовых признаков

Например, если в dataset есть пропущенные значения возраста, их могут заменить средним значением или удалить такие строки.

## Почему это важно

Модели машинного обучения ожидают, что данные имеют определённый формат и структуру.

Если данные содержат ошибки или пропуски, это может:

- ухудшить качество модели
- сделать обучение нестабильным
- привести к неправильным предсказаниям

Поэтому preprocessing является одним из обязательных этапов ML pipeline.

## Итог

Preprocessing — это процесс очистки и подготовки данных, который делает dataset пригодным для обучения модели.
