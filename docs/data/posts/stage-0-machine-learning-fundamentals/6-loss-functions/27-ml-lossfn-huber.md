# 27. Что такое Huber loss?

Huber loss — это функция потерь, которая сочетает свойства **MSE и MAE**.

Она ведёт себя по-разному для маленьких и больших ошибок.

## Основная идея

Если ошибка маленькая → используется квадратичная ошибка (как MSE).

Если ошибка большая → используется линейная ошибка (как MAE).

## Почему это полезно

Huber loss:

- чувствительна к небольшим ошибкам
- устойчива к выбросам

## Параметр δ

Huber loss использует параметр δ.

Он определяет границу между MSE и MAE режимами.

## Итог

Huber loss объединяет преимущества MSE и MAE.
