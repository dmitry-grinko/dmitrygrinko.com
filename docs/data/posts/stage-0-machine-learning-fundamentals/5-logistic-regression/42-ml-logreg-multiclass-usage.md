# 42. Как logistic regression используется для нескольких классов?

Изначально logistic regression была разработана для **бинарной классификации**.

Но её можно адаптировать для задач с несколькими классами.

## Основная идея

Для multiclass задач обычно используют один из двух подходов:

- one-vs-rest  
- softmax regression

## Как это работает

Модель обучается так, чтобы различать каждый класс по отдельности.

Например, если есть три класса:

A  
B  
C

можно построить несколько бинарных моделей.

## Результат

Каждая модель оценивает вероятность своего класса.

Затем выбирается класс с максимальной вероятностью.

## Итог

Logistic regression может использоваться для multiclass задач через специальные стратегии обучения.
