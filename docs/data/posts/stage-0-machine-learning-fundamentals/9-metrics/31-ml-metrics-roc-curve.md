# 31. Что такое ROC curve?

ROC curve (Receiver Operating Characteristic curve) — это график, который показывает **как меняется качество классификатора при разных значениях порога (threshold)**.

Многие модели классификации возвращают не просто класс, а вероятность принадлежности к классу.

Чтобы получить окончательное решение, используется порог.

ROC кривая показывает, как меняются ошибки модели при изменении этого порога.

## Что изображается на графике

По осям графика откладываются:

- True Positive Rate (TPR)
- False Positive Rate (FPR)

## Зачем используется ROC

ROC кривая позволяет понять, насколько хорошо модель разделяет классы при разных threshold.

## Итог

ROC curve — это график, показывающий поведение классификатора при разных значениях порога.
