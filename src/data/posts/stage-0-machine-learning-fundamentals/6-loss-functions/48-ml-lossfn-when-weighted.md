# 48. Когда используется weighted loss?

Weighted loss используется, когда **разные ошибки имеют разную важность**.

Это часто происходит в задачах с несбалансированными классами.

## Пример

Fraud detection

Мошеннические операции встречаются редко.

Если модель пропустит мошенничество, это серьёзная ошибка.

Поэтому таким примерам можно дать больший вес.

## Другие случаи

Weighted loss используется в:

- медицинской диагностике  
- detection задачах  
- imbalanced datasets

## Итог

Weighted loss используется, когда необходимо изменить важность ошибок разных примеров.
