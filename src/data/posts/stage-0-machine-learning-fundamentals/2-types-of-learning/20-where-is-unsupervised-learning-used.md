# 20. В каких задачах используют unsupervised learning?

Unsupervised learning применяется в ситуациях, когда dataset не содержит правильных ответов, но данные всё равно нужно анализировать.

Такие методы часто используются для исследования datasets и поиска скрытых закономерностей.

## Сегментация клиентов

Одна из самых распространённых задач — сегментация клиентов.

Например, компания может разделить клиентов на группы на основе:

- поведения  
- частоты покупок  
- дохода  

Это помогает лучше понимать аудиторию и строить маркетинговые стратегии.

## Анализ поведения пользователей

Unsupervised методы могут использоваться для анализа того, как пользователи взаимодействуют с продуктом.

Например:

- выявление типов пользователей  
- анализ пользовательских сценариев  
- обнаружение необычного поведения

## Анализ больших datasets

Когда dataset очень большой и сложный, unsupervised learning помогает понять его структуру.

Например:

- обнаружить группы похожих объектов  
- выявить скрытые закономерности  
- исследовать структуру данных

## Обнаружение аномалий

Unsupervised методы также используются для поиска необычных объектов.

Например:

- обнаружение мошенничества  
- выявление ошибок в данных  
- обнаружение аномальных транзакций

## Итог

Unsupervised learning используется для анализа структуры данных, сегментации объектов и обнаружения необычных наблюдений.
