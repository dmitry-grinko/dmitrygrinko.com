# 40. Зачем изменять learning rate во время обучения?

Изменение learning rate помогает улучшить процесс оптимизации.

## Начало обучения

Большой learning rate позволяет быстро уменьшать функцию потерь.

## Поздние этапы обучения

Меньший learning rate позволяет делать более точные обновления параметров.

Это помогает аккуратно приблизиться к минимуму функции.

## Практический эффект

Изменение learning rate:

- ускоряет обучение  
- улучшает convergence  
- повышает стабильность оптимизации

## Итог

Learning rate изменяют во время обучения, чтобы сочетать быстрое обучение и точную оптимизацию.
