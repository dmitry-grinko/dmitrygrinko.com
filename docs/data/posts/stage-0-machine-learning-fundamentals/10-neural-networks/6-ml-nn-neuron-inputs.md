# 6. Что такое входы (inputs) нейрона?

Inputs — это значения, которые поступают в нейрон и используются для вычисления его выхода.

Каждый вход представляет одну характеристику данных.

## Примеры

Если нейрон используется для обработки изображения, входами могут быть:

- значения пикселей  
- признаки изображения

Если сеть обрабатывает таблицу данных, входами могут быть:

- возраст  
- доход  
- количество покупок

## Связь со слоем

Входы нейрона обычно являются выходами предыдущего слоя.

## Итог

Inputs — это значения, которые нейрон получает для выполнения вычислений.
