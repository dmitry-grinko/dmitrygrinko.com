# 34. Какие задачи используют self-supervised learning?

Self-supervised learning активно используется в задачах, где доступно большое количество неразмеченных данных.

Такие методы особенно важны в современных deep learning системах.

## Обработка текста

В NLP self-supervised learning используется для обучения языковых моделей.

Модель может получать текст и пытаться предсказывать:

- пропущенные слова  
- следующее слово в предложении  
- структуру текста

Это позволяет модели изучать язык без ручной разметки.

## Computer vision

В задачах компьютерного зрения self-supervised методы могут использоваться для обучения моделей на изображениях.

Например, модель может:

- восстанавливать скрытые части изображения  
- определять положение фрагментов изображения  
- предсказывать преобразования изображения

## Представления данных

Self-supervised learning часто используется для обучения **представлений данных (representations)**.

После такого обучения модель может использоваться для других задач.

## Итог

Self-supervised learning применяется в задачах:

- обработки текста  
- анализа изображений  
- обучения представлений данных
