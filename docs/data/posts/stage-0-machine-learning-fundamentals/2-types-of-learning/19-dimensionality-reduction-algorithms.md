# 19. Какие алгоритмы используются для уменьшения размерности?

Существует несколько алгоритмов, которые используются для dimensionality reduction. Они помогают представить данные в пространстве меньшей размерности, сохраняя при этом основные закономерности.

## PCA (Principal Component Analysis)

PCA — один из самых популярных методов уменьшения размерности.

Он преобразует исходные признаки в новый набор признаков, которые называются **главными компонентами**.

Эти компоненты выбираются таким образом, чтобы сохранить как можно больше информации из исходных данных.

PCA часто используется для:

- анализа данных  
- предварительной обработки dataset  
- визуализации данных

## t-SNE

t-SNE — алгоритм, который часто применяется для визуализации высокоразмерных данных.

Он пытается расположить объекты в двумерном или трёхмерном пространстве так, чтобы похожие объекты оставались близко друг к другу.

Этот метод часто используется для анализа:

- изображений  
- текстовых данных  
- эмбеддингов

## UMAP

UMAP — современный алгоритм уменьшения размерности, который часто используется как альтернатива t-SNE.

Он может работать быстрее и лучше сохранять глобальную структуру данных.

UMAP широко применяется в задачах анализа больших datasets.

## Итог

Некоторые популярные алгоритмы dimensionality reduction:

- PCA  
- t-SNE  
- UMAP  

Каждый из них используется в разных ситуациях в зависимости от задачи.
