Help & Keyboard Shortcuts

Learn how to navigate the blog efficiently using keyboard shortcuts and other features.

🌐 Global Shortcuts

These shortcuts work from anywhere on the site:

Ctrl + S
Toggle Search Open or close the search input
Ctrl + C
Cycle Categories Navigate through main categories
Esc
Close Search Close search overlay and results
🌙/☀️ Button
Toggle Theme Switch between light and dark modes

🔍 Search Navigation

When search results are visible:

↑ ↓
Navigate Results Move focus between search results
Enter
Open Result Open the currently focused search result

📄 Post Card Navigation

On blog listing pages (Home, Categories, Subcategories):

↑ ↓ ← →
Navigate Posts Move focus between post cards
Enter
Open Post Open the currently focused post

🏷️ Tags

Tags are labels you add to posts yourself. They stay in this browser only — nothing is sent to a server.

Post
Add or remove tags on this post Scroll to Tags under the article. Click +, enter a tag (or pick one from the suggestions), then Add. Each tag shows as a chip: click the tag name to see all posts with that tag; click × to remove the tag from this post only.
Header
Tags menu After you have at least one tag, Tags appears next to Blog. Use it to open all posts with a given tag.
Reset
Clear everything Reset at the bottom of the Tags menu removes every tag from every post on this device. Pages that are already open refresh to match.

📖 Article Page

When you are reading a post (same actions as the arrow buttons at the bottom):

Ctrl + Shift + ←
Previous Article Go to the previous article in the curriculum order
Ctrl + Shift + →
Next Article Go to the next article in the curriculum order
