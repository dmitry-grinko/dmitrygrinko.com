# 46. Когда RMSE предпочтительнее MAE?

RMSE предпочтительнее, когда **большие ошибки особенно нежелательны**.

Поскольку RMSE возводит ошибки в квадрат, большие отклонения сильно увеличивают значение метрики.

## Примеры

Задачи, где большие ошибки критичны:

- прогнозирование нагрузки на энергосистему  
- прогнозирование спроса  
- финансовые прогнозы

## Практическое значение

RMSE заставляет модель сильнее избегать крупных ошибок.

## Итог

RMSE предпочтителен, когда большие ошибки должны наказываться сильнее.
