# 50. В каких задачах линейная регрессия остаётся хорошим baseline?

Линейная регрессия часто используется как baseline модель в задачах регрессии.

Она позволяет быстро получить первое приближение решения.

## Прогнозирование числовых значений

Линейная регрессия хорошо подходит для задач:

- прогнозирования цен
- прогнозирования продаж
- оценки спроса

## Экономические модели

Во многих экономических задачах зависимость между переменными часто приближённо линейная.

## Табличные данные

Если dataset состоит из табличных признаков, линейная регрессия может работать достаточно хорошо.

## Быстрая проверка гипотез

Линейная модель позволяет быстро проверить, есть ли зависимость между признаками и target.

## Итог

Линейная регрессия остаётся хорошим baseline для многих задач регрессии, особенно при работе с табличными данными.
