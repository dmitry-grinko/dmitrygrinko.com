# 25. Как выглядит формула log loss?

Формула log loss для бинарной классификации выглядит так:

Loss = − ( y log(p) + (1 − y) log(1 − p) )

где:

- y — истинный класс (0 или 1)  
- p — вероятность, предсказанная моделью

## Интерпретация

Если:

y = 1

формула превращается в:

Loss = − log(p)

Если:

y = 0

Loss = − log(1 − p)

## Что это означает

Чем ближе вероятность к правильному классу, тем меньше значение функции потерь.

## Итог

Log loss вычисляет ошибку на основе логарифма предсказанной вероятности правильного класса.
