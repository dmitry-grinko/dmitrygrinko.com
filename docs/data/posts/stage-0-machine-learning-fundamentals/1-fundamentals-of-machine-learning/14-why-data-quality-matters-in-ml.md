# 14. Почему качество данных критично для machine learning?

Модель машинного обучения учится на данных. Поэтому она может выучить только те закономерности, которые присутствуют в dataset.

Если данные плохие, модель тоже будет работать плохо.

Основные проблемы данных:

- **ошибки и шум** — неправильные значения или неточные измерения
- **плохая разметка** — label указан неверно
- **неполные данные** — важные признаки отсутствуют
- **смещение данных (bias)** — dataset не отражает реальную ситуацию

В таких случаях модель будет делать неправильные выводы, даже если используется хороший алгоритм.

## Итог

Качество модели напрямую зависит от качества данных.

Поэтому в реальных ML-проектах значительная часть работы связана с:

- сбором данных
- очисткой данных
- проверкой разметки
- подготовкой dataset
