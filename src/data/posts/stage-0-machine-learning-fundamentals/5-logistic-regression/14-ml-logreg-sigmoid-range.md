# 14. Почему выход sigmoid находится между 0 и 1?

Выход sigmoid всегда находится между 0 и 1 из-за её математической формы.

Формула функции:

σ(z) = 1 / (1 + e⁻ᶻ)

Экспонента e⁻ᶻ всегда положительна.

Это означает, что знаменатель выражения всегда больше 1.

## Следствие

Результат дроби никогда не может быть:

- меньше 0  
- больше 1

## Поведение функции

Если z становится очень большим:

e⁻ᶻ → 0

результат приближается к 1.

Если z становится очень отрицательным:

e⁻ᶻ становится большим

результат приближается к 0.

## Итог

Математическая форма sigmoid автоматически ограничивает её выход диапазоном от 0 до 1.
