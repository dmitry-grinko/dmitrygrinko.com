# 17. Что такое mean absolute error (MAE)?

Mean Absolute Error (MAE) — это метрика, которая измеряет **среднюю абсолютную ошибку модели**.

Она вычисляет среднее значение абсолютных отклонений между prediction и target.

## Формула

MAE = (1/n) Σ |y − ŷ|

## Как работает MAE

Каждая ошибка берётся по модулю.

Это означает:

- отрицательные и положительные ошибки учитываются одинаково  
- большие ошибки не усиливаются квадратом

## Пример

Ошибки модели:

-10  
5  
20

MAE:

(10 + 5 + 20) / 3 = 11.67

## Особенность MAE

MAE менее чувствителен к выбросам, чем MSE.

Поскольку ошибка не возводится в квадрат.

## Итог

MAE — это среднее значение абсолютных ошибок между предсказаниями модели и реальными значениями.
