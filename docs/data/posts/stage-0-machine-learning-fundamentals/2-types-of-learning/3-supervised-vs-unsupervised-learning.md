# 3. Чем supervised learning отличается от unsupervised learning?

Главное различие между supervised learning и unsupervised learning заключается в том, **есть ли в данных правильные ответы (labels)**.

В supervised learning модель обучается на данных, где известен правильный результат.  
В unsupervised learning таких ответов нет.

## Supervised learning

В supervised learning dataset содержит:

- features — входные признаки  
- label — правильный ответ

Модель получает примеры вида:

features → правильный результат

Во время обучения модель пытается научиться предсказывать label на основе features.

Такие модели используются, когда задача имеет **чётко определённый правильный ответ**.

## Unsupervised learning

В unsupervised learning dataset содержит только входные данные.

Правильных ответов в данных нет.

Модель должна самостоятельно находить:

- структуру данных  
- группы похожих объектов  
- скрытые закономерности

## Пример

Представим dataset с информацией о клиентах.

Если известно, какие клиенты совершили покупку, модель может учиться предсказывать это. Это будет supervised learning.

Если есть только данные о клиентах без правильного ответа, модель может попытаться разделить клиентов на группы с похожим поведением. Это будет unsupervised learning.

## Итог

Основное отличие:

- **supervised learning** использует размеченные данные  
- **unsupervised learning** работает с неразмеченными данными
