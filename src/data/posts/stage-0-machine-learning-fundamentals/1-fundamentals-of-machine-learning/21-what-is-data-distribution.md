# 21. Что такое distribution данных?

Distribution данных (распределение данных) — это то, **как значения данных распределены внутри dataset**.

Другими словами, distribution показывает:

- какие значения встречаются чаще
- какие реже
- какие диапазоны данных вообще присутствуют

Это помогает понять структуру данных, с которыми работает модель.

## Пример

Представим dataset с возрастом пользователей:

| age |
|----|
| 18 |
| 21 |
| 22 |
| 23 |
| 24 |
| 25 |
| 60 |

Если построить распределение, можно увидеть, что:

- большинство пользователей находятся в диапазоне **20–25 лет**
- значение **60** встречается редко

Это и есть distribution данных.

## Почему это важно в machine learning

Модели машинного обучения обучаются на определённом распределении данных.

Если новые данные имеют **другое распределение**, качество модели может ухудшиться.

Например:

- модель обучалась на пользователях 20–30 лет
- в production начинают приходить пользователи 50–70 лет

В этом случае модель может работать хуже, потому что такие данные она почти не видела во время обучения.

## Итог

Distribution данных — это то, как значения данных распределены внутри dataset.

Понимание распределения помогает:

- лучше анализировать данные
- находить аномалии
- понимать, насколько training data соответствует реальным данным.
