# 44. Как архитектура сети влияет на стабильность обучения?

Архитектура сети сильно влияет на то, насколько легко её обучать.

Некоторые архитектуры помогают градиентам распространяться стабильнее.

## Факторы, влияющие на обучение

### Глубина сети

Очень глубокие сети сложнее обучать.

### Тип функции активации

ReLU помогает уменьшить проблему исчезающих градиентов.

### Специальные архитектуры

Современные сети используют:

- residual connections
- normalization layers

## Пример

Residual connections позволяют градиенту проходить напрямую через несколько слоёв.

Это сильно облегчает обучение очень глубоких сетей.

## Итог

Правильная архитектура может значительно повысить стабильность обучения модели.
