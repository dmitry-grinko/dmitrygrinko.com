# 23. В каких ситуациях размеченных данных мало?

Во многих задачах machine learning размеченных данных оказывается значительно меньше, чем неразмеченных.

Это связано с тем, что **сбор данных обычно автоматизирован**, а разметка требует участия человека.

## Экспертная разметка

В некоторых областях только специалисты могут правильно размечать данные.

Например:

- медицинские изображения должны размечать врачи  
- юридические документы требуют экспертов  
- научные данные требуют специализированных знаний

Такой процесс может быть медленным и дорогим.

## Большие потоки данных

Многие современные системы постоянно генерируют новые данные.

Например:

- пользовательские действия в приложениях  
- изображения с камер  
- тексты из интернета  
- данные датчиков

Объем таких данных может быть огромным, но разметить их полностью невозможно.

## Сложность разметки

Иногда сама задача разметки может быть сложной.

Например:

- выделение объектов на изображениях  
- аннотирование видео  
- классификация сложных текстов

Разметка таких datasets может требовать большого количества времени.

## Итог

Размеченных данных часто мало, потому что их создание требует значительных ресурсов, экспертных знаний и времени.
