# 9. Почему градиент используется для обновления параметров модели?

Градиент показывает, как изменение параметров влияет на функцию потерь.

Поэтому его можно использовать, чтобы понять **в каком направлении изменять параметры модели**.

## Основная идея

Если градиент показывает направление роста функции, то движение в противоположном направлении уменьшает функцию.

Это позволяет уменьшать ошибку модели.

## Обновление параметров

Параметры обновляются по правилу:

new_parameter = old_parameter - learning_rate × gradient

## Результат

После обновления параметров функция потерь становится меньше.

Повторяя этот процесс много раз, можно найти минимум функции.

## Итог

Градиент используется для обновления параметров, потому что он показывает направление изменения функции потерь.
