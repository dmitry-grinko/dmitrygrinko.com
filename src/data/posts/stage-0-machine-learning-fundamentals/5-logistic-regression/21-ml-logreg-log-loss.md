# 21. Что такое log loss?

Log loss (или logistic loss, binary cross-entropy) — это функция потерь, которая используется для обучения моделей классификации, в том числе logistic regression.

Она измеряет, **насколько сильно предсказанная вероятность отличается от правильного класса**.

## Основная идея

В отличие от простой ошибки классификации, log loss учитывает **уверенность модели**.

Например:

- вероятность 0.9 для правильного класса — хорошее предсказание  
- вероятность 0.6 — менее уверенное  
- вероятность 0.1 — сильная ошибка

Log loss увеличивается, когда модель уверенно делает неправильные предсказания.

## Пример

Реальный класс:

y = 1

Предсказание модели:

p = 0.9 → маленький loss  
p = 0.6 → средний loss  
p = 0.1 → большой loss

## Итог

Log loss измеряет качество предсказанных вероятностей и используется как основная функция потерь в logistic regression.
