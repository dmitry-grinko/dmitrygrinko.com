# 18. Как параметры модели обновляются на каждой итерации?

На каждой итерации алгоритм выполняет обновление параметров на основе градиента функции потерь.

## Процесс

1. вычисляется градиент функции потерь  
2. определяется направление уменьшения функции  
3. параметры корректируются

## Формула обновления

θ_new = θ_old − η ∇L

## Что происходит

Если градиент большой, параметры изменяются сильнее.

Если градиент маленький, изменения будут небольшими.

## Повторение процесса

Этот процесс повторяется много раз, пока модель не приблизится к минимуму функции потерь.

## Итог

Параметры модели обновляются на основе градиента функции потерь на каждой итерации обучения.
