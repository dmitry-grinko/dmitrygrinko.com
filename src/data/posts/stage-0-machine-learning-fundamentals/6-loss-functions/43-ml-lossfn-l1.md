# 43. Что такое L1 регуляризация?

L1 регуляризация добавляет к функции потерь штраф за **абсолютные значения параметров модели**.

## Формула

L1 penalty = λ Σ |w|

где:

- w — параметры модели  
- λ — коэффициент регуляризации

## Особенность

L1 регуляризация может делать некоторые коэффициенты **равными нулю**.

Это означает, что соответствующие признаки исключаются из модели.

## Практическое значение

L1 регуляризация может выполнять:

- feature selection  
- уменьшение сложности модели

## Итог

L1 регуляризация штрафует абсолютные значения коэффициентов и может обнулять часть параметров.
