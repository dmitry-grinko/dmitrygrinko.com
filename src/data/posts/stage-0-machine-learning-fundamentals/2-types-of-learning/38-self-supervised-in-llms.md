# 38. Как self-supervised learning используется в large language models?

Self-supervised learning является основой обучения современных large language models.

Такие модели обучаются на огромных текстовых datasets и используют задачи предсказания текста как сигнал обучения.

## Основная задача обучения

Во многих языковых моделях используется простая идея.

Модель получает текст и должна предсказать следующее слово.

Например:

"The capital of France is"

Модель должна предсказать слово "Paris".

Повторяя эту задачу миллиарды раз на огромных datasets, модель начинает изучать структуру языка.

## Что модель изучает

Во время такого обучения модель постепенно учится:

- понимать синтаксис языка  
- распознавать контекст  
- связывать слова и идеи  
- формировать представления текста

Это позволяет модели решать множество разных задач.

## Почему это работает

Главное преимущество такого подхода заключается в том, что тексты для обучения можно получать автоматически из интернета.

Не требуется ручная разметка данных.

## Итог

Large language models обучаются с помощью self-supervised learning, используя задачи предсказания текста на огромных текстовых datasets.
