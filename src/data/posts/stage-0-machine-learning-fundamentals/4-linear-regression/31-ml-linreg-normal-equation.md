# 31. Что такое normal equation в линейной регрессии?

Normal equation — это **аналитический способ вычисления параметров линейной регрессии** без использования gradient descent.

Он позволяет найти оптимальные коэффициенты модели напрямую через матричные операции.

## Основная идея

Вместо того чтобы постепенно уменьшать ошибку, как делает gradient descent, normal equation вычисляет параметры модели **за один шаг**.

Формула выглядит так:

θ = (XᵀX)⁻¹ Xᵀ y

где:

- X — матрица признаков  
- y — вектор целевых значений  
- θ — вектор параметров модели

## Что происходит в этой формуле

Формула фактически решает задачу минимизации MSE аналитически.

Результатом является набор коэффициентов, который минимизирует квадратичную ошибку.

## Когда это используется

Normal equation работает хорошо, когда:

- dataset небольшой  
- количество признаков невелико

## Итог

Normal equation — это аналитический метод вычисления коэффициентов линейной регрессии без итерационного обучения.
