# 15. Почему минимизация loss приводит к улучшению модели?

Функция потерь измеряет, насколько предсказания модели отличаются от правильных ответов.

Если loss уменьшается, значит предсказания становятся ближе к target.

## Пример

Если модель предсказывает:

реальное значение = 100  
prediction = 70

ошибка большая.

После обучения:

prediction = 95

ошибка уменьшается.

## Результат

Минимизация loss заставляет модель:

- улучшать предсказания  
- корректировать параметры  
- лучше объяснять данные

## Итог

Минимизация loss приводит к улучшению модели, потому что уменьшает ошибку предсказаний.
