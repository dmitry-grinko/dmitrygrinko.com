# 22. Почему важно, чтобы training и test данные имели похожую distribution?

Чтобы оценка модели была корректной, training и test dataset должны иметь **похожее распределение данных**.

Если распределения сильно отличаются, результаты тестирования будут вводить в заблуждение.

## Что происходит при разной distribution

Если training dataset отличается от test dataset, модель может столкнуться с ситуацией, когда она видит данные, которых не было во время обучения.

Например:

training data:

- пользователи из Европы

test data:

- пользователи из Азии

Модель может плохо работать просто потому, что она не обучалась на таких данных.

## Почему это проблема

Метрики тестирования должны показывать, как модель будет работать в реальном мире.

Если test dataset отличается от training dataset, тест перестаёт быть честной оценкой.

## Итог

Training и test dataset должны иметь похожую distribution, чтобы оценка модели отражала её реальное качество.
