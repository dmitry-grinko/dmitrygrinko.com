# 17. Что такое threshold в классификации?

Threshold — это **порог вероятности**, который используется для преобразования вероятности в класс.

Он определяет границу между классами.

## Как работает threshold

Если вероятность выше threshold, объект относится к положительному классу.

Если ниже — к отрицательному.

## Пример

P(spam) = 0.62

Если threshold = 0.5

письмо будет классифицировано как spam.

Если threshold = 0.8

то же письмо будет классифицировано как not spam.

## Почему threshold важен

Изменение threshold может сильно изменить поведение модели.

Это позволяет балансировать разные типы ошибок.

## Итог

Threshold — это порог вероятности, который определяет, какой класс будет предсказан.
