# 45. Как определить, что модель обучается правильно?

Чтобы понять, что модель обучается правильно, обычно сравнивают её качество на разных частях данных.

Главная идея простая: модель должна показывать **хорошие и примерно похожие результаты** как на обучающих данных, так и на новых данных.

## Что обычно проверяют

Во время обучения оценивают метрики модели на:

- training set
- validation set (или test set)

Если модель обучается корректно, метрики на этих данных будут достаточно близкими.

Например:

training accuracy = 0.86
validation accuracy = 0.84

Такая ситуация обычно означает, что модель нашла реальные закономерности и умеет их обобщать.

## Признаки проблем

Если результаты сильно отличаются, это может указывать на проблему.

**Переобучение (overfitting)**

training accuracy = 0.99
validation accuracy = 0.72

Модель слишком хорошо запомнила training data и плохо работает на новых данных.

**Недообучение (underfitting)**

training accuracy = 0.60
validation accuracy = 0.58

Модель слишком простая и не может уловить закономерности в данных.

## Итог

Модель обычно считается обученной правильно, если:

- качество на training data достаточно высокое
- качество на validation или test data близко к training
- модель сохраняет хорошее качество на новых данных
