# 20. Какие проблемы возникают при маленьком dataset?

Маленький dataset может создавать несколько серьёзных проблем для обучения моделей.

Основная проблема заключается в том, что модель видит слишком мало примеров.

## Переобучение

Если данных мало, модель может начать запоминать training data.

В этом случае она будет хорошо работать на обучающих данных, но плохо на новых.

## Нестабильные оценки

При маленьком dataset результаты оценки модели могут сильно зависеть от того, какие данные попали в training и test выборки.

Это делает метрики менее надёжными.

## Ограниченная информация

Маленький dataset может не отражать всю структуру реальных данных.

Модель может не увидеть редкие случаи или важные закономерности.

## Сложность моделей

Сложные модели обычно требуют большого количества данных.

При маленьком dataset они часто переобучаются.

## Итог

Маленький dataset может приводить к:

- переобучению  
- нестабильным оценкам  
- слабой способности модели обобщать данные
