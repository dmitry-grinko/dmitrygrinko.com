# 48. Как интерпретировать значение R²?

R² показывает, насколько хорошо модель объясняет наблюдаемые данные.

## Интерпретация

R² = 0  
модель не лучше простой константы

R² = 0.5  
модель объясняет половину вариации данных

R² = 1  
идеальное предсказание

## Ограничение

Высокое значение R² не всегда означает хорошую модель.

Например, модель может переобучиться.

## Итог

R² показывает долю вариации данных, объясняемую моделью.
