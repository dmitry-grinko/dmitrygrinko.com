# 48. Когда SGD может работать лучше Adam?

Несмотря на популярность Adam, в некоторых задачах SGD может давать лучшие результаты.

## Основная причина

SGD иногда лучше **обобщает на новых данных**.

## Примеры

В задачах computer vision модели, обученные с SGD, могут показывать более высокую точность.

## Возможная причина

Adam быстрее находит решение, но оно может быть более сложным и хуже обобщаться.

SGD может находить более «простые» решения.

## Итог

SGD может работать лучше Adam в задачах, где важна хорошая обобщающая способность модели.
