# 1. Что такое тип обучения (learning paradigm) в machine learning?

Тип обучения (learning paradigm) — это способ, которым модель машинного обучения получает знания из данных.

Другими словами, это **общая схема обучения**, которая определяет:

- какие данные доступны модели  
- какую задачу она решает  
- какую информацию получает во время обучения

Разные типы обучения отличаются тем, **есть ли в данных правильные ответы**, и каким образом модель должна находить закономерности.

## Почему это важно

Тип обучения определяет, как именно строится процесс обучения модели.

Например:

- есть ли в dataset правильные ответы  
- должна ли модель сама находить структуру в данных  
- получает ли модель обратную связь во время обучения

От этого зависит:

- какие данные можно использовать  
- какие алгоритмы применяются  
- какие задачи можно решать

## Пример

Представим dataset с изображениями животных.

В одном случае у каждого изображения есть правильная метка:

- cat  
- dog  
- bird  

Модель может учиться сопоставлять изображение и правильную категорию.

В другом случае меток нет. Есть только сами изображения. Тогда модель может пытаться самостоятельно находить структуру в данных, например группировать похожие изображения.

Это два разных типа обучения.

## Основная идея

Тип обучения определяет, **какую информацию получает модель во время обучения** и **как она использует данные для поиска закономерностей**.

## Итог

Тип обучения (learning paradigm) — это общий подход к обучению моделей машинного обучения, который определяет, какие данные доступны модели и каким образом она извлекает из них закономерности.
