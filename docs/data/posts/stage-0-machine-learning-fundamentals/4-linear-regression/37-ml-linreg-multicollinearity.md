# 37. Что такое multicollinearity?

Multicollinearity — это ситуация, когда **несколько признаков в модели сильно коррелируют друг с другом**.

Это означает, что признаки содержат похожую информацию.

## Пример

В dataset могут быть признаки:

- площадь квартиры  
- площадь кухни  
- площадь всех комнат

Эти переменные могут быть сильно связаны между собой.

## Почему это проблема

Когда признаки сильно коррелируют:

- модель не может точно определить вклад каждого признака  
- коэффициенты становятся нестабильными

Небольшое изменение данных может сильно изменить значения коэффициентов.

## Влияние на модель

Multicollinearity не всегда ухудшает prediction.

Но она делает модель менее интерпретируемой и менее устойчивой.

## Итог

Multicollinearity — это сильная корреляция между признаками в модели.
