# 46. Какие ограничения есть у logistic regression?

Logistic regression — простая и эффективная модель, но у неё есть несколько ограничений.

## Линейная граница решений

Модель предполагает линейную границу между классами.

Если реальные зависимости сильно нелинейные, модель может работать плохо.

## Ограниченная гибкость

Logistic regression не может автоматически находить сложные взаимодействия признаков.

## Чувствительность к признакам

Качество модели сильно зависит от:

- выбора признаков  
- feature engineering

## Ограничения данных

Если данные сильно шумные или сложные, более сложные модели могут работать лучше.

## Итог

Основные ограничения logistic regression связаны с её линейной природой.
