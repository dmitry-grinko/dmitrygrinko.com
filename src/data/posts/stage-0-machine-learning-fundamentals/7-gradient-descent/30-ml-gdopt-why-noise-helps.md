# 30. Почему шум может помогать оптимизации?

Хотя шум делает градиенты менее точными, он может быть полезен для оптимизации.

## Основная причина

Шум помогает алгоритму **выходить из плохих локальных решений**.

## Пример

Если алгоритм попал в небольшой локальный минимум, точный градиент может удерживать его там.

Шум может слегка изменить направление обновления и помочь выйти из этой области.

## Практическое следствие

Поэтому методы с шумными градиентами, такие как SGD или mini-batch обучение, часто работают лучше на сложных моделях.

## Итог

Шум в градиентах может помогать оптимизации, потому что он делает процесс поиска решения менее детерминированным.
