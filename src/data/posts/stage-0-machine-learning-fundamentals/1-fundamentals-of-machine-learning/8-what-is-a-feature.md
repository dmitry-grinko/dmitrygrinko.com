# 8. Что такое feature?

Feature (признак) — это отдельная характеристика объекта, которую модель использует для принятия решения.

Проще говоря, feature — это **входные данные модели**.

Например, если модель предсказывает цену квартиры, features могут быть:

- площадь
- количество комнат
- этаж
- район
- год постройки

Каждый объект в dataset представлен набором таких признаков.

| area | rooms | floor | price |
|-----|------|------|------|
| 45  | 1    | 3    | 120000 |
| 70  | 2    | 5    | 185000 |

Здесь:

- `area`, `rooms`, `floor` — features
- `price` — значение, которое модель пытается предсказать

## Важная деталь

В машинном обучении качество модели часто зависит не столько от алгоритма, сколько от **выбранных признаков**.

Поэтому в реальных проектах много внимания уделяется:

- выбору полезных признаков
- преобразованию данных
- созданию новых признаков

Этот процесс называется **feature engineering**.

## Итог

Feature — это отдельная характеристика объекта, которая используется моделью как входные данные для обучения и предсказаний.
