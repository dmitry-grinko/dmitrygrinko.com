# 39. Почему gradient descent используется для обучения сетей?

Gradient descent используется потому, что он позволяет **эффективно находить параметры, минимизирующие функцию потерь**.

В нейронных сетях пространство параметров может быть очень большим.

Иногда это миллионы параметров.

## Основная идея

Gradient descent делает небольшие шаги в направлении уменьшения ошибки.

Каждый шаг:

- немного корректирует веса
- уменьшает функцию потерь

## Почему это работает

Градиент показывает направление, в котором функция растёт быстрее всего.

Поэтому движение в противоположном направлении уменьшает её.

## Практическая причина

Gradient descent:

- прост
- масштабируется
- хорошо работает с большими моделями

## Итог

Gradient descent используется для постепенного уменьшения ошибки модели.
