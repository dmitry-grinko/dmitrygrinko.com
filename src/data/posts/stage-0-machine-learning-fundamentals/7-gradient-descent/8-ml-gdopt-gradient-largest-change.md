# 8. Почему градиент показывает направление наибольшего изменения функции?

Градиент указывает направление, в котором функция увеличивается **быстрее всего**.

Это свойство следует из математического определения производной.

## Интуитивное объяснение

Представим горную поверхность.

Если стоять в какой-то точке, градиент укажет направление самого крутого подъёма.

## Почему это важно

Если двигаться **в противоположном направлении**, можно быстрее всего уменьшить значение функции.

Именно это делает gradient descent.

## Итог

Градиент показывает направление наибольшего роста функции, поэтому движение против градиента уменьшает значение функции быстрее всего.
