# 41. Что такое dropout?

Dropout — это метод регуляризации, используемый в нейронных сетях, при котором **случайная часть нейронов временно отключается во время обучения**.

Это означает, что на каждой итерации обучения сеть работает с немного другой архитектурой.

## Основная идея

Во время обучения часть нейронов случайно «выключается».

Их выходы просто обнуляются.

В результате сеть не может слишком сильно полагаться на отдельные нейроны.

## Что это даёт

Каждый нейрон должен работать вместе с разными комбинациями других нейронов.

Это делает модель более устойчивой.

## Итог

Dropout — это метод регуляризации, который случайно отключает нейроны во время обучения.
