# 10. Что такое prediction?

Prediction — это результат, который выдаёт модель машинного обучения после обработки входных данных.

Проще говоря, prediction — это **ответ модели**.

Общая схема выглядит так:

```
features → model → prediction
```

Где:

- **features** — входные признаки объекта
- **model** — обученная модель
- **prediction** — результат, который модель вычисляет на основе этих признаков

## Пример

Представим модель, которая предсказывает цену квартиры.

В систему передаются признаки квартиры:

| area | rooms |
|-----|------|
| 70  | 2 |

Модель обрабатывает эти данные и выдаёт prediction:

```
predicted price = 185000
```

Это и есть prediction — значение, которое модель оценила на основе признаков.

## Связь с label

Во время обучения модели используется ещё одно значение — **label**.

Label — это правильный ответ из обучающих данных.

Например:

| area | rooms | price |
|-----|------|------|
| 70  | 2    | 180000 |

Здесь:

- `price` — label
- предсказание модели — prediction

Во время обучения алгоритм сравнивает **prediction** и **label** и корректирует параметры модели, чтобы уменьшить ошибку.

После обучения label обычно уже неизвестен, и система работает только так:

features → model → prediction

## Тип prediction зависит от задачи

Тип предсказания зависит от того, какую задачу решает модель.

**Классификация**

```
Prediction — это категория.

Примеры:

- spam / not spam
- cat / dog
- fraud / normal transaction
```

**Регрессия**

```
Prediction — числовое значение.

Примеры:

- цена квартиры
- температура
- спрос на товар
```

**Вероятность**

```
Иногда prediction — это вероятность события.

Например:

probability of spam = 0.82
```

## Итог

Prediction — это результат работы модели машинного обучения: значение, которое она вычисляет на основе входных признаков после обучения.
