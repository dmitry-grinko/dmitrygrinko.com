# 45. Что такое deep learning?

Deep learning — это область машинного обучения, которая использует **глубокие нейронные сети**.

Слово "deep" означает, что сеть содержит **много слоёв**.

## Основная идея

Глубокие сети могут автоматически извлекать сложные признаки из данных.

В отличие от классических методов, где признаки часто создаются вручную.

## Пример

В задаче распознавания изображений сеть может автоматически научиться обнаруживать:

- края  
- текстуры  
- формы  
- объекты

## Примеры моделей

Deep learning включает такие архитектуры как:

- CNN (convolutional neural networks)  
- RNN (recurrent neural networks)  
- Transformers

## Итог

Deep learning — это направление машинного обучения, основанное на использовании глубоких нейронных сетей.
