# 2. Какие задачи решает linear regression?

Linear regression используется в задачах, где нужно **предсказать числовое значение**.

Такие задачи называются **regression tasks**.

## Примеры задач

Linear regression может применяться в разных областях.

Например:

- предсказание цены недвижимости  
- прогнозирование продаж  
- оценка спроса  
- прогноз температуры  
- прогноз времени доставки  

Во всех этих задачах результат является **числом**.

## Пример

Представим dataset:

| area | rooms | price |
|----|----|----|
| 45 | 1 | 120000 |
| 70 | 2 | 185000 |
| 80 | 3 | 240000 |

Задача модели — научиться предсказывать цену квартиры на основе её характеристик.

## Когда используется linear regression

Linear regression хорошо работает, когда:

- зависимость между переменными близка к линейной
- данные не слишком шумные
- признаки имеют понятную интерпретацию

## Итог

Linear regression используется для задач, где необходимо предсказать **непрерывное числовое значение** на основе признаков.
