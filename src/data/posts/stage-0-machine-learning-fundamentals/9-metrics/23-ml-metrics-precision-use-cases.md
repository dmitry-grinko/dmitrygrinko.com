# 23. В каких задачах важна высокая precision?

Высокая precision важна в задачах, где **ложные срабатывания являются дорогими или нежелательными**.

## Примеры

### Спам-фильтры

Если письмо ошибочно помечено как спам, пользователь может пропустить важное сообщение.

### Фильтрация контента

Ошибочное удаление нормального контента может быть проблемой.

### Финансовые системы

Ложное обвинение в мошенничестве может вызвать серьёзные последствия.

## Практическая цель

В таких задачах важно, чтобы положительные предсказания были максимально точными.

## Итог

Высокая precision важна, когда ложные положительные предсказания особенно нежелательны.
