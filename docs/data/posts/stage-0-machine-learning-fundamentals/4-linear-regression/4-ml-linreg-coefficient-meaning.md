# 4. Что означает коэффициент в линейной регрессии?

Коэффициент показывает, **насколько сильно признак влияет на результат модели**.

Каждый признак имеет свой коэффициент.

## Интерпретация коэффициента

Если коэффициент положительный, увеличение признака увеличивает предсказание.

Если коэффициент отрицательный, увеличение признака уменьшает предсказание.

## Пример

Модель:

price = 2000 × area + 10000

Здесь коэффициент 2000 означает:

каждый дополнительный квадратный метр увеличивает цену на 2000.

## Несколько признаков

Если модель содержит несколько признаков:

price = 2000 × area + 5000 × rooms + 10000

каждый коэффициент показывает влияние конкретного признака.

## Почему коэффициенты важны

Коэффициенты позволяют интерпретировать модель.

Можно понять:

- какие признаки важнее
- какие признаки увеличивают или уменьшают результат

## Итог

Коэффициент в линейной регрессии показывает силу и направление влияния признака на предсказание модели.
