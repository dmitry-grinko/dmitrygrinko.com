# 35. Что такое AUC (Area Under the Curve)?

AUC (Area Under the Curve) — это метрика, которая измеряет **площадь под ROC кривой**.

Она показывает, насколько хорошо модель различает два класса.

## Диапазон значений

AUC принимает значения от 0 до 1.

- 0.5 — модель не лучше случайного угадывания  
- 1.0 — идеальная модель

## Интерпретация

AUC можно интерпретировать как вероятность того, что модель правильно ранжирует случайную пару объектов:

- один положительный
- один отрицательный

## Итог

AUC измеряет способность модели различать классы независимо от выбранного threshold.
