# 25. Какие методы semi-supervised learning существуют?

Существует несколько методов, которые позволяют обучать модели на смеси размеченных и неразмеченных данных.

Эти методы используют различные способы извлечения информации из неразмеченного dataset.

## Self-training

В self-training модель сначала обучается на размеченных данных.

После этого она делает предсказания для неразмеченных примеров.

Если модель уверена в своём предсказании, эти примеры могут добавляться в training dataset как псевдоразмеченные.

## Co-training

В co-training используются несколько моделей.

Каждая модель обучается на своей части признаков и может помогать размечать данные для других моделей.

Такой подход может улучшать устойчивость обучения.

## Graph-based методы

Некоторые методы представляют dataset как граф, где объекты соединяются на основе сходства.

Информация о размеченных примерах распространяется по этому графу на неразмеченные данные.

## Итог

Некоторые методы semi-supervised learning включают:

- self-training  
- co-training  
- graph-based методы

Все они пытаются использовать структуру неразмеченных данных для улучшения обучения модели.
