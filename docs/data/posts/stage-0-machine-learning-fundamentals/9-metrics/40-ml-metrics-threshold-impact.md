# 40. Как threshold влияет на метрики классификации?

Threshold — это порог вероятности, при котором модель решает, относится ли объект к положительному классу.

Изменение threshold влияет на все метрики классификации.

## Если уменьшить threshold

Модель чаще предсказывает положительный класс.

Это приводит к:

- увеличению recall  
- уменьшению precision

## Если увеличить threshold

Модель становится более осторожной.

Это приводит к:

- увеличению precision  
- уменьшению recall

## Практическая задача

Выбор threshold зависит от того, какие ошибки более критичны для задачи.

## Итог

Threshold напрямую влияет на баланс между precision, recall и другими метриками классификации.
