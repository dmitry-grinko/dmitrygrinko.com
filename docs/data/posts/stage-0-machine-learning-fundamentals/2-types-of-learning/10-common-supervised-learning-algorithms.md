# 10. Какие алгоритмы часто используются для supervised learning?

Для задач supervised learning существует большое количество алгоритмов. Все они решают одну и ту же задачу — пытаются найти зависимость между входными признаками и правильным результатом.

Разные алгоритмы используют разные подходы для поиска этой зависимости.

## Линейные модели

Линейные модели предполагают, что зависимость между признаками и результатом можно описать линейной функцией.

К таким алгоритмам относятся:

- Linear Regression  
- Logistic Regression

Они просты, интерпретируемы и часто используются как baseline.

## Деревья решений

Decision trees строят модель в виде набора правил.

Алгоритм постепенно делит данные на группы на основе признаков.

Деревья решений являются основой для многих популярных алгоритмов.

## Ансамблевые методы

Ансамблевые методы объединяют несколько моделей для повышения качества.

Примеры:

- Random Forest  
- Gradient Boosting  
- XGBoost  
- LightGBM

Такие алгоритмы часто показывают высокую точность на табличных данных.

## Нейронные сети

Нейронные сети используются в задачах, где данные имеют сложную структуру.

Например:

- изображения  
- текст  
- звук

Современные deep learning модели являются развитием этой идеи.

## Итог

Для задач supervised learning используется множество алгоритмов, включая:

- линейные модели  
- деревья решений  
- ансамблевые методы  
- нейронные сети

Выбор алгоритма зависит от задачи, данных и требований к модели.
