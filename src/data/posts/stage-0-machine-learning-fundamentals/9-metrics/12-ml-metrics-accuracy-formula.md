# 12. Как вычисляется accuracy?

Accuracy вычисляется как отношение числа правильных предсказаний к общему количеству предсказаний.

## Формула

accuracy = (TP + TN) / (TP + TN + FP + FN)

где:

- TP — true positives  
- TN — true negatives  
- FP — false positives  
- FN — false negatives

## Что означает формула

TP и TN — это правильные предсказания.

FP и FN — это ошибки модели.

Accuracy показывает, какая доля всех предсказаний оказалась правильной.

## Итог

Accuracy — это отношение числа правильных предсказаний к общему числу предсказаний.
