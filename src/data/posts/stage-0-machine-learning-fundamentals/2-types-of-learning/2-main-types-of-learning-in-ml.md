# 2. Какие основные типы обучения существуют в machine learning?

В машинном обучении существует несколько основных типов обучения. Они отличаются тем, **какая информация доступна модели во время обучения** и **какую задачу она должна решать**.

На практике чаще всего выделяют несколько базовых подходов.

## Supervised learning

В supervised learning модель обучается на данных, где для каждого примера известен **правильный ответ (label)**.

Модель получает:

- входные данные (features)  
- правильный результат (label)

Задача модели — научиться предсказывать правильный ответ для новых данных.

## Unsupervised learning

В unsupervised learning правильных ответов в данных нет.

Dataset содержит только входные данные, и модель должна **самостоятельно находить структуру или закономерности**.

Например, модель может:

- группировать похожие объекты  
- находить скрытые структуры в данных

## Semi-supervised learning

В этом подходе используется **небольшое количество размеченных данных и большое количество неразмеченных**.

Модель использует размеченные данные для обучения, а неразмеченные — для улучшения качества модели.

Такой подход часто применяется, когда разметка данных дорогая или сложная.

## Self-supervised learning

Self-supervised learning — это подход, при котором модель сама создаёт обучающие задачи на основе данных.

Этот метод широко используется в современных системах:

- NLP  
- computer vision  
- large language models

## Reinforcement learning

В reinforcement learning модель обучается через взаимодействие со средой.

Она получает:

- состояние среды  
- действие  
- награду (reward)

Цель модели — научиться принимать решения, которые максимизируют суммарную награду.

## Итог

Основные типы обучения в machine learning:

- supervised learning  
- unsupervised learning  
- semi-supervised learning  
- self-supervised learning  
- reinforcement learning
