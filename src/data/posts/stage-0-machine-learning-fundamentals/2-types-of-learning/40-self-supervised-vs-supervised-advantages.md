# 40. Какие преимущества self-supervised learning по сравнению с supervised learning?

Self-supervised learning имеет несколько преимуществ по сравнению с традиционным supervised learning.

Главное преимущество связано с тем, что этот подход позволяет использовать огромные объемы неразмеченных данных.

## Меньшая зависимость от разметки

В supervised learning модель требует dataset с правильными ответами.

Создание таких datasets часто требует:

- времени  
- человеческого труда  
- экспертных знаний

Self-supervised learning уменьшает необходимость ручной разметки.

## Использование больших datasets

Большая часть данных в мире не имеет разметки.

Self-supervised learning позволяет обучать модели на:

- текстах  
- изображениях  
- аудио  
- видео

Это значительно расширяет доступные данные для обучения.

## Предварительное обучение моделей

Self-supervised learning часто используется для **pretraining**.

Модель сначала обучается на огромном dataset, а затем дообучается на небольшой размеченной задаче.

Этот подход широко используется в современных системах deep learning.

## Итог

Self-supervised learning позволяет обучать модели на больших неразмеченных datasets и уменьшает зависимость от ручной разметки данных.
