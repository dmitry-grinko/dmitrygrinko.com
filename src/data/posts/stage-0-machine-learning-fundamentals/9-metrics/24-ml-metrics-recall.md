# 24. Что такое recall?

Recall — это метрика классификации, которая показывает **какую долю реальных положительных объектов модель смогла обнаружить**.

Она отвечает на вопрос:

> Сколько настоящих положительных случаев модель нашла?

## Формула

recall = TP / (TP + FN)

где:

TP — true positives  
FN — false negatives

## Интерпретация

Высокий recall означает, что модель редко пропускает положительные случаи.

## Пример

Если в dataset есть 100 случаев мошенничества, а модель обнаружила 90 из них:

recall = 90 / 100 = 0.9

## Итог

Recall показывает способность модели находить положительные объекты.
