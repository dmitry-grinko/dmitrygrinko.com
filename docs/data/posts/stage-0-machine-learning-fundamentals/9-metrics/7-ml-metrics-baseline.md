# 7. Что такое baseline метрика?

Baseline метрика — это **качество простой модели или простого правила**, с которым сравнивают более сложные модели.

Она служит отправной точкой для оценки прогресса.

## Пример baseline

Для классификации baseline может быть:

предсказывать самый частый класс.

Если в dataset 70% объектов одного класса, baseline accuracy будет 70%.

## Почему это полезно

Baseline помогает понять, действительно ли новая модель улучшает результат.

## Итог

Baseline метрика — это минимальный уровень качества, который должна превзойти модель.
