# 28. Когда Huber loss полезен?

Huber loss полезен, когда dataset содержит **выбросы**, но при этом важно точно учитывать небольшие ошибки.

## Типичные случаи

Regression задачи с шумными данными.

Например:

- прогнозирование цен  
- финансовые данные  
- сенсорные измерения

## Почему не MSE

MSE слишком сильно реагирует на выбросы.

## Почему не MAE

MAE может давать менее стабильные градиенты.

## Итог

Huber loss используется, когда требуется баланс между устойчивостью и точностью.
