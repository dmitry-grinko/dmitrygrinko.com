# 35. Что такое missing data?

Missing data — это пропущенные значения в dataset.

Это ситуации, когда для некоторого признака значение отсутствует.

## Пример

Dataset:

| age | income | purchases |
|----|-------|----------|
| 25 | 40000 | 3 |
| 32 | — | 7 |
| 45 | 80000 | 12 |

Во второй строке значение income отсутствует.

## Почему возникают missing values

Пропущенные значения могут появляться по разным причинам:

- ошибки сбора данных  
- неполные анкеты  
- проблемы с датчиками  
- технические ошибки

## Почему это проблема

Многие алгоритмы машинного обучения не умеют работать с пропущенными значениями.

Если dataset содержит missing values, модель может:

- выдавать ошибки  
- обучаться неправильно

## Итог

Missing data — это отсутствующие значения признаков в dataset, которые необходимо обрабатывать перед обучением модели.
