# 37. Что такое imputation?

Imputation — это процесс **заполнения пропущенных значений** в dataset.

Вместо удаления строк с missing data пропущенные значения заменяются некоторым вычисленным значением.

## Простые методы imputation

Самые распространённые методы:

Среднее значение

Пропущенные значения заменяются средним значением признака.

Медиана

Часто используется для данных с выбросами.

Наиболее частое значение

Используется для категориальных признаков.

## Пример

Dataset:

| age | income |
|----|-------|
| 25 | 40000 |
| 32 | — |
| 45 | 80000 |

Если средний income равен 60000, пропущенное значение можно заменить этим числом.

## Более сложные методы

Иногда используются более сложные подходы:

- модели регрессии  
- KNN imputation  
- итеративная имputation

## Итог

Imputation — это метод заполнения пропущенных значений для сохранения данных в dataset.
