# 28. Почему модели деградируют со временем?

Модели машинного обучения часто со временем начинают работать хуже.

Это связано с тем, что реальные данные постепенно меняются.

## Изменение поведения пользователей

Поведение людей может меняться.

Например:

- новые привычки  
- новые технологии  
- новые продукты

Модель, обученная на старых данных, может не учитывать эти изменения.

## Изменение среды

Иногда изменяются сами условия системы.

Например:

- новые рынки  
- новые категории товаров  
- новые правила бизнеса

Это может менять структуру данных.

## Dataset drift

Со временем распределение данных может изменяться.

Это приводит к тому, что модель работает в условиях, для которых она не обучалась.

## Итог

Модели деградируют со временем, потому что реальные данные и условия системы постепенно меняются.
