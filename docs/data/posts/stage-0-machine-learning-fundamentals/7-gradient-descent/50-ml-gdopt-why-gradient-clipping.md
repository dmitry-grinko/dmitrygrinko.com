# 50. Почему gradient clipping используется при обучении нейронных сетей?

Gradient clipping используется, чтобы предотвратить проблему **exploding gradients**.

## Что происходит без clipping

Если градиенты становятся слишком большими:

- параметры модели могут резко изменяться  
- обучение становится нестабильным  
- функция потерь может резко увеличиваться

## Как помогает clipping

Gradient clipping ограничивает максимальный размер градиента.

Это стабилизирует процесс оптимизации.

## Практический результат

Clipping позволяет обучать глубокие модели без разрушительных обновлений параметров.

## Итог

Gradient clipping используется для стабилизации обучения нейронных сетей.
