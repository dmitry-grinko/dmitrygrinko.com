# 44. Почему dropout отключается во время inference?

Во время inference dropout отключается, потому что модель должна использовать **все обученные нейроны**.

Во время обучения dropout используется только как способ регуляризации.

## Что происходит при inference

Все нейроны активны.

Однако веса масштабируются так, чтобы компенсировать эффект dropout.

## Почему это важно

Если оставить dropout во время inference, предсказания модели станут случайными.

## Итог

Dropout используется только во время обучения и отключается во время inference.
