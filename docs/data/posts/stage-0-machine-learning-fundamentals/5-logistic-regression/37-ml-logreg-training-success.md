# 37. Как определить, что обучение прошло успешно?

Есть несколько признаков того, что обучение модели прошло успешно.

## Снижение функции потерь

Во время обучения значение log loss должно постепенно уменьшаться.

## Стабилизация ошибки

После определённого количества итераций loss перестаёт сильно изменяться.

## Хорошие метрики

Модель должна показывать хорошие результаты на:

- validation dataset  
- test dataset

## Отсутствие переобучения

Если модель работает хорошо только на training data, обучение нельзя считать успешным.

## Итог

Успешное обучение характеризуется уменьшением loss и хорошими результатами на новых данных.
