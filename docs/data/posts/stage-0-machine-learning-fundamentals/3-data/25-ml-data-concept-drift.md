# 25. Что такое concept drift?

Concept drift — это ситуация, когда **меняется сама зависимость между признаками и target**.

То есть меняется правило, которое модель пытается выучить.

## Пример

Представим модель, которая определяет спам.

Со временем спамеры начинают использовать новые методы.

Слова и структуры писем меняются.

В результате зависимость между текстом письма и тем, является ли оно спамом, изменяется.

## Чем concept drift отличается от covariate shift

При covariate shift:

- меняется распределение признаков

При concept drift:

- меняется сама функция зависимости

features → target

## Почему это серьёзная проблема

Если concept drift произошёл, старая модель может стать неактуальной.

Даже если она была хорошо обучена.

## Итог

Concept drift — это изменение зависимости между признаками и целевой переменной со временем.
