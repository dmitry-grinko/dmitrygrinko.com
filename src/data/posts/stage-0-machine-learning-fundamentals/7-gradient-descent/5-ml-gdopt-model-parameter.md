# 5. Что такое параметр модели?

Параметры модели — это значения, которые **определяют поведение модели**.

Именно параметры позволяют модели преобразовывать входные данные в предсказания.

## Примеры параметров

В линейной регрессии параметры — это коэффициенты:

y = w₁x₁ + w₂x₂ + b

где:

- w₁, w₂ — веса  
- b — bias

В нейронных сетях параметрами являются:

- веса между нейронами  
- bias нейронов

## Что делает обучение

Во время обучения параметры постепенно изменяются.

Алгоритм ищет такие значения параметров, которые минимизируют функцию потерь.

## Размер параметров

В простых моделях может быть десятки параметров.

В современных нейронных сетях их может быть **миллионы или миллиарды**.

## Итог

Параметры модели — это значения, которые определяют, как модель делает предсказания.
