# 34. Что такое L2 regularization?

L2 regularization — это метод регуляризации, при котором к функции потерь добавляется штраф за **квадраты параметров модели**.

## Формула

penalty = λ Σ w²

## Что происходит

Если веса модели становятся большими, штраф быстро увеличивается.

Алгоритм обучения начинает уменьшать значения параметров.

## Отличие от L1

L2 регуляризация обычно **уменьшает веса**, но редко делает их равными нулю.

## Практический эффект

Это делает модель более стабильной и менее чувствительной к шуму.

## Итог

L2 regularization уменьшает величину параметров модели, снижая риск переобучения.
