# 42. Как momentum ускоряет обучение?

Momentum ускоряет обучение за счёт накопления информации о предыдущих обновлениях параметров.

## Основная идея

Если градиенты долго указывают в одном направлении, momentum увеличивает шаг обновления.

Если направление резко меняется, momentum сглаживает колебания.

## Интуитивная аналогия

Представим шар, катящийся по склону.

Если он долго движется вниз, его скорость увеличивается.

Momentum работает похожим образом.

## Практический эффект

Momentum позволяет:

- быстрее двигаться по пологим участкам  
- уменьшать зигзагообразные движения  
- быстрее достигать минимума

## Итог

Momentum ускоряет обучение, используя накопленную информацию о предыдущих шагах оптимизации.
