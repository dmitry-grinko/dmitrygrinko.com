# 14. Что такое ReLU функция?

ReLU (Rectified Linear Unit) — одна из самых распространённых функций активации в нейронных сетях.

## Формула

ReLU(x) = max(0, x)

## Как она работает

Если вход отрицательный — результат равен 0.  
Если вход положительный — значение остаётся без изменений.

## Пример

x = -3 → 0  
x = 5 → 5

## Итог

ReLU пропускает положительные значения и обнуляет отрицательные.
