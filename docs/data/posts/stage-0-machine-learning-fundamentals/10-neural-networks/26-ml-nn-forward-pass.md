# 26. Что такое forward pass?

Forward pass — это процесс, в котором входные данные проходят через нейронную сеть **от входного слоя к выходному**, и сеть вычисляет своё предсказание.

Это прямой проход данных через модель.

## Что входит в forward pass

Во время forward pass сеть делает только вычисления:

- умножение на веса  
- добавление bias  
- применение activation function  
- передача результатов дальше

На этом этапе веса не меняются.

## Пример

Если сеть получает изображение, во время forward pass она шаг за шагом преобразует его и в конце выдаёт, например:

- cat = 0.7  
- dog = 0.2  
- horse = 0.1  

Это и есть результат forward pass.

## Зачем он нужен

Без forward pass сеть вообще не смогла бы сделать предсказание.

Кроме того, именно после forward pass можно вычислить ошибку модели.

## Итог

Forward pass — это прямой проход данных через сеть, во время которого вычисляется предсказание модели.
