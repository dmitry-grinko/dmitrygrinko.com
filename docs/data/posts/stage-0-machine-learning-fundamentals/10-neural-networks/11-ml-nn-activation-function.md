# 11. Что такое activation function?

Activation function — это функция, которая применяется к выходу нейрона после вычисления линейной комбинации входов.

Напомним, что нейрон сначала вычисляет:

z = w₁x₁ + w₂x₂ + ... + wₙxₙ + b

После этого применяется функция активации:

y = activation(z)

## Зачем это нужно

Функция активации преобразует линейный результат нейрона в новый сигнал, который передаётся дальше по сети.

Она определяет, как нейрон реагирует на входной сигнал.

## Итог

Activation function — это функция, которая преобразует линейный выход нейрона в итоговый сигнал.
