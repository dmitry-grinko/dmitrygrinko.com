# 20. Почему градиенты важны для обучения?

Градиенты показывают, **как нужно изменить параметры модели**, чтобы уменьшить функцию потерь.

## Основная идея

Градиент — это направление наибольшего роста функции.

Поэтому параметры изменяются в противоположном направлении.

## Использование

Алгоритмы обучения делают:

1. вычисление градиента
2. обновление параметров
3. уменьшение loss

## Без градиентов

Алгоритм не знал бы, в каком направлении изменять параметры модели.

## Итог

Градиенты позволяют алгоритму оптимизации находить параметры, уменьшающие функцию потерь.
