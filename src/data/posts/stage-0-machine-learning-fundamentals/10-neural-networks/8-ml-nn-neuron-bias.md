# 8. Что такое bias в нейроне?

Bias — это дополнительный параметр нейрона, который добавляется к линейной комбинации входов.

## Формула

z = w₁x₁ + w₂x₂ + ... + wₙxₙ + b

где b — bias.

## Зачем нужен bias

Bias позволяет модели **смещать функцию**, не привязывая её строго к нулю.

Без bias модель была бы сильно ограничена.

## Интуитивная аналогия

Bias можно рассматривать как свободный член в линейной функции.

## Итог

Bias — это параметр, который позволяет нейрону смещать результат вычислений.
