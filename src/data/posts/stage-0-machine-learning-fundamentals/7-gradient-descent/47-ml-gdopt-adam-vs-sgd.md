# 47. Чем Adam отличается от SGD?

Adam и SGD отличаются способом обновления параметров.

## SGD

SGD использует градиент напрямую:

θ = θ − η ∇L

Learning rate одинаков для всех параметров.

## Adam

Adam использует:

- накопленные градиенты  
- адаптивные learning rate для каждого параметра

## Практическая разница

SGD:

- проще  
- требует более точной настройки learning rate

Adam:

- адаптивный  
- часто быстрее сходится

## Итог

Adam использует более сложную стратегию обновления параметров, чем SGD.
