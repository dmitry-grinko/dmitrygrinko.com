# 2. Почему logistic regression используется для задач классификации?

Logistic regression используется для классификации, потому что она предсказывает **вероятность принадлежности к классу**.

На основе этой вероятности можно определить итоговый класс объекта.

## Основная идея

Модель вычисляет вероятность:

P(class = 1 | features)

Затем эта вероятность сравнивается с порогом.

Если вероятность выше порога — объект относится к положительному классу.

Если ниже — к отрицательному.

## Пример

Модель оценила вероятность спама:

P(spam) = 0.82

Если порог равен 0.5, письмо будет классифицировано как spam.

## Почему это удобно

Предсказание вероятности полезно, потому что:

- можно регулировать threshold
- можно оценивать уверенность модели
- можно использовать разные метрики

## Итог

Logistic regression используется для классификации, потому что она предсказывает вероятность принадлежности объекта к классу.
