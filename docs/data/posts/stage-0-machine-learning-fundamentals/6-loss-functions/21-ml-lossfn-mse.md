# 21. Что такое Mean Squared Error (MSE)?

Mean Squared Error (MSE) — это функция потерь, которая измеряет **средний квадрат разницы между предсказанием модели и реальным значением**.

Она часто используется в задачах регрессии.

## Формула

MSE вычисляется так:

MSE = (1/n) Σ (y - ŷ)²

где:

- y — реальное значение  
- ŷ — предсказание модели  
- n — количество примеров

## Интуитивная идея

Сначала вычисляется ошибка:

error = y - ŷ

Затем она возводится в квадрат.

После этого ошибки усредняются по всему dataset.

## Почему используется квадрат

Квадрат делает две вещи:

- убирает знак ошибки  
- сильнее штрафует большие ошибки

## Итог

MSE измеряет средний квадрат ошибки предсказаний модели.
