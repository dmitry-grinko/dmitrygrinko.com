# 31. Что такое параметры нейронной сети?

Параметры нейронной сети — это значения, которые модель **обучает на данных**. Именно они определяют, как сеть преобразует входные данные в предсказание.

В нейронных сетях параметрами являются:

- веса (weights)
- bias

## Почему они называются параметрами

Потому что они определяют форму функции, которую реализует сеть.

Если изменить веса — изменится и поведение модели.

## Пример

Допустим, нейрон получает два входа:

x₁ = 2  
x₂ = 4  

и имеет веса:

w₁ = 0.5  
w₂ = -0.2  

bias = 1

Тогда вычисление:

z = 0.5 × 2 + (-0.2) × 4 + 1  
z = 1 - 0.8 + 1 = 1.2

Если изменить веса, результат будет другим.

## Важная идея

Обучение нейронной сети — это процесс **поиска правильных параметров**, которые минимизируют ошибку.

## Итог

Параметры нейронной сети — это веса и bias, которые определяют поведение модели.
