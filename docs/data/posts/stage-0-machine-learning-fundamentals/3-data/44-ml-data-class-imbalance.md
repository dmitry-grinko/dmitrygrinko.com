# 44. Что такое class imbalance?

Class imbalance — это ситуация, когда в dataset классы представлены **неравномерно**.

Один класс может встречаться значительно чаще, чем другой.

## Пример

Задача обнаружения мошенничества.

Dataset:

| class |
|------|
| normal |
| normal |
| normal |
| normal |
| fraud |

В этом примере большинство транзакций нормальные, а мошеннические встречаются редко.

## Почему это происходит

Во многих реальных задачах редкие события встречаются гораздо реже.

Например:

- мошенничество  
- заболевания  
- отказ оборудования

Такие случаи могут составлять менее 1% dataset.

## Почему это важно

Если модель обучается на несбалансированных данных, она может научиться просто предсказывать самый частый класс.

Например:

всегда предсказывать "normal".

Это может давать высокую accuracy, но модель будет бесполезной.

## Итог

Class imbalance — это ситуация, когда один класс в dataset значительно преобладает над другими.
