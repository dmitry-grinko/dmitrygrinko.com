# 31. Что такое log loss (cross entropy)?

Log loss (или cross entropy) — это функция потерь, которая используется в задачах классификации.

Она измеряет, **насколько предсказанная моделью вероятность отличается от правильного класса**.

## Основная идея

В задачах классификации модель предсказывает вероятность класса.

Например:

P(spam) = 0.9

Если реальный класс — spam, это хорошее предсказание.

Если вероятность была бы 0.1, ошибка была бы большой.

Log loss превращает эту разницу в числовую ошибку.

## Свойство функции

Log loss сильно штрафует **уверенные неправильные предсказания**.

## Итог

Log loss измеряет, насколько хорошо модель предсказывает вероятности классов.
