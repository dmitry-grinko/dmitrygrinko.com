# 35. Чем L1 отличается от L2 регуляризации?

L1 и L2 регуляризация отличаются тем, **как именно они штрафуют параметры модели**.

## L1 регуляризация

Использует абсолютные значения весов.

penalty = Σ |w|

Особенность:

может обнулять некоторые параметры модели.

## L2 регуляризация

Использует квадраты весов.

penalty = Σ w²

Особенность:

уменьшает веса, но обычно не делает их равными нулю.

## Практическая разница

L1 может уменьшать число используемых признаков.

L2 чаще используется для стабилизации модели.

## Итог

L1 может занулять веса, а L2 просто уменьшает их величину.
