# 13. Что такое mean squared error (MSE)?

Mean Squared Error (MSE) — одна из самых распространённых функций потерь в задачах регрессии.

Она измеряет среднее значение **квадратов ошибок предсказания**.

## Формула

MSE вычисляется так:

MSE = (1/n) Σ (y − ŷ)²

где:

- y — реальное значение  
- ŷ — предсказание модели  
- n — количество наблюдений

## Почему используется квадрат ошибки

Ошибки возводятся в квадрат, чтобы:

- большие ошибки наказывались сильнее  
- отрицательные и положительные ошибки не компенсировали друг друга

## Пример

Если ошибки модели:

-10  
5  
20

квадраты ошибок будут:

100  
25  
400

Среднее значение этих чисел и будет MSE.

## Итог

MSE — это среднее значение квадратов ошибок между реальными значениями и предсказаниями модели.
