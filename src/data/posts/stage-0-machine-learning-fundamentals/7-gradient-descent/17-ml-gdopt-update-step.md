# 17. Что такое шаг обновления параметров?

Шаг обновления параметров — это изменение параметров модели, которое происходит на одной итерации обучения.

Он вычисляется на основе градиента функции потерь.

## Формула

Δθ = − η ∇L

где:

- Δθ — изменение параметров  
- η — learning rate  
- ∇L — градиент

## Интерпретация

Градиент показывает направление роста функции.

Алгоритм движется в противоположную сторону.

Размер шага определяется learning rate.

## Итог

Шаг обновления параметров — это изменение параметров модели на одной итерации оптимизации.
