# 4. Что такое supervised learning?

Supervised learning — это тип обучения в машинном обучении, при котором модель обучается на **размеченных данных**.

Это означает, что для каждого примера в dataset известен правильный результат, который называется **label** или **target**.

Модель получает пары вида:

features → label

Задача модели — научиться предсказывать правильный label для новых данных.

## Как проходит обучение

Во время обучения модель:

1. получает входные данные (features)  
2. делает prediction  
3. сравнивает prediction с правильным ответом (label)  
4. корректирует параметры модели

Этот процесс повторяется много раз, пока модель не начнёт делать более точные предсказания.

## Пример

Представим dataset с информацией о квартирах.

| area | rooms | price |
|-----|------|------|
| 45  | 1    | 120000 |
| 70  | 2    | 185000 |
| 80  | 3    | 240000 |

Здесь:

- area и rooms — features  
- price — label

Модель может обучиться предсказывать цену квартиры на основе её характеристик.

## Где используется supervised learning

Supervised learning применяется во многих задачах:

- предсказание цен  
- обнаружение спама  
- кредитный скоринг  
- распознавание изображений  
- прогнозирование спроса

## Итог

Supervised learning — это подход, при котором модель обучается на размеченных данных и учится предсказывать правильный результат для новых примеров.
