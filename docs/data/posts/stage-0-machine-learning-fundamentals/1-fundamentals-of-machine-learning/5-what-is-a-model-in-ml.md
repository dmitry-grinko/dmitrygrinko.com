# 5. Что такое модель (model) в machine learning?

В машинном обучении **модель** — это математическая функция, которая преобразует входные данные в предсказание.

После обучения именно модель используется для получения результата.

Обобщённо это можно представить так:

```
prediction = model(features)
```


Где:

- `features` — входные признаки
- `prediction` — предсказание модели

Например:

- вероятность клика по рекламе
- категория изображения
- прогноз цены
- вероятность спама

## Что на самом деле хранит модель

Модель состоит из **параметров**, которые были найдены во время обучения.

Эти параметры кодируют закономерности, найденные в данных.

Например:

- веса признаков в линейной регрессии
- структура деревьев в gradient boosting
- веса нейронной сети

Именно эти параметры определяют поведение модели.

## Пример

В линейной регрессии модель может выглядеть так:

```
price = w1 * area + w2 * rooms + b
```


Где:

- `area` — площадь квартиры
- `rooms` — количество комнат
- `w1`, `w2`, `b` — параметры модели

Обучение как раз и состоит в том, чтобы подобрать значения этих параметров.

## Где используется модель

После обучения модель становится обычным компонентом системы.

Она может работать:

- в API
- в сервисе рекомендаций
- в аналитическом pipeline
- в real-time системе

## Итог

Модель в машинном обучении — это **обученная математическая функция**, которая:

- принимает признаки объекта
- использует параметры, найденные на данных
- выдаёт предсказание
