# 9. Как размер dataset влияет на переобучение?

Размер dataset напрямую влияет на риск переобучения.

Чем больше данных доступно для обучения, тем легче модели найти реальные закономерности.

## Маленький dataset

Когда данных мало:

- модель видит ограниченное количество примеров  
- случайные особенности становятся более заметными  
- модель может начать запоминать данные

## Большой dataset

Когда dataset большой:

- случайный шум усредняется  
- закономерности становятся более очевидными  
- модель учится обобщать

## Практическое следствие

Многие современные модели требуют очень больших dataset, чтобы избежать переобучения.

## Итог

Чем меньше dataset, тем выше риск переобучения.
