# 16. Что такое sigmoid функция?

Sigmoid — это функция активации, которая преобразует любое число в диапазон от 0 до 1.

## Формула

σ(x) = 1 / (1 + e⁻ˣ)

## Свойство функции

При больших отрицательных значениях результат близок к 0.

При больших положительных значениях результат близок к 1.

## Интерпретация

Поэтому sigmoid удобно интерпретировать как вероятность.

## Итог

Sigmoid — функция активации, которая переводит значение в диапазон от 0 до 1.
