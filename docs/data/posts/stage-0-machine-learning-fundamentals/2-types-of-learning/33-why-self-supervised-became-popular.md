# 33. Почему self-supervised learning стал популярным?

Self-supervised learning стал особенно популярным в последние годы, потому что он позволяет обучать модели на огромных datasets без необходимости ручной разметки.

Это делает его особенно полезным для современных систем машинного обучения.

## Огромные объемы данных

Сегодня доступно огромное количество неразмеченных данных:

- тексты из интернета  
- изображения  
- видео  
- аудио

Разметить такие datasets вручную практически невозможно.

Self-supervised learning позволяет использовать эти данные напрямую.

## Обучение больших моделей

Современные модели, такие как большие нейронные сети, требуют огромного количества обучающих данных.

Self-supervised learning помогает обучать такие модели, используя автоматически созданные задачи.

## Предварительное обучение моделей

Часто модели сначала обучаются на огромных неразмеченных datasets, чтобы научиться представлять данные.

После этого они дообучаются на небольшом размеченном dataset.

Этот процесс называется **pretraining**.

## Итог

Self-supervised learning стал популярным потому что он:

- позволяет использовать огромные datasets  
- уменьшает необходимость ручной разметки  
- помогает обучать большие модели
