# 37. Что такое градиенты в нейронных сетях?

Градиент — это величина, которая показывает, **как изменится функция потерь при изменении параметра**.

Он указывает направление наибольшего роста функции.

Для обучения модели используется противоположное направление.

## Пример

Допустим, есть параметр w.

Если:

∂Loss/∂w = 0.8

это означает, что увеличение w увеличивает ошибку.

Значит, нужно уменьшить w.

## Важная идея

Градиенты показывают:

- какие параметры нужно увеличить
- какие уменьшить
- насколько сильно

## Итог

Градиенты показывают, как изменение параметров влияет на ошибку модели.
