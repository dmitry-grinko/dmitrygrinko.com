# 7. Почему supervised learning требует размеченные данные?

Supervised learning требует размеченные данные, потому что модель должна знать, **какой результат является правильным**.

Без правильных ответов модель не может понять, насколько её предсказание было точным.

Разметка данных создаёт так называемый **сигнал обучения**.

## Как работает обучение

Во время обучения модель делает предсказание для каждого примера.

Затем происходит сравнение:

prediction → сравнивается → label

Если предсказание отличается от правильного ответа, модель получает сигнал ошибки и корректирует свои параметры.

Этот процесс повторяется много раз.

Постепенно модель начинает делать более точные предсказания.

## Что происходит без разметки

Если dataset не содержит правильных ответов, модель не может напрямую проверить, насколько её предсказание правильное.

В этом случае невозможно обучить модель решать задачу предсказания.

Именно поэтому supervised learning требует dataset, где известен правильный результат.

## Почему разметка может быть сложной

Во многих задачах разметка данных требует значительных ресурсов.

Например:

- медицинские изображения должны размечать специалисты  
- тексты могут требовать ручной классификации  
- видео и изображения требуют длительной обработки

Поэтому создание качественного размеченного dataset часто является самой трудоёмкой частью ML-проекта.

## Итог

Supervised learning требует размеченные данные, потому что модель должна сравнивать свои предсказания с правильными ответами и использовать эту ошибку для обучения.
