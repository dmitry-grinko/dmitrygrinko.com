# 3. Чем logistic regression отличается от linear regression?

Linear regression и logistic regression используют похожую структуру модели, но решают разные задачи.

## Linear regression

Linear regression используется для **предсказания числовых значений**.

Например:

- цена квартиры
- температура
- объём продаж

Выход модели может быть любым числом.

## Logistic regression

Logistic regression используется для **классификации**.

Она предсказывает вероятность, которая находится в диапазоне:

0 — 1

## Основное различие

Linear regression:

предсказывает числовое значение

Logistic regression:

предсказывает вероятность класса

## Пример

Linear regression:

price = 200000

Logistic regression:

P(spam) = 0.82

## Итог

Linear regression используется для регрессии, а logistic regression — для задач классификации.
