# 40. Какие способы борьбы с multicollinearity существуют?

Существует несколько способов уменьшить влияние multicollinearity.

## Удаление признаков

Если два признака содержат почти одинаковую информацию, один из них можно удалить.

Это самый простой подход.

## Объединение признаков

Иногда признаки можно объединить в один новый признак.

Например:

общая площадь вместо нескольких отдельных площадей.

## Dimensionality reduction

Методы уменьшения размерности, такие как PCA, могут преобразовать признаки в новый набор переменных.

## Регуляризация

Методы регуляризации, такие как Ridge regression, помогают уменьшить влияние multicollinearity.

## Итог

С multicollinearity борются через:

- удаление признаков  
- объединение признаков  
- уменьшение размерности  
- регуляризацию
