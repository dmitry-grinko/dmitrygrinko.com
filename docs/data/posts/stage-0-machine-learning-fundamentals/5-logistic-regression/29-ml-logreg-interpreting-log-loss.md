# 29. Как интерпретировать значение log loss?

Log loss измеряет качество вероятностных предсказаний модели.

Чем меньше значение log loss, тем лучше работает модель.

## Интерпретация

log loss = 0

означает идеальные предсказания.

Чем больше значение, тем хуже модель оценивает вероятности.

## Сравнение моделей

Log loss обычно используется для сравнения нескольких моделей.

Модель с меньшим log loss считается лучше.

## Особенность

Log loss учитывает не только правильность предсказаний, но и **уверенность модели**.

## Итог

Меньшее значение log loss означает более точные и лучше откалиброванные вероятности.
