# 13. Чем unsupervised learning отличается от supervised learning?

Основное различие между supervised и unsupervised learning заключается в **наличии правильных ответов в dataset**.

В supervised learning данные содержат информацию о том, каким должен быть результат. В unsupervised learning таких ответов нет.

## Supervised learning

В supervised learning каждый пример в dataset имеет известный результат.

Например:

| area | rooms | price |
|-----|------|------|
| 45 | 1 | 120000 |

Здесь модель знает, что правильный результат — это цена квартиры.

Задача модели — научиться предсказывать этот результат для новых данных.

## Unsupervised learning

В unsupervised learning dataset содержит только признаки.

Например:

| age | income | purchases |
|----|-------|----------|
| 25 | 40000 | 3 |

В этом случае модель не знает, какой результат нужно предсказывать.

Она пытается обнаружить:

- группы похожих объектов  
- закономерности между признаками  
- структуру данных

## Различие в задачах

Supervised learning обычно используется для:

- предсказания числовых значений  
- классификации объектов

Unsupervised learning чаще применяется для:

- анализа данных  
- сегментации объектов  
- поиска структуры в dataset

## Итог

Главное отличие:

- supervised learning использует размеченные данные  
- unsupervised learning работает с неразмеченными данными
