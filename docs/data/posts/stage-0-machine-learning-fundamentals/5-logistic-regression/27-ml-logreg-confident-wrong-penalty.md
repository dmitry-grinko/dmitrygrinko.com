# 27. Почему уверенные неправильные предсказания сильно увеличивают loss?

В log loss используется логарифм вероятности правильного класса.

Когда вероятность правильного класса становится очень маленькой, логарифм резко уменьшается.

## Пример

Если модель предсказывает:

p = 0.9 → log loss маленький  
p = 0.5 → log loss средний  
p = 0.01 → log loss очень большой

## Интуитивная причина

Модель должна быть **уверенной только тогда, когда она права**.

Если модель уверена и ошибается, это считается серьёзной ошибкой.

## Итог

Log loss сильно увеличивается для уверенных неправильных предсказаний, чтобы модель избегала таких ошибок.
