# 6. Как loss function связана с оптимизацией модели?

Оптимизация модели — это процесс поиска параметров, которые минимизируют функцию потерь.

## Основная идея

Loss function задаёт цель обучения.

Алгоритм оптимизации пытается найти параметры модели, которые делают loss минимальным.

## Пример

Алгоритм gradient descent выполняет следующие шаги:

1. вычисляет loss  
2. вычисляет градиент функции потерь  
3. изменяет параметры модели

## Результат

После большого числа итераций модель находит параметры, при которых ошибка минимальна.

## Итог

Loss function определяет цель оптимизации модели.
