# 44. Чем L1 отличается от L2 регуляризации?

L1 и L2 — это два разных способа штрафовать большие коэффициенты модели.

Оба метода используются для борьбы с переобучением.

## L1 регуляризация

L1 использует сумму абсолютных значений коэффициентов.

penalty = Σ |w|

Особенность L1:

- может делать коэффициенты равными нулю  
- выполняет feature selection

## L2 регуляризация

L2 использует сумму квадратов коэффициентов.

penalty = Σ w²

Особенность L2:

- уменьшает коэффициенты  
- но редко делает их нулевыми

## Различие в поведении

L1:

- делает модель более разреженной  
- может исключать признаки

L2:

- сглаживает коэффициенты  
- сохраняет все признаки

## Итог

L1 регуляризация может обнулять коэффициенты, а L2 просто уменьшает их значения.
