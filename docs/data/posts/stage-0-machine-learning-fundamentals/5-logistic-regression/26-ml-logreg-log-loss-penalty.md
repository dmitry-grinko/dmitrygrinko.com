# 26. Как log loss штрафует неправильные предсказания?

Log loss использует логарифм вероятности правильного класса.

Из-за этого он **очень сильно увеличивается**, когда модель уверенно делает неправильное предсказание.

## Пример

Если истинный класс:

y = 1

и модель предсказывает:

p = 0.9 → маленький loss  
p = 0.6 → умеренный loss  
p = 0.1 → большой loss

## Почему это важно

Такой механизм заставляет модель:

- избегать уверенных ошибок  
- лучше калибровать вероятности

## Итог

Log loss увеличивается особенно сильно, когда модель с высокой уверенностью предсказывает неправильный класс.
